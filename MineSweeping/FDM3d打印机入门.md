# FDM 3D 打印快速上手指南

写给第一次摸打印机的人。不废话，只讲"怎么做"和"为什么翻车"。

---

## 这是什么

**FDM（熔融沉积成型）** 就是把塑料线材加热熔化成"牙膏"，一层层挤出来堆成一个立体东西。跟你用热熔胶枪画圈堆高差不多，只是精度高很多。

市面上有两种机身：

| 类型                           | 特点             | 适合打的材料              |
| ------------------------------ | ---------------- | ------------------------- |
| **封箱型**（如拓竹 P1S） | 有外壳，仓温稳定 | PLA、PETG、ABS、ASA、尼龙 |
| **开放型**（如 Ender 3） | 没外壳，散热快   | PLA、PETG、TPU            |

> 封箱机能打开放机能打的几乎所有材料。开放机打 ABS/ASA 基本必翘边——没箱体保温，冷热一拉就变形了。

---

## 你需要什么

- **3D 模型文件**：`.stl`、`.step(.stp)`、`.3mf`、`.amf` 都行。自己建模的话，任何 CAD 软件另存为以上格式即可。网上下的模型建议跑一遍切片预览确认没问题。
- **切片软件**：把模型转成打印机认识的指令文件（`.gcode` 或厂家私有格式）。常见的有 Bambu Studio、Creality Print、PrusaSlicer、Cura、OrcaSlicer。**用你打印机对应的就行，别混用。**
- **耗材（线材）**：一卷塑料线，1.75mm 直径最常见。具体怎么选见下方「材料速查」。
- **一台 FDM 打印机**。

## 从模型到实物：完整流程

### 1. 切片（最关键的一步）

切片就是把 3D 模型"翻译"成打印机指令。这一步翻车，后面全白搭。

打开切片软件，导入模型后，**依次确认这几项**：

**① 选对机型**
别用错打印机配置。封箱机选了开放机的配置文件，温度策略不对，打出来就是废的。

**② 选对喷嘴直径**
常见 0.4mm。如果你换了 0.6mm 或 0.2mm 喷嘴，切片里也要对应改。0.4mm 的切片发到 0.6mm 机器上——挤出量不对，要么欠挤出要么糊成一团。

**③ 选对耗材类型**
软件里选的耗材要和实物一致。PLA 配置打 PETG = 温度不够粘不上，PETG 配置打 PLA = 温度过高拉丝拉到崩溃。不要问故意混打会怎样，问就是打印机机魂不悦，当场坏给你看。

**④ 悬空部分开支撑**
模型上悬空超过 ~45° 的斜面、桥接、架空结构，都要开支撑。不开的话塑料挤到空气里，直接塌下来。切片完切到「预览」视图，拖进度条从上往下看一遍——首层悬空的、大角度外伸的，都是翻车预定。

**⑤ 调参数（初次先默认）**
层高、填充密度、打印速度这些，新手先用默认设置。层高低 = 精细但慢，填充高 = 结实但费料费时。熟了再调。

确认无误 → 切片 → 导出。

### 2. 打印机准备

**调平**
绝大多数现代打印机有自动调平，点一下等它跑完就行。没有自动调平的机器需要手动拧四个角的热床旋钮——用一张 A4 纸垫在喷嘴和热床之间，调到纸能拉动但有阻力为止。四个角各来一遍。

**放打印板**

- 手只抓边缘，别摸中间。手指上的油 = 粘不住。
- 不确定粘附力的话，用酒精擦一遍板面，再喷一层薄胶（固体胶、发胶喷雾都行）。
- 90%以上的板子都有防呆机制，just看一遍，别放反了。
- 板子底边对齐，落下去，听到磁吸/卡扣到位的声音就行。**横着硬拽 = 毁热床。**

**进料**

- 确认料盘上的材料和切片设置一致。
- 按打印机菜单里的「进料」操作。料要穿过断料检测传感器（如果有的话），否则机器认为你没放料，不开始打印。

### 3. 发送打印

两种方式：

- **U 盘**：导出 `.gcode` 到 U 盘 → 插打印机 → 屏幕上选文件 → 开始。
- **联网发送**（WiFi 打印）：切片完直接点「发送」，等传输完毕，机器开始动作了再走。

  以拓竹 P1S 为例的配网流程（其他品牌的 WiFi 打印大同小异）：

  1. 打开手机热点（打印机连你的热点或同一局域网 WiFi）。
  2. 打印机屏幕上切到「齿轮」图标 →「账号」，会显示一个二维码。
  3. 手机打开对应 App（如 Bambu Handy）→「我的」→ 扫码，扫描打印机屏幕上的二维码。
  4. 等握手——有时两边显示不同步，刷新一下就行。App 显示已连接即成功。
  5. 切回切片软件（如 Bambu Studio）→「设备」标签页，确认机器已在线。
  6. 之后切片完直接点「发送」即可。

**首层永远盯着看。** 95% 的翻车都发生在首层——没粘住、翘边、喷嘴刮板子。首层贴好了，后面基本不会出大事。

### 4. 打印中途

- **不要动打印板，不要开门**（封箱机）。开门 = 仓温骤降 = 模型可能翘边变形。
- 看到喷嘴堆料想清理？**先暂停**，再快速处理。别在打印进行中伸手。
- 已经严重翻车（意面状、大面积堆料）？**直接取消，不要救。** 清理干净重新来。
- 耗材用尽 → 大部分机器有断料检测，会自动暂停，换料后继续。

### 5. 取件

- 打完等一会。热床和板子都有余温，刚打完的模型也比较软。
- 取板子：抓住边缘向上掀，再水平拉出。**别横着拽。**
- 弯一弯板子，模型大概率自己弹下来。
- 粘太紧的：先用塑料铲刀，搞不定再上金属铲刀（但小心别刮花板子）。
- **喷嘴打完还有 200+ 度，取件别碰。**

## 材料速查

| 材料                  | 难度 | 喷嘴温度    | 热床温度   | 特点                   | 注意                                     |
| --------------------- | ---- | ----------- | ---------- | ---------------------- | ---------------------------------------- |
| **PLA**         | 入门 | 190–220°C | 50–60°C  | 好打、没毒、便宜       | 不耐热，夏天放车里会变形                 |
| **PETG**        | 初级 | 230–250°C | 70–80°C  | 比 PLA 结实、耐热一些  | 容易拉丝，调好回抽参数                   |
| **TPU**         | 中级 | 220–240°C | 50–60°C  | 柔性、有弹性           | 打得慢，进料系统不能是长程挤出           |
| **ABS**         | 进阶 | 230–260°C | 90–110°C | 结实、耐热、可丙酮抛光 | **必须有封箱**，有味道，通风       |
| **ASA**         | 进阶 | 240–260°C | 90–110°C | 户外用，抗 UV          | 同 ABS，必须封箱+通风                    |
| **碳纤/玻纤混** | 进阶 | 视基材      | 视基材     | 刚性高                 | **磨损普通黄铜喷嘴**，需硬化钢喷嘴 |

> **新手建议从 PLA 开始。** 打熟 PLA 了再碰 PETG，然后 ABS。TPU 是另一条技能树。

## 常见翻车 & 怎么办

| 现象                        | 可能原因                               | 试试                                   |
| --------------------------- | -------------------------------------- | -------------------------------------- |
| **首层不粘**          | 喷嘴离床太远 / 床脏了                  | 重新调平；酒精擦板；上胶               |
| **翘边**              | 热床温度不够 / 冷风直吹 / 开放机打 ABS | 加热床温度；关窗关空调；开放机别打 ABS |
| **拉丝**              | 温度太高 / 回抽不够                    | 降 5°C 试试；加大回抽距离             |
| **层间开裂**          | 温度太低 / 打印太快                    | 升 5–10°C；降速                      |
| **堵头**              | 温度不够 / 料里有杂质 / 喷嘴磨损       | 升到材料上限温度推料；换喷嘴           |
| **意面（Spaghetti）** | 模型中途脱离热床 / 支撑倒了            | 加 brim（裙边）；检查支撑设置；涂胶    |

## 安全底线

- 喷嘴 200+°C，热床 60–100+°C，**别碰。** 打完了也等会再摸。
- ABS、ASA 等材料加热时有 VOC 挥发，**保持通风**。封箱 ≠ 密封，该通风还是通风。
- 打印机开着的时候别无人值守太久。虽然火灾风险很低，但不是零。
- 碳纤/玻纤耗材：粉尘有害 + 磨损喷嘴。打磨碳纤件时要戴口罩。
- 用完的料盘收好密封（最好带干燥剂），受潮的料 = 打出来全是气泡和脆层。
