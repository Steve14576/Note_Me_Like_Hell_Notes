# 第6-7章：雷诺数、欧拉方程、伯努利方程、Pitot管、水头

---

## 1. 雷诺数（Reynolds Number）

### 说人话

雷诺数是一个**无量纲数**，告诉你流体到底是"规矩地层层流动"还是"乱成一锅粥"。

$$Re = \frac{\rho v D}{\mu} = \frac{v D}{\nu}$$

- $v$：流体平均速度
- $D$：管道直径（特征长度）
- $\nu = \mu/\rho$：运动粘度

### 物理含义

$$Re = \frac{\text{惯性力}}{\text{粘性力}}$$

- $Re$ 小 → 粘性力主导 → 流体"听话" → **层流**
- $Re$ 大 → 惯性力主导 → 流体"造反" → **湍流**

### 圆管流动的临界雷诺数

| 范围 | 流态 |
|------|------|
| $Re < 2300$ | 层流（Laminar） |
| $2300 < Re < 4000$ | 过渡区（不确定） |
| $Re > 4000$ | 湍流（Turbulent） |

### 经典实验

雷诺1883年的染色线实验：
- 层流时：染色线保持一条直线
- 湍流时：染色线迅速扩散混合

## 2. 欧拉方程（Euler's Equation）

### 说人话

欧拉方程是**忽略粘性**的流体运动方程（理想流体）。它就是流体版的"牛顿第二定律"。

### 推导思路

对流体微元应用 $F = ma$：

沿流线方向（s方向）：

$$-\frac{1}{\rho}\frac{\partial p}{\partial s} - g\frac{\partial z}{\partial s} = v\frac{\partial v}{\partial s}$$

意思是：**压力梯度力 + 重力 = 惯性力（加速度）**

### 为什么重要

积分欧拉方程沿流线 → 得到**伯努利方程**！

---

## 3. 伯努利方程（Bernoulli's Equation）

### 说人话

沿着一条流线，把能量加起来，总量不变（忽略粘性损失时）。就像坐过山车——高的时候慢（势能大动能小），低的时候快（动能大势能小），但总能量不变。

### 公式（沿流线，理想流体，稳态，不可压缩）

$$p + \frac{1}{2}\rho v^2 + \rho g z = \text{常数}$$

或者除以 $\rho g$，变成**水头形式**：

$$\frac{p}{\rho g} + \frac{v^2}{2g} + z = \text{常数}$$

### 三个项的物理意义

| 项 | 名称 | 说人话 |
|----|------|--------|
| $\frac{p}{\rho g}$ | 压力水头 | 压力能 |
| $\frac{v^2}{2g}$ | 速度水头 | 动能 |
| $z$ | 位置水头 | 势能（高度） |

三者之和 = **总水头** $H$，沿流线不变。

### 适用条件（很重要！）

1. **稳态**流动（不随时间变化）
2. **不可压缩**流体（$\rho$ = 常数）
3. **无粘性**（理想流体）或沿流线应用
4. 沿**同一条流线**

### 实际流体的伯努利方程（有损失）

实际中有粘性损失，在两个截面之间：

$$\frac{p_1}{\rho g} + \frac{v_1^2}{2g} + z_1 = \frac{p_2}{\rho g} + \frac{v_2^2}{2g} + z_2 + h_L$$

$h_L$ = 水头损失（head loss），后面会详细讲。

---

## 4. 水头（Head）的概念

### 说人话

"水头"就是把能量用"等效水柱高度"来表示。与其说"这里压力是98100 Pa"，不如说"这里压力水头是10米"——更直观。

### 各种水头

| 水头 | 表达式 | 含义 |
|------|--------|------|
| 压力水头 | $h_p = p/(\rho g)$ | 压力对应的水柱高度 |
| 速度水头 | $h_v = v^2/(2g)$ | 动能对应的水柱高度 |
| 位置水头 | $z$ | 海拔高度 |
| 总水头 | $H = h_p + h_v + z$ | 总机械能 |
| 测压管水头 | $h_p + z$ | 压力水头 + 位置水头 |

### 水头线

- **总水头线（EGL，Energy Grade Line）**：$H = \frac{p}{\rho g} + \frac{v^2}{2g} + z$，沿流向**只降不升**
- **测压管水头线（HGL，Hydraulic Grade Line）**：$\frac{p}{\rho g} + z$，可升可降

EGL 和 HGL 的差 = 速度水头 $v^2/(2g)$

## 5. 皮托管（Pitot Tube）

### 说人话

皮托管是测量**点速度**的仪器。原理超级简单：把一根管子正对着水流，管里的水会被"堵住"不动，动能全部转化为压能，水位升高。升高的那部分就代表了速度。

### 工作原理

在流线上取两个点：
- **A点**：来流未受干扰，速度为 $v$，压力为 $p$（静压）
- **B点**：皮托管开口处，流体被"堵住"，速度为0，压力为 $p_0$（驻点压力/总压）

对A、B用伯努利方程：

$$p + \frac{1}{2}\rho v^2 = p_0$$

所以：

$$v = \sqrt{\frac{2(p_0 - p)}{\rho}} = \sqrt{2g \Delta h}$$

其中 $\Delta h$ 是皮托管测得的液柱高差。

### 实际皮托管

实际使用时要加修正系数 $c$（一般接近1）：

$$v = c\sqrt{2g\Delta h}$$

### 皮托-静压管（Pitot-static tube）

实际中常用的是组合式：中心孔测总压，侧面小孔测静压，差值就是动压。

```
        总压孔（正对来流）
         ↓
  ====[●]========
       |  |
       |  └── 静压孔（侧面）
       |
    Δh = 总压 - 静压 = ρv²/2
```

## 6. 伯努利方程的应用实例

### 例1：水箱出口速度（Torricelli定理）

大水箱侧面开个小孔，水面到小孔的高度为 $h$：

- 水面（1）：$v_1 ≈ 0$（水箱很大），$p_1 = p_{atm}$
- 出口（2）：$p_2 = p_{atm}$

代入伯努利：

$$0 + 0 + h = 0 + \frac{v_2^2}{2g} + 0$$

$$v_2 = \sqrt{2gh}$$

**说人话**：出口速度就像从高度 $h$ 自由落体的速度一样！

### 例2：文丘里管（下一章详细讲）

管子先缩小再扩大。缩小处速度大，压力小；比较两处的压差就能算流量。

## 本章核心公式汇总

| 公式 | 含义 |
|------|------|
| $Re = \frac{vD}{\nu}$ | 雷诺数 |
| $p + \frac{1}{2}\rho v^2 + \rho gz = C$ | 伯努利方程（压力形式） |
| $\frac{p}{\rho g} + \frac{v^2}{2g} + z = C$ | 伯努利方程（水头形式） |
| $v = \sqrt{2g\Delta h}$ | 皮托管测速公式 |
| $v = \sqrt{2gh}$ | Torricelli出流公式 |

---

## 关键理解检查

1. 管子变细 → 速度变大 → 压力**变小**（不是变大！）
2. 伯努利方程是**能量守恒**，不是力平衡
3. 水头损失只能是正的（能量只能损失，不能凭空产生）
4. 皮托管测的是**点速度**，不是平均速度
