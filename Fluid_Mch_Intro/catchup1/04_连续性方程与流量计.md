# 第8章：连续性方程、Venturi流量计、Orifice孔板流量计

---

## 1. 连续性方程（Continuity Equation）

### 说人话

水管里的水不会凭空消失也不会凭空产生。如果管子变细了，流速就必须加快，才能保证同样多的水通过。

就像高速公路从4车道变成2车道——车必须开更快（或者堵车），才能保证车流量不变。

### 数学表达

对于**稳态、不可压缩**流体：

$$A_1 v_1 = A_2 v_2 = Q = \text{常数}$$

- $A$：截面积
- $v$：截面平均速度
- $Q$：体积流量（m³/s）

### 质量守恒形式（可压缩）

$$\rho_1 A_1 v_1 = \rho_2 A_2 v_2 = \dot{m} = \text{常数}$$

$\dot{m}$ = 质量流量（kg/s）

### 关键推论

管子变细（$A$ 减小）→ 速度**增大**（$v$ 增大）→ 压力**减小**（伯努利）

这个逻辑链是一切流量测量装置的基础！

## 2. 文丘里流量计（Venturi Flow Meter）

### 说人话

管子中间做一个"收腰"（喉部变细）。根据连续性方程，喉部速度大；根据伯努利方程，喉部压力小。量一下两处的**压差**，就能**反算出流量**。

### 结构

```
     ← 流向
  ═══╗         ╔═══
     ║ 收缩段  ║ 扩散段
     ║    ╔════╝
     ║    ║ 喉部
     ║    ╚════╗
     ║         ║
  ═══╝         ╚═══
  截面1        截面2
  (大管)       (喉部)
```

### 推导

在截面1和截面2之间用伯努利方程 + 连续性方程：

**伯努利**（同一水平，忽略损失）：
$$\frac{p_1}{\rho g} + \frac{v_1^2}{2g} = \frac{p_2}{\rho g} + \frac{v_2^2}{2g}$$

**连续性**：
$$A_1 v_1 = A_2 v_2 \quad \Rightarrow \quad v_1 = \frac{A_2}{A_1} v_2$$

消去 $v_1$：

$$v_2 = \frac{1}{\sqrt{1 - (A_2/A_1)^2}} \sqrt{\frac{2(p_1 - p_2)}{\rho}}$$

流量：

$$Q_{理论} = A_2 v_2 = \frac{A_2}{\sqrt{1 - (d_2/d_1)^4}} \sqrt{2g\Delta h}$$

其中 $\Delta h = \frac{p_1 - p_2}{\rho g}$ 是压差对应的水头差。

### 流量系数 $\mu$（Discharge Coefficient）

实际流量比理论值小（因为有粘性损失），所以：

$$Q_{实际} = \mu \cdot Q_{理论}$$

- 文丘里管的 $\mu$ 通常在 **0.95 ~ 0.99** 之间（损失小，效率高）
- $\mu$ 与雷诺数有关，需要实验标定

### 文丘里管的优缺点

| 优点 | 缺点 |
|------|------|
| 压力损失小（扩散段逐渐恢复压力） | 造价高 |
| 精度高 | 体积大 |
| 可测量大流量 | 安装要求高 |

## 3. 孔板流量计（Orifice Plate Flow Meter）

### 说人话

比文丘里管更粗暴的方法：直接在管子中间放一块带孔的板子。水被迫从小孔挤过去，形成高速射流，同样利用压差算流量。

### 结构

```
     ← 流向
  ═══════╗   ╔═══════
         ║ ┃ ║         ← 孔板（中间一个圆孔）
  ═══════╝   ╚═══════
  截面1   缩脉  截面2
         (vena contracta)
```

### 关键现象：缩脉（Vena Contracta）

流体通过孔板后，不会立即充满整个管道。射流会继续收缩一小段距离，在**孔板下游约半个管径处**达到最小截面——这就是**缩脉**。

- 缩脉处：速度最大，压力最小
- 缩脉截面 < 孔板开孔面积

### 流量公式

$$Q = C_d A_0 \sqrt{\frac{2\Delta p}{\rho(1 - \beta^4)}}$$

其中：
- $C_d$：流量系数（约 0.6~0.65）
- $A_0$：孔板开孔面积
- $\Delta p = p_1 - p_2$：孔板前后压差
- $\beta = d_0/D$：孔径比（孔板孔径/管径）

简化形式（当 $\beta$ 较小时 $1-\beta^4 ≈ 1$）：

$$Q ≈ C_d A_0 \sqrt{\frac{2\Delta p}{\rho}}$$

### 为什么 $C_d$ 比文丘里管的 $\mu$ 小很多？

因为孔板流量计有两个损失来源：
1. 缩脉效应（实际通流面积 < 孔面积）
2. 孔板后的涡流和湍流混合（不可逆损失大）

### 孔板的优缺点

| 优点 | 缺点 |
|------|------|
| 结构极其简单 | 压力损失大（约40%~90%的压差不可恢复） |
| 造价低 | 精度比文丘里管低 |
| 容易安装和更换 | 对安装条件敏感 |

## 4. 三种流量测量方式对比

| 特性 | 文丘里管 | 孔板 | 皮托管 |
|------|---------|------|--------|
| 测量对象 | 流量 | 流量 | 点速度 |
| 流量系数 | 0.95~0.99 | 0.6~0.65 | — |
| 压力损失 | 小(5~15%) | 大(40~90%) | 极小 |
| 精度 | 高 | 中 | 高（点测量） |
| 成本 | 高 | 低 | 中 |
| 原理 | 伯努利+连续性 | 伯努利+连续性 | 伯努利 |

## 本章核心公式汇总

| 公式 | 含义 |
|------|------|
| $A_1 v_1 = A_2 v_2$ | 连续性方程 |
| $Q = \mu \frac{A_2}{\sqrt{1-(d_2/d_1)^4}} \sqrt{2g\Delta h}$ | 文丘里流量 |
| $Q = C_d A_0 \sqrt{\frac{2\Delta p}{\rho(1-\beta^4)}}$ | 孔板流量 |

## 解题思路

1. 画出截面位置（大管、喉部/孔板处）
2. 列**连续性方程**关联两个截面的速度
3. 列**伯努利方程**关联两个截面的压力
4. 联立求解，别忘了乘流量系数
