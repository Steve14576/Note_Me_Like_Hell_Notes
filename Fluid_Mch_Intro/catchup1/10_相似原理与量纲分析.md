# 第14章：相似原理与量纲分析（Similitude & Dimensional Analysis）

---

## 1. 为什么需要相似原理和量纲分析？

### 说人话

想测试一艘航母在海上的阻力，你不可能造一艘真正的航母来做实验。你会造个小模型放在水池里测。但问题来了：

- 模型多大才合理？
- 模型里的水速设多少？
- 测出来的力怎么换算成真实值？

**相似原理**就是解决"模型实验结果怎么等效到原型"的理论。
**量纲分析**是用来精简变量、找出无量纲参数的数学工具。

---

## 2. 相似的三个层次

### (1) 几何相似（Geometric Similarity）

模型和原型**形状一样**，所有对应长度成比例：

$$\lambda_L = \frac{L_m}{L_p} = \text{常数}$$

- 下标 $m$ = 模型（model），$p$ = 原型（prototype）
- 所有线性尺寸缩放同一比例
- 面积比 = $\lambda_L^2$，体积比 = $\lambda_L^3$

### (2) 运动相似（Kinematic Similarity）

对应点的**速度方向相同**，速度大小成比例：

$$\lambda_v = \frac{v_m}{v_p} = \text{常数}$$

同时保证：
- 对应点的加速度也成比例
- 流线图案相似（"流动的形状"一样）

### (3) 动力相似（Dynamic Similarity）

对应点的**力的方向相同**，力的大小成比例：

$$\lambda_F = \frac{F_m}{F_p} = \text{常数}$$

所有种类的力（惯性力、粘性力、重力、压力等）都保持相同比例。

> 💡 **核心结论**：如果满足动力相似，则几何相似和运动相似自动满足。动力相似是最高要求。

---

## 3. 动力相似的判据——无量纲数

### 说人话

完全的动力相似通常做不到（不可能所有力都同时满足比例）。实际中，选择**最重要的那个力比值**相等即可。

### 雷诺数相似（粘性力主导）

$$Re_m = Re_p \quad \Rightarrow \quad \frac{v_m L_m}{\nu_m} = \frac{v_p L_p}{\nu_p}$$

适用于：
- 管道流动
- 低速飞行器
- 全部浸没的水下物体

### 弗劳德数相似（重力主导）

$$Fr_m = Fr_p \quad \Rightarrow \quad \frac{v_m}{\sqrt{g L_m}} = \frac{v_p}{\sqrt{g L_p}}$$

适用于：
- 明渠流动
- 船舶（水面波浪阻力）
- 溢洪道模型

### 马赫数相似（可压缩性主导）

$$Ma_m = Ma_p \quad \Rightarrow \quad \frac{v_m}{c_m} = \frac{v_p}{c_p}$$

适用于：高速空气动力学（飞机超音速飞行）

> 💡 **实际困境**：同时满足Re和Fr相似通常不可能（除非用不同流体）。工程中根据主要效应选择一个准则。

---

## 4. 量纲齐次性（Dimensional Homogeneity）

### 原则

**任何物理方程的每一项必须具有相同的量纲。**

这不只是"检查公式对不对"的工具，更是推导公式的基础。

### 基本量纲（MLT系统）

| 物理量 | 量纲 |
|--------|------|
| 质量 | M |
| 长度 | L |
| 时间 | T |
| 速度 | $LT^{-1}$ |
| 加速度 | $LT^{-2}$ |
| 力 | $MLT^{-2}$ |
| 压力 | $ML^{-1}T^{-2}$ |
| 密度 | $ML^{-3}$ |
| 动力粘度 | $ML^{-1}T^{-1}$ |
| 运动粘度 | $L^2T^{-1}$ |

---

## 5. 方程无量纲化（Non-dimensionalisation）

### 说人话

把方程里的每个变量除以一个"参考值"，使其变成无量纲的。这样做的好处：
- 减少变量数目
- 自然出现无量纲参数（Re、Fr等）
- 揭示物理机制的相对重要性

### 例子：N-S方程无量纲化

定义无量纲变量：$\vec{v}^* = \vec{v}/U$，$p^* = p/(\rho U^2)$，$\vec{x}^* = \vec{x}/L$，$t^* = tU/L$

代入N-S方程后自然出现：

$$\frac{1}{Re}\nabla^{*2}\vec{v}^* \quad \text{和} \quad \frac{1}{Fr^2}\hat{g}$$

- $Re$ 大 → 粘性项可忽略
- $Fr$ 大 → 重力项可忽略

---

## 6. 量纲分析（Dimensional Analysis）

### 说人话

你知道结果取决于哪些变量，但不知道具体公式长什么样。量纲分析帮你：
1. 把 $n$ 个变量缩减为 $n-k$ 个无量纲组（$k$ = 基本量纲数，通常3）
2. 大大减少实验工作量

---

## 7. Buckingham Pi定理（白金汉Π定理）

### 定理内容

如果一个物理问题涉及 $n$ 个变量，这些变量包含 $k$ 个基本量纲（通常M、L、T共3个），则可以组成 $n - k$ 个独立的无量纲 $\Pi$ 参数。

$$f(q_1, q_2, ..., q_n) = 0 \quad \Rightarrow \quad \phi(\Pi_1, \Pi_2, ..., \Pi_{n-k}) = 0$$

### Pi定理的操作步骤

> 💡 **这是考试重点！按步骤来就不会错。**

**Step 1**：列出所有相关变量（$n$ 个）

例：球体阻力 $F_D$ 取决于 $v, D, \rho, \mu$（共5个变量）

**Step 2**：确定基本量纲数 $k$

写出所有变量的量纲，看有几个独立量纲。通常 $k = 3$（M, L, T）

**Step 3**：选择 $k$ 个"重复变量"

选择要求：
- 包含所有基本量纲
- 彼此之间不能组成无量纲组
- 通常选 $\rho$、$v$、$D$（密度、速度、长度各取一个）

**Step 4**：逐个构造 $\Pi$ 参数

对每个剩余变量，与重复变量组合使量纲为零：

$$\Pi_i = q_i \cdot \rho^a \cdot v^b \cdot D^c$$

解方程组使 $[M^0 L^0 T^0]$

**Step 5**：写出最终关系

$$\Pi_1 = \phi(\Pi_2, \Pi_3, ...)$$

## 8. Pi定理完整例题——球体阻力

### 问题

球体在流体中运动的阻力 $F_D$ 取决于哪些无量纲参数？

### Step 1：列变量

$F_D, v, D, \rho, \mu$ → $n = 5$

### Step 2：量纲矩阵

| 变量 | M | L | T |
|------|---|---|---|
| $F_D$ | 1 | 1 | -2 |
| $v$ | 0 | 1 | -1 |
| $D$ | 0 | 1 | 0 |
| $\rho$ | 1 | -3 | 0 |
| $\mu$ | 1 | -1 | -1 |

$k = 3$（M, L, T），所以 $\Pi$ 参数个数 = $5 - 3 = 2$

### Step 3：选重复变量

选 $\rho, v, D$（包含M、L、T）

### Step 4：构造Π₁

$$\Pi_1 = F_D \cdot \rho^a \cdot v^b \cdot D^c$$

量纲：$[MLT^{-2}] \cdot [ML^{-3}]^a \cdot [LT^{-1}]^b \cdot [L]^c = M^0L^0T^0$

- M: $1 + a = 0 \Rightarrow a = -1$
- T: $-2 - b = 0 \Rightarrow b = -2$
- L: $1 - 3a + b + c = 0 \Rightarrow 1 + 3 - 2 + c = 0 \Rightarrow c = -2$

$$\Pi_1 = \frac{F_D}{\rho v^2 D^2}$$

这就是**阻力系数** $C_D$（差一个常数 $\frac{1}{2}$ 和面积 $A$）！

### Step 4：构造Π₂

$$\Pi_2 = \mu \cdot \rho^a \cdot v^b \cdot D^c$$

量纲：$[ML^{-1}T^{-1}] \cdot [ML^{-3}]^a \cdot [LT^{-1}]^b \cdot [L]^c = M^0L^0T^0$

- M: $1 + a = 0 \Rightarrow a = -1$
- T: $-1 - b = 0 \Rightarrow b = -1$
- L: $-1 - 3a + b + c = 0 \Rightarrow -1 + 3 - 1 + c = 0 \Rightarrow c = -1$

$$\Pi_2 = \frac{\mu}{\rho v D} = \frac{1}{Re}$$

### Step 5：最终结论

$$\frac{F_D}{\rho v^2 D^2} = \phi\left(\frac{1}{Re}\right)$$

即：**阻力系数是雷诺数的函数**！

$$C_D = f(Re)$$

> 💡 **威力**：原本需要做5个变量的实验（工作量巨大），现在只需画一条 $C_D$ vs $Re$ 的曲线就全搞定了！

---

## 9. 球体阻力系数 vs 雷诺数（$C_D$ vs $Re$）

### 阻力系数定义

$$C_D = \frac{F_D}{\frac{1}{2}\rho v^2 A}$$

其中 $A = \frac{\pi D^2}{4}$（正面投影面积）

### $C_D$-$Re$ 曲线的特征

```
log(C_D)
  |
  |  \  ← Stokes区 (C_D = 24/Re)
  |   \
  |    \_________  ← 过渡区
  |              \
  |               \___ C_D ≈ 0.44  ← 牛顿区（阻力平方区）
  |                          \
  |                           ↓ 阻力危机（Re≈5×10⁵）
  |                          /
  |                    C_D ≈ 0.1
  |_________________________________ log(Re)
      1    10   100  10⁴   10⁵  10⁶
```

### 各区间特征

| 区间 | Re范围 | $C_D$ | 特征 |
|------|--------|--------|------|
| Stokes流 | Re < 1 | $24/Re$ | 粘性完全主导，$F_D = 3\pi\mu vD$ |
| 过渡区 | 1 < Re < 1000 | 逐渐下降 | 尾流开始形成 |
| 牛顿区 | $10^3 < Re < 5×10^5$ | ≈ 0.44 | 湍流尾流，阻力∝$v^2$ |
| 阻力危机 | $Re ≈ 5×10^5$ | 突降至~0.1 | 边界层转捩为湍流→分离点后移 |

### Stokes阻力公式（Re < 1）

$$F_D = 3\pi \mu v D$$

$$C_D = \frac{24}{Re}$$

> 💡 **应用**：雾滴沉降、血液中细胞运动、泥沙沉淀——都是低Re的Stokes流。

### 阻力危机（Drag Crisis）

$Re ≈ 5×10^5$ 时，$C_D$ 突然从0.44降到0.1左右。原因：
- 球体表面的边界层从层流变为湍流
- 湍流边界层能抵抗逆压梯度更远才分离
- 分离点后移 → 低压尾流区变小 → 压差阻力减小

> 💡 **经典应用**：高尔夫球表面的小凹坑就是为了提前触发边界层转捩，让 $C_D$ 处在低值区，飞得更远！

## 10. 沉降速度（Terminal Velocity）

当球体在流体中自由下落达到匀速时：

$$\text{重力} = \text{浮力} + \text{阻力}$$

$$\frac{\pi D^3}{6}(\rho_s - \rho_f)g = \frac{1}{2}\rho_f v_t^2 \cdot \frac{\pi D^2}{4} \cdot C_D$$

对于Stokes流（$Re < 1$）：

$$v_t = \frac{(\rho_s - \rho_f)gD^2}{18\mu}$$

## 本章核心公式汇总

| 公式 | 含义 |
|------|------|
| $\Pi$ 参数个数 = $n - k$ | Buckingham Pi定理 |
| $Re_m = Re_p$ | 粘性主导时的动力相似条件 |
| $Fr_m = Fr_p$ | 重力主导时的动力相似条件 |
| $C_D = F_D / (\frac{1}{2}\rho v^2 A)$ | 阻力系数定义 |
| $C_D = 24/Re$ | Stokes阻力（Re<1） |
| $F_D = 3\pi\mu vD$ | Stokes阻力力 |
| $v_t = \frac{(\rho_s-\rho_f)gD^2}{18\mu}$ | Stokes沉降速度 |

## Pi定理解题检查清单

1. 列出所有变量（别漏！）
2. 写出每个变量的MLT量纲
3. 确定 $k$（通常=3）
4. 选好重复变量（包含全部基本量纲，常选 $\rho, v, L$）
5. 逐个构造 $\Pi$，列量纲方程解 $a, b, c$
6. 检验：$\Pi$ 确实无量纲？
7. 写出函数关系 $\Pi_1 = f(\Pi_2, ...)$
