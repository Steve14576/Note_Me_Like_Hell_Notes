# 第10章：稳态流动能量方程、泵/涡轮机效率

---

## 1. 稳态流动能量方程（Steady Flow Energy Equation, SFEE）

### 说人话

伯努利方程的"豪华升级版"。伯努利只管理想流体（无损失、无机械做功）。SFEE 把**水头损失**和**泵/涡轮机**都加进来了。

### 公式

$$\frac{p_1}{\rho g} + \frac{v_1^2}{2g} + z_1 + h_p = \frac{p_2}{\rho g} + \frac{v_2^2}{2g} + z_2 + h_T + h_L$$

各项含义：
- 左边：截面1的能量 + 泵输入的能量
- 右边：截面2的能量 + 涡轮机提取的能量 + 损失的能量

| 符号 | 名称 | 含义 |
|------|------|------|
| $h_p$ | 泵水头 | 泵给流体增加的能量（以水头表示） |
| $h_T$ | 涡轮机水头 | 涡轮机从流体提取的能量 |
| $h_L$ | 水头损失 | 粘性摩擦等导致的不可逆损失 |

### 和伯努利方程的关系

- 没有泵（$h_p = 0$）、没有涡轮机（$h_T = 0$）、没有损失（$h_L = 0$）→ 就退化为伯努利方程
- 实际工程中 $h_L$ 不能忽略

## 2. 功率（Power）

### 泵的水功率（流体获得的功率）

$$\dot{W}_{水} = \rho g Q h_p$$

- $Q$：体积流量
- $h_p$：泵水头（泵给流体增加的水头）

### 涡轮机的水功率（流体输出的功率）

$$\dot{W}_{水} = \rho g Q h_T$$

### 单位换算提醒

- 1 kW = 1000 W
- 1 马力（hp）= 746 W

## 3. 效率（Efficiency）

### 说人话

效率就是"有用的输出 / 总的输入"。能量在每一步传递中都有损失。

### 泵的效率

```
电能 → 电动机 → 机械能 → 泵 → 水的能量
     η_motor      η_pump
```

**泵效率**（泵本身把机械能转化为水能的效率）：

$$\eta_{pump} = \frac{\dot{W}_{水}}{\dot{W}_{轴}} = \frac{\rho g Q h_p}{\dot{W}_{shaft}}$$

**电动机效率**：

$$\eta_{motor} = \frac{\dot{W}_{轴}}{\dot{W}_{电}}$$

**组合效率**（从电到水）：

$$\eta_{overall} = \eta_{motor} \times \eta_{pump} = \frac{\rho g Q h_p}{\dot{W}_{电}}$$

### 涡轮机/发电机效率

```
水的能量 → 涡轮机 → 机械能 → 发电机 → 电能
          η_turbine        η_generator
```

**涡轮机效率**：

$$\eta_{turbine} = \frac{\dot{W}_{轴}}{\dot{W}_{水}} = \frac{\dot{W}_{shaft}}{\rho g Q h_T}$$

**发电机效率**：

$$\eta_{generator} = \frac{\dot{W}_{电}}{\dot{W}_{轴}}$$

**组合效率**（从水到电）：

$$\eta_{overall} = \eta_{turbine} \times \eta_{generator} = \frac{\dot{W}_{电}}{\rho g Q h_T}$$

## 4. 解题套路

### 典型题目：水泵把水从低处抽到高处

已知：流量Q，高度差$\Delta z$，管路损失$h_L$，泵效率$\eta$

求：需要多大功率的泵？

**步骤**：

1. 列能量方程（1=低处水面，2=高处水面）：
   $$0 + 0 + z_1 + h_p = 0 + 0 + z_2 + h_L$$
   $$h_p = (z_2 - z_1) + h_L = \Delta z + h_L$$

2. 水功率：$\dot{W}_{水} = \rho g Q h_p$

3. 实际需要的输入功率：$\dot{W}_{输入} = \frac{\dot{W}_{水}}{\eta_{overall}}$
