# 第13章：管道湍流、湍流摩擦系数、管路系统损失

---

## 1. 管道中的湍流（Turbulent Flow in Pipes）

### 说人话

层流时速度分布是完美抛物线（中心最快，壁面为零）。湍流时因为混合剧烈，速度分布变得**更"平坦"**——大部分区域速度差不多，只有靠近壁面的薄层速度急剧下降。

### 湍流速度剖面——幂律公式

$$u(r) = u_{max}\left(1 - \frac{r}{R}\right)^{1/n}$$

- $n$ ≈ 7（经验值，$Re$ 越大 $n$ 越大）
- 比层流的抛物线要"钝"得多

### 平均速度与最大速度

- 层流：$\bar{v} = u_{max}/2$（差一倍）
- 湍流（n=7）：$\bar{v} \approx 0.82 \cdot u_{max}$（接近最大速度）

> 💡 **实际意义**：湍流时用皮托管测中心速度，近似就是平均速度（误差小）；层流时差一倍，不能这样估。

## 2. 湍流摩擦系数（Turbulent Flow Friction Factor）

### Darcy-Weisbach方程（通用，层流湍流都用）

$$h_L = f \frac{L}{D} \frac{v^2}{2g}$$

关键是确定 $f$：

### 层流（回顾）

$$f = \frac{64}{Re}$$

### 湍流——取决于Re和粗糙度

**(1) 光滑管（Blasius公式，$Re < 10^5$）**

$$f = \frac{0.316}{Re^{1/4}}$$

**(2) 粗糙管（完全湍流区/阻力平方区）**

$$f = \frac{1}{[-2.0\log(\varepsilon/D / 3.7)]^2}$$

此时 $f$ 只与相对粗糙度 $\varepsilon/D$ 有关，与 $Re$ 无关！

**(3) 过渡区（Colebrook方程，隐式）**

$$\frac{1}{\sqrt{f}} = -2.0\log\left(\frac{\varepsilon/D}{3.7} + \frac{2.51}{Re\sqrt{f}}\right)$$

需要迭代求解，或者查**Moody图**。

### Moody图的使用

```
纵轴：f（摩擦系数）
横轴：Re（对数坐标）
参数线：不同的 ε/D（相对粗糙度）

使用步骤：
1. 算Re
2. 确定ε/D（查管材粗糙度表）
3. 在Moody图上找到交点→读出f
```

> 💡 **关键规律**：
> - 层流区：所有线重合成一条（$f = 64/Re$）
> - 过渡区：$f$ 同时取决于 $Re$ 和 $\varepsilon/D$
> - 完全湍流区：线变水平（$f$ 只取决于 $\varepsilon/D$）

### 常见管材粗糙度 $\varepsilon$

| 管材 | $\varepsilon$ (mm) |
|------|-------------------|
| 玻璃、塑料 | 0（光滑） |
| 铜管、黄铜 | 0.0015 |
| 钢管（新的） | 0.045 |
| 铸铁管 | 0.26 |
| 混凝土 | 0.3~3.0 |
| 锈钢管 | 2.0 |

## 3. 管路系统中的损失（Losses in Pipe Systems）

### 总水头损失 = 沿程损失 + 局部损失

$$h_L = h_f + \sum h_j = f\frac{L}{D}\frac{v^2}{2g} + \sum \zeta_i \frac{v_i^2}{2g}$$

### 局部损失的通用表达

$$h_j = \zeta \frac{v^2}{2g}$$

$\zeta$ = 局部阻力系数（无量纲），取决于几何形状。

---

## 4. 突扩损失（Losses in a Sudden Expansion）

### 说人话

管子突然变粗，流体从小管射入大管后来不及填满整个截面，在角落形成大量涡旋，能量被白白耗散。

### Borda-Carnot公式（精确理论解）

$$h_{突扩} = \frac{(v_1 - v_2)^2}{2g}$$

用面积比表示：

$$\zeta_{突扩} = \left(1 - \frac{A_1}{A_2}\right)^2 = \left(1 - \frac{d_1^2}{d_2^2}\right)^2$$

（以上游速度 $v_1$ 为基准）

### 特例：管道出口（排入大水池）

$A_2 → ∞$，所以 $\zeta = 1.0$

$$h_{出口} = \frac{v^2}{2g}$$

**全部动能都损失掉了**！

> 💡 **物理理解**：水从管子喷入大水池，速度从 $v$ 变为 ≈0，这些动能全部变成热被耗散。

---

## 5. 突缩损失（Losses in a Sudden Contraction）

$$\zeta_{突缩} \approx 0.5\left(1 - \frac{A_2}{A_1}\right)$$

（以下游速度 $v_2$ 为基准）

### 特例：管道入口（从大水池进入管道）

$A_1 → ∞$，所以 $\zeta \approx 0.5$

$$h_{入口} = 0.5 \frac{v^2}{2g}$$

> 💡 **对比**：出口损失系数=1.0，入口损失系数=0.5。所以突扩比突缩损失大！

### 改善方法

- 尖锐入口 $\zeta = 0.5$
- 圆角入口 $\zeta = 0.04$（减少90%！）
- 锥形渐扩代替突扩

---

## 6. 弯管损失（Losses in Pipe Bends）

### 说人话

管子拐弯处，流体要改变方向→产生离心力→内侧分离→形成二次流/涡旋→耗散能量。

### 弯管损失系数

$$h_{弯} = \zeta_{弯} \frac{v^2}{2g}$$

$\zeta_{弯}$ 取决于：
- 弯曲角度（90°、45°、180°）
- 相对弯曲半径 $R/D$（$R$ = 弯管中心线曲率半径）
- 管道表面粗糙度

### 典型值

| 弯管类型 | $\zeta$ |
|---------|---------|
| 90° 尖角弯头 | 1.1 |
| 90° 标准弯头 | 0.9 |
| 90° 长半径弯头（R/D=1.5） | 0.6 |
| 90° 弯管（R/D≥2） | 0.2~0.4 |
| 45° 弯头 | 0.4 |

> 💡 **设计原则**：弯曲半径越大，损失越小。能用大R弯管就不要用尖角弯头。

## 8. 等效管长法（另一种处理局部损失的方法）

将局部损失等效为一段直管的沿程损失：

$$\zeta \frac{v^2}{2g} = f\frac{L_{eq}}{D}\frac{v^2}{2g}$$

$$L_{eq} = \frac{\zeta D}{f}$$

然后总损失变成：

$$h_L = f\frac{L + \sum L_{eq}}{D}\frac{v^2}{2g}$$

---

## 本章核心公式汇总

| 公式 | 含义 |
|------|------|
| $h_L = f\frac{L}{D}\frac{v^2}{2g}$ | Darcy-Weisbach（沿程损失） |
| $f = 0.316/Re^{1/4}$ | Blasius（光滑管湍流） |
| $h_{突扩} = (v_1-v_2)^2/(2g)$ | 突扩损失（Borda-Carnot） |
| $\zeta_{突扩} = (1-A_1/A_2)^2$ | 突扩阻力系数 |
| $\zeta_{突缩} ≈ 0.5(1-A_2/A_1)$ | 突缩阻力系数 |
| $h_j = \zeta \cdot v^2/(2g)$ | 局部损失通用公式 |
| $h_{出口} = v^2/(2g)$ | 管道出口损失（ζ=1） |
| $h_{入口} = 0.5 v^2/(2g)$ | 尖锐入口损失（ζ=0.5） |

## 解题套路

1. 画出管路系统，标出所有管段和管件
2. 列能量方程（SFEE）：选上游和下游的大水面或已知点
3. 沿程损失：$f\frac{L}{D}\frac{v^2}{2g}$（算Re→查f）
4. 局部损失：逐个管件查 $\zeta$，加总
5. 总损失 = 沿程 + 局部代入能量方程求解
