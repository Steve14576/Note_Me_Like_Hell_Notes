# 第9章：线性动量方程、射流冲击、Reaction of a Jet

---

## 1. 线性动量方程（Linear Momentum Equation）

### 说人话

牛顿第二定律 $F = ma$ 用在流体上。只不过流体一直在流动，所以不是"质量×加速度"，而是"单位时间内动量的变化"。

### 公式（稳态流动）

$$\sum \vec{F} = \dot{m}(\vec{v}_{out} - \vec{v}_{in}) = \rho Q (\vec{v}_{out} - \vec{v}_{in})$$

- $\sum \vec{F}$：作用在控制体上的所有外力之和
- $\dot{m} = \rho Q$：质量流量
- $\vec{v}_{out}$：流出控制体的速度
- $\vec{v}_{in}$：流入控制体的速度

**注意：这是矢量方程！** 要分x、y方向分别列。

### 分量形式

$$\sum F_x = \rho Q (v_{2x} - v_{1x})$$
$$\sum F_y = \rho Q (v_{2y} - v_{1y})$$

### 外力包括哪些？

1. 压力（作用在控制体各截面上）
2. 重力
3. 固体壁面对流体的反力（这通常是我们要求的）

---

## 2. 动量方程的应用步骤

1. **选控制体**（CV）：把你关心的区域框起来
2. **画出所有进出截面**，标明速度方向
3. **确定坐标方向**（通常x取流动主方向）
4. **列出所有外力**（别忘了进出口截面的压力！）
5. **列动量方程**，分x、y方向

## 3. 射流冲击平板（Jet Impinging on a Plate）

### 情况1：射流垂直打在固定平板上

```
    射流 v₁ →  |  ← 平板
               |
              ↗ ↘  水向两侧散开
```

- 入口速度：$v_1$（水平向右）
- 出口速度：水沿板面散开，x方向分量 = 0

$$F_x = \rho Q (v_{2x} - v_{1x}) = \rho Q (0 - v_1) = -\rho Q v_1$$

板子对水的力是 $-\rho Q v_1$（向左推水），所以水对板子的力是 $+\rho Q v_1$（向右推板）。

$$F_{板} = \rho Q v_1 = \rho A v_1^2$$

### 情况2：射流打在倾斜平板上

板子与射流方向夹角 $\theta$：

$$F_{\perp 板} = \rho Q v_1 \sin\theta$$

### 情况3：射流打在弯曲叶片上（转过角度θ）

```
  v₁ →  ╔══╗  → v₂ (转了θ角)
         ╚══╝
```

$$F_x = \rho Q (v_2 \cos\theta - v_1)$$
$$F_y = \rho Q (v_2 \sin\theta - 0)$$

如果忽略损失，$|v_1| = |v_2|$（速度大小不变，只是方向变了）：

$$F_x = \rho Q v_1 (\cos\theta - 1)$$
$$F_y = \rho Q v_1 \sin\theta$$

**特殊情况**：射流被180°折返（U形弯）：

$$F_x = \rho Q ((-v_1) - v_1) = -2\rho Q v_1$$

力是垂直打板时的**两倍**！

## 4. 射流偏转 / Reaction of a Jet（射流反力）

### 说人话

消防水龙头喷水时，消防员要用很大力气才能抓住水管。火箭喷气往后，自己往前飞。这就是**射流反力**——动量方程的反向应用。

### 喷嘴的反力

水从管子喷出，管子/喷嘴受到的反力：

$$F_{反力} = \rho Q v_{出} + (p_{出} - p_{大气}) A_{出}$$

如果出口是大气压（自由射流）：

$$F_{反力} = \rho Q v_{出} = \rho A v_{出}^2$$

### 弯管中的流体力

弯管让流体转向，流体对弯管施加力。这在管道设计中非常重要（需要设计管道支撑）。

对于水平弯管（转过角度θ）：

$$F_x = p_1 A_1 - p_2 A_2 \cos\theta - R_x = \rho Q (v_2 \cos\theta - v_1)$$
$$F_y = -p_2 A_2 \sin\theta - R_y = \rho Q (v_2 \sin\theta - 0)$$

其中 $R_x$, $R_y$ 是管道支撑对流体的力。

---

## 5. 动量修正系数 $\beta$

### 说人话

上面的公式用的都是**平均速度**。但实际管道中速度分布不均匀（管中心快，壁面慢），用平均速度算动量会有误差。$\beta$ 就是修正这个误差的。

$$\beta = \frac{\int \rho v^2 dA}{\rho \bar{v}^2 A}$$

- 层流（抛物线速度分布）：$\beta = 4/3 ≈ 1.33$
- 湍流（速度分布比较平坦）：$\beta ≈ 1.01 ~ 1.05$

工程中湍流时通常取 $\beta = 1$。

---

## 本章核心公式

| 公式 | 含义 |
|------|------|
| $\sum F = \rho Q (v_{out} - v_{in})$ | 动量方程（矢量） |
| $F = \rho A v^2$ | 射流垂直打固定平板的力 |
| $F = 2\rho A v^2$ | 射流180°折返时的力 |
| $F = \rho Q v_1 (1 - \cos\theta)$ | 射流转过θ角的x方向力 |

---

## 解题关键

1. **选好控制体**是成功的一半
2. 动量方程是**矢量方程**，一定要注意正方向
3. 外力别忘了**压力项**（特别是入口出口截面的压力）
4. 求"固体受力"时，先求"流体受力"，再反向
