# 第4-5章：静力学力分析（Hydrostatics）与浮力

---

## 1. 静水压力的基本特性

### 说人话

静止的水里没有剪切力（因为流体定义就是"有剪切力就变形"，现在不动了说明没有）。所以静水中只有**压力**——垂直于表面、指向表面内部的力。

### 两个核心性质

1. **静水压力垂直于受力面，指向表面内部**
2. **同一点各方向压力相等**（帕斯卡定理的推论）

### 压力随深度变化

$$p = p_0 + \rho g h$$

这个公式永远是你的好朋友。意思就是：**往下走一米，压力增加 $\rho g$ × 1米**。

---

## 2. 压力的表示方法

### 绝对压力 vs 表压 vs 真空度

```
            ↑ 绝对压力
            |
            |  ← 大气压 p_atm (约101325 Pa)
            |
实际压力 →  |------- 表压 = p_abs - p_atm （正值）
            |
            |  如果压力低于大气压：
            |------- 真空度 = p_atm - p_abs （正值）
            |
            0 (绝对零压)
```

- **绝对压力** $p_{abs}$：以绝对真空为参考零点
- **表压** $p_{gauge}$：以大气压为参考零点，$p_{gauge} = p_{abs} - p_{atm}$
- **真空度**：当绝对压力 < 大气压时，$p_{vac} = p_{atm} - p_{abs}$

日常用的压力表测的都是**表压**。

## 3. 静水压力对平面的作用（Hydrostatic Force on Plane Surfaces）

### 问题

一个倾斜的平板浸在水中，水对它的总压力是多少？作用点在哪里？

### 总压力（合力大小）

$$F = \rho g \bar{h} A = p_C \cdot A$$

- $\bar{h}$：平面形心处的深度（从自由液面竖直往下量）
- $A$：平面面积
- $p_C$：形心处的压力

**说人话**：总压力 = 形心处压力 × 面积。就这么简单。

### 作用点（压力中心）

压力不是均匀分布的（下面比上面大），所以合力不作用在形心，而是偏下方一点的**压力中心**。

$$y_P = y_C + \frac{I_{xc}}{y_C \cdot A}$$

- $y_P$：压力中心到液面的倾斜距离
- $y_C$：形心到液面的倾斜距离
- $I_{xc}$：关于形心轴的惯性矩

或者用深度表示：

$$h_P = h_C + \frac{I_{xc} \sin^2\theta}{\bar{h} \cdot A}$$

**关键理解**：
- 压力中心**永远在形心之下**（因为 $I_{xc}$ 总是正的）
- 平板沉得越深，$y_C$ 越大，$\frac{I_{xc}}{y_C A}$ 越小，压力中心就越接近形心

### 常用形状的惯性矩

| 形状 | $I_{xc}$（关于形心轴） |
|------|----------------------|
| 矩形（宽b，高d） | $\frac{bd^3}{12}$ |
| 圆形（直径D） | $\frac{\pi D^4}{64}$ |
| 三角形（底b，高d） | $\frac{bd^3}{36}$ |

### 计算步骤（解题套路）

1. 确定液面位置，画出平面在液体中的姿态
2. 找到平面形心，算出形心深度 $\bar{h}$
3. 合力：$F = \rho g \bar{h} A$
4. 查表得 $I_{xc}$
5. 压力中心：$y_P = y_C + \frac{I_{xc}}{y_C A}$

## 4. 静水压力对曲面的作用

### 说人话

弯曲表面上每一小块的压力方向不同（都垂直于表面），所以不能直接积分。聪明的办法是分成**水平分力**和**竖直分力**分别算。

### 水平分力

$$F_H = \rho g \bar{h}_c A_v$$

- $A_v$：曲面在**竖直平面上的投影面积**
- $\bar{h}_c$：投影面形心的深度

**说人话**：水平分力等于"把曲面压扁到墙上"后按平面算的结果。

### 竖直分力

$$F_V = \rho g V$$

- $V$：曲面上方（或下方）的流体体积

**说人话**：竖直分力等于曲面"头顶上"那部分水的重量。

- 如果曲面上方有水 → 竖直分力向下
- 如果曲面下方有水而上方没有 → 竖直分力向上

### 合力

$$F = \sqrt{F_H^2 + F_V^2}$$

方向：$\tan\alpha = F_V / F_H$

## 5. 浮力（Buoyancy）

### 阿基米德原理

> 浸在流体中的物体受到的浮力等于它**排开流体的重量**。

$$F_B = \rho_f g V_{排}$$

- $\rho_f$：流体密度
- $V_{排}$：物体排开的流体体积（浸没体积）

### 浮力的方向

竖直向上，作用点在**排开流体体积的形心**处（称为浮心）。

### 物体沉浮判据

| 条件 | 结果 |
|------|------|
| $\rho_{物} < \rho_{液}$ | 漂浮（部分浸没） |
| $\rho_{物} = \rho_{液}$ | 悬浮（随遇平衡） |
| $\rho_{物} > \rho_{液}$ | 下沉 |

### 漂浮物体

对于漂浮物体，浮力 = 重力：

$$\rho_f g V_{排} = \rho_{物} g V_{物}$$

所以：

$$\frac{V_{排}}{V_{物}} = \frac{\rho_{物}}{\rho_f}$$

冰山浮在海水中大约有 90% 体积在水下（因为 $\rho_{冰}/\rho_{海水}$ ≈ 0.9）。

### 浮力稳定性

- **重心**在**浮心**之下 → 绝对稳定
- **重心**在**浮心**之上 → 需要看**定倾中心**（metacenter）
  - 定倾中心在重心之上 → 稳定
  - 定倾中心在重心之下 → 不稳定（翻船）

## 本章核心公式汇总

| 公式 | 含义 |
|------|------|
| $p = p_0 + \rho g h$ | 静水压力随深度增加 |
| $F = \rho g \bar{h} A$ | 平面上的静水总压力 |
| $y_P = y_C + \frac{I_{xc}}{y_C A}$ | 压力中心位置 |
| $F_H = \rho g \bar{h}_c A_v$ | 曲面水平分力 |
| $F_V = \rho g V$ | 曲面竖直分力 |
| $F_B = \rho_f g V_{排}$ | 浮力（阿基米德原理） |

---

## 解题技巧

1. **画图**！把液面、形心、压力中心都标在图上
2. 算平面压力时，先找形心深度，一切从形心出发
3. 算曲面压力时，记住"水平看投影，竖直看体积"
4. 浮力题先判断沉浮状态，再列平衡方程
