## 一、纯物质（Pure Substance）

### 1.1 定义

> **纯物质**：化学组成固定不变的物质。

不只是纯元素或化合物。空气虽然是混合物（氮气+氧气+…），但它的成分比例不随位置和时间变化，所以在热力学中**空气算纯物质**。水也是纯物质。

关键要求：化学组成**均匀且不变**。如果加了油和水（不互溶），就不是纯物质。

### 1.2 相（Phase）

> **相**：物质在空间中均匀分布的一种物理状态。

物质有三种常见相：**固态**、**液态**、**气态**。同一种物质在不同条件下可以处于不同相。

本篇主要研究液-气相变（水↔水蒸气），这是工程中最常见的相变。

## 二、相变五阶段——把水烧开的全过程

PPT 用活塞气缸中的水在恒压下加热为例，展示了相变的五个阶段。这是整篇课的核心模型，必须吃透。

> **假设条件**：活塞自由移动 → 压力 $P$ 全程保持恒定（等压过程）。

### 2.1 五个阶段

| 阶段   | 名称                                     | 发生了什么                 | 温度变化            | 体积变化 |
| ------ | ---------------------------------------- | -------------------------- | ------------------- | -------- |
| 1 → 2 | **过冷液体** (Compressed Liquid)   | 加热液体，温度上升         | 升高                | 微增     |
| 2      | **饱和液体** (Saturated Liquid)    | 液体刚好达到沸点，即将汽化 | =$T_{\text{sat}}$ | —       |
| 2 → 4 | **气液混合物** (Saturated Mixture) | 液体逐渐变成气体，气液共存 | **不变**      | 大增     |
| 4      | **饱和蒸气** (Saturated Vapour)    | 最后一滴液体刚好汽化完     | =$T_{\text{sat}}$ | —       |
| 4 → 5 | **过热蒸气** (Superheated Vapour)  | 继续加热气体，温度上升     | 升高                | 增       |

### 2.2 逐段详解

**阶段 1（过冷液体）**：水在常温下被加热。温度从比如 20°C 往上升。这时候水离沸腾还远，叫做"过冷液体"——温度低于当前压力下的沸点。这一阶段吸收的热量叫**显热**（sensible heat），因为它让温度"可感知地"升高。

**阶段 2（饱和液体）**：温度升到沸点（1 atm 下 = 100°C）。水刚好要开始汽化但还没开始。这个临界状态叫"饱和液体"。

**阶段 2→4（气液混合物）**：继续加热，水开始沸腾。水变成蒸汽，但**温度保持不变**！吸收的热量全部用于改变相态（液→气），而不是升高温度。这部分热量叫**潜热**（latent heat）。

> 这个阶段中，液气和气体共存。混合物中气体的比例从 0%（阶段2）逐渐增到 100%（阶段4）。

**阶段 4（饱和蒸气）**：所有液体都变成气体了。如果此时移除热量，气体就会开始凝结——所以叫"饱和蒸气"（即将凝结的蒸气）。

**阶段 5（过热蒸气）**：继续加热，气体温度继续上升。此时气体的温度高于沸点，叫做"过热蒸气"——即使移除一些热量也不会立刻凝结（要降到沸点才开始）。

### 2.3 显热 vs 潜热

|          | 显热 (Sensible Heat) | 潜热 (Latent Heat)                |
| -------- | -------------------- | --------------------------------- |
| 发生在   | 阶段 1→2 或 4→5    | 阶段 2→4                         |
| 温度变化 | 有（温度升高或降低） | **无**（温度恒定）          |
| 能量用途 | 提高分子运动动能     | 破坏/重建分子间作用力（改变相态） |

> 通俗说：显热让你"感觉"到变热了；潜热偷偷改变状态但温度计读数不变。

### 2.4 潜热的两种

| 类型     | 英文                        | 发生在 | 水的值 (1 atm) |
| -------- | --------------------------- | ------ | -------------- |
| 熔化潜热 | Latent heat of fusion       | 固↔液 | 333.7 kJ/kg    |
| 汽化潜热 | Latent heat of vaporization | 液↔气 | 2256.5 kJ/kg   |

> 汽化潜热远大于熔化潜热——把水变蒸汽比把冰变水需要多得多的能量。

### 2.5 相变时的能量去了哪里？

相变时 $T$ 不变、$P$ 不变，但 $v$ 在涨（液体变气体）。吸收的潜热 $Q$ 去了两个地方：

$$Q_{\text{latent}} = \Delta U + P \Delta V$$

| 去向 | 含义 | 占比（水 1 atm） |
|------|------|------|
| $\Delta U$ | 打断分子间吸引力（增加分子势能） | **大部分（~87%）** |
| $P \Delta V$ | 膨胀做边界功 | 小部分（~13%） |

> **关键误区纠正**：相变时 $T$ 不变，但 $\Delta U \neq 0$！
>
> 你可能觉得 $T$ 不变 → $\Delta U = c_v \Delta T = 0$——但 $\Delta U = c_v \Delta T$ **只对理想气体单相区有效**。相变时物质是气液混合物，不是理想气体。$T$ 不变只意味着**分子动能不变**，但打断分子间键需要能量（增加**势能**），所以 $U$ 是变化的。
>
> 一句话：潜热的大部分用来**断键**（$\Delta U$），小部分用来**膨胀做功**（$P\Delta V$）。

## 三、饱和温度与饱和压力

### 3.1 核心关系

> **饱和温度 $T_{\text{sat}}$**：在给定压力下，纯物质发生相变的温度（即沸点）。
>
> **饱和压力 $P_{\text{sat}}$**：在给定温度下，纯物质发生相变的压力。

关键点：**沸点不是固定的！它取决于压力。**

| 压力              | 沸点（饱和温度） | 场景                |
| ----------------- | ---------------- | ------------------- |
| 1 atm (101.3 kPa) | 100°C           | 海平面烧水          |
| 0.7 atm (~70 kPa) | ~90°C           | 海拔 3000 m 烧水    |
| 10 MPa            | 316°C           | 高压锅 / 蒸汽发电厂 |

> 高山上水不到 100°C 就开了——因为气压低，饱和温度降低。这就是为什么高海拔地区做饭要用高压锅。

### 3.2 $T_{\text{sat}}$ 和 $P_{\text{sat}}$ 是一一对应的

知道压力就能确定沸点，知道温度就能确定沸点压力。它们是一一对应的，不是独立的。

> **这解释了为什么 T-v 图和 P-v 图的平线段是同一件事**：
>
> 相变时 $P$ 和 $T$ **同时被锁死**——锁定 $P$ 则 $T$ 自动锁在 $T_{\text{sat}}(P)$，锁定 $T$ 则 $P$ 自动锁在 $P_{\text{sat}}(T)$。所以：
> - 等压加热到 $T_{\text{sat}}$ → $T$ 被锁死 → T-v 图出现平线
> - 等温减压到 $P_{\text{sat}}$ → $P$ 被锁死 → P-v 图也出现平线
>
> **两种平线段是同一种相变，只是进入方式不同。进了平线段后 $P$ 和 $T$ 都不变，$v$ 在涨，物理过程完全一样。**

> 这意味着在饱和状态下（阶段 2→4），$T$ 和 $P$ 不是两个独立性质——知道一个就知道另一个。要用**其他性质**（如 $v$ 或 $x$）来确定具体状态。

## 四、T-v 图和 P-v 图——相变的可视化

### 4.1 从初中的 T-t 图说起

你初中学过 T-t 图（温度-时间图）：横轴是时间，纵轴是温度，加热一块冰的曲线是"斜→平→斜→平→斜"：

```
T ↑
  |        /  蒸汽升温
  |       /
  |──────/───  沸腾（100°C，温度不变，吸潜热）
  |     /
  |    /    水升温
  |   /
  |──/──────  融化（0°C，温度不变，吸潜热）
  | /
  |/  冰升温
  └──────────────→ t（时间）
```

**平线 = 相变阶段**（温度不变，吸的热全部用于改变相态），这是你已有的认知。大学的图保留了这个核心概念，但横轴变了。

### 4.2 T-v 图——把横轴从"时间"换成"比容"

> **T-v 图**：纵轴还是温度 $T$，但横轴从时间 $t$ 变成了**比容 $v$**（每千克物质占多大空间）。

**为什么横轴从时间换成比容？**

因为时间跟物质状态没有直接关系——加热快慢取决于火力大小，但物质在什么温度沸腾、体积胀多少，跟时间无关。比容 $v$ 才是物质的状态属性（第一篇讲过：状态由强度性质确定）。

**形状为什么还是"斜→平→斜"？**

因为在等压加热中，时间 $t$ 和比容 $v$ 都在单调增加：
- 液体升温阶段：温度升，体积微胀 → $v$ 缓慢增大 → 斜线
- 相变阶段：温度不变，液体变气体 → $v$ 大幅增大 → 平线（T 不变但 $v$ 在涨）
- 气体升温阶段：温度升，气体膨胀 → $v$ 继续增大 → 斜线

所以 T-v 图上**单条等压线**的形状跟 T-t 图类似——都是"斜平斜"。**平线的意义完全一样**：相变阶段，温度不变，吸收的热是潜热。

**物理场景**：等压加热 = 活塞气缸中活塞上面压着重物（压力恒定），加热气体让它膨胀。也可以是开口杯子烧水（压力 = 1 atm，恒定）。**关键：压力固定，体积（比容）可以变。**

```
T ↑
  |          阶段4→5（过热蒸气，温度升）
  |         /
  |        /  阶段2→4（混合物，温度不变，v大增）← 平线 = 潜热
  |   ────/────  ← T_sat（饱和温度）
  |  /
  | / 阶段1→2（过冷液体，温度升，v微增）
  |/
  └──────────────────────→ v（比容，不是时间！）
```

> **注意：这条"斜平斜"是 T-v 图上的等压线。后面讲 P-v 图时等压线长得完全不同——是水平线！** 不要搞混。

> **T-t vs T-v 的核心区别**：T-t 只是一次实验的一条线；T-v 可以画**多条不同压力的等压线**，这就是钟形的来源。

### 4.3 钟形是怎么来的——把多条等压线的端点连起来

T-v 图上画多条不同压力的等压加热线：

```
T ↑
  |        ╱ 临界点
  |       ╱ ╲
  |      ╱   ╲     ← 更高压力的等压线（平段更短、更高）
  |     ╱ ──── ╲
  |    ╱ ──────── ╲  ← 更低压力的等压线（平段更长、更低）
  |   ╱            ╲
  |  ╱  饱和液体线  ╲  饱和蒸气线
  | ╱                ╲
  └────────────────────→ v
```

**怎么画的**：
1. 画 1 atm 的等压线：斜→平→斜，平线在 100°C
2. 画 10 MPa 的等压线：斜→平→斜，平线在 316°C（压力高→沸点高），平线段更短（液气比容差变小）
3. 画更多压力的等压线……
4. 把所有平线段的**左端点**（饱和液体点）连起来 → **饱和液体线**（钟形左半边）
5. 把所有平线段的**右端点**（饱和蒸气点）连起来 → **饱和蒸气线**（钟形右半边）
6. 压力越高，平线段越短，最后缩成一个点 → **临界点**（钟形顶点）

> **钟形不是一条线画出来的，是所有等压线端点的包络线。** 钟形内部 = 气液混合区，左边 = 纯液体，右边 = 纯气体。

### 4.4 P-v 图——把纵轴从温度换成压力

> **P-v 图**：纵轴是压力 $P$，横轴还是比容 $v$。

把 T-v 图的纵轴从 $T$ 换成 $P$，钟形还在（饱和液体线和饱和蒸气线不变），但等压线变了：

| | T-v 图 | P-v 图 |
|---|---|---|
| 纵轴 | 温度 $T$ | 压力 $P$ |
| 等压线形状 | 斜→平→斜（曲线） | **水平线**（因为纵轴就是 $P$，$P$ 不变 = 水平） |
| 钟形 | 有 | 有（**同一批物理点，纵轴换了**） |

> **P-v 钟形跟 T-v 钟形是同一批点画出来的**：每个压力下都有一个饱和液体点（$v_f$）和饱和蒸气点（$v_g$）。T-v 图用 $T$ 当纵轴画这些点，P-v 图用 $P$ 当纵轴画**同一批点**。连端点、包络，得到同样的钟形。不用管理想气体方程——钟形内部是气液混合物，不是理想气体。

在 P-v 图上：
- 等压线 = 水平线（**不是"斜平斜"！P-v 图的纵轴就是 P，P 不变 = 水平线**）
- 水平线穿过钟形内部 = 混合区（相变段）
- 水平线在钟形左边 = 纯液体
- 水平线在钟形右边 = 纯气体
- **水平线在钟形内的长度 = 相变过程中比容的变化量**

**T-v vs P-v 等压线对比**（同一条等压线在两个图上的样子）：

```
T-v 图（纵轴 T）                    P-v 图（纵轴 P）
T ↑                                 P ↑
  |       / 蒸气升温                   |══════════════════ 等压线=水平线
  |      /                            |    ╱钟形╲
  |─────/── 平=相变(T不变)            |═══╱══════════╲═══ ← 同一条等压线
  |   /                              |  ╱  相变段    ╲
  |  / 液体升温                       | ╱ (钟形内部)   ╲
  └────────→ v                       └───────────────→ v
  等压线 = 斜平斜                      等压线 = 水平线
```

### 4.5 饱和液体线与饱和蒸气线——钟形的物理意义

钟形的左边界叫**饱和液体线**（$x = 0$，100% 液体），右边界叫**饱和蒸气线**（$x = 1$，100% 气体）。这两条线围出来的区域就是混合区。

> **钟形不只是数学上的包络线，它有明确的物理意义——相变边界：**
> - 钟形**内部** = 气液两相共存（正在相变中）
> - 钟形**左外** = 纯液体（还没开始汽化）
> - 钟形**右外** = 纯气体（已经全部汽化）
> - 你的过程线**穿过钟形边界** = 开始或结束相变

对于一条等压加热线：
- **T-v 图**：斜（液体升温）→ 平（相变，在钟形内部）→ 斜（气体升温），钟形是背景
- **P-v 图**：水平线穿过钟形内部的那段 = 相变段，钟形外 = 纯单相

**等温线在 P-v 图上长什么样？**

除了等压线（水平线），P-v 图上还能画**等温线**（constant T 曲线）。等温过程 = 温度不变，减小压力让体积膨胀：

```
P ↑
  |  纯液体
  | ╱
  |●  ← 减压到 P_sat(T)，液体开始汽化
  |━━━━━━ ← 等温线在钟形内变成水平段（P 和 T 都不变，v 在涨）
  |     ●  ← 全部变成气体
  |      ╲ ← 继续减压，气体膨胀
  |       ╲ 纯气体
  └──────────→ v
```

> PPT 蓝框说的就是这个："Pressure reduces until liquid starts to convert to vapour"（减压到开始汽化）和"Pressure reduces until all liquid converts to vapour"（减压到全部汽化）。

**相变的本质：路径穿过钟形边界**

不管是"温致"（等压加热）还是"压致"（等温减压），相变都发生在**过程路径穿过钟形边界**的那一刻：
- 等压加热（T-v 图斜平斜）：温度升到 $T_{\text{sat}}$ → 穿过钟形左边界 → 相变
- 等温减压（P-v 图等温线）：压力降到 $P_{\text{sat}}$ → 穿过钟形顶部边界 → 相变

> **热力学状态没有"惯性"**：物质的状态只取决于当前的 $P, T, v$，跟怎么到达的无关。P-v 图是所有可能状态的"地图"，过程 = 地图上的一条路径。你可以任意想象路径，只要穿过钟形边界就发生相变。

### 4.6 临界点——平线段缩到零后会怎样？

> **临界点**：饱和液体线和饱和蒸气线交汇的顶点。此时 $v_f = v_g$（饱和液体和饱和蒸气比容相同），液态和气态**无法区分**。

水的临界点：$T_{\text{cr}} = 374°C$，$P_{\text{cr}} = 22.1$ MPa。

**超过临界点后会发生什么？**

平线段缩到零 = 临界点。再往上（压力 > $P_{\text{cr}}$）：
- T-v 图上：等压线变成**光滑曲线**，没有"斜平斜"了——**没有相变，没有潜热，没有沸腾**，从"液体状"到"气体状"是连续渐变
- P-v 图上：水平线在钟形**上方**，根本不穿过钟形——全程都是同一种东西
- 钟形到临界点就**结束了**，上方没有钟形

> 通俗比喻：正常烧水到 100°C 会冒气泡（相变，有平线段）。但超过临界点，加热流体它从"像液体"慢慢变成"像气体"，**全程不沸腾**——就像调色从蓝渐变到红，中间没有跳变。
>
> 超过临界点的物质叫**超临界流体**，性质介于液体和气体之间。了解即可，考试不深究。

### 4.7 三相点（Triple Point）

> **三相点**：固、液、气三相共存的唯一温度和压力点。

水的三相点：$T = 0.01°C$，$P = 0.611$ kPa。

> PPT 还展示了包含固相的 P-v 图。大多数物质凝固时收缩（固体比液体密度大），但水是例外——凝固时膨胀（冰比水轻），所以冰浮在水面上。了解即可。

## 五、焓（Enthalpy, h）——提前定义

PPT 在这一章引入了焓，因为物性表里有焓的值。但焓的完整物理意义要到第五篇（开口系统）才展开。这里先给出定义和基本解释。

### 5.1 定义

$$
h = u + P \cdot v
$$

- $h$ = 比焓 [kJ/kg]（小写 h）
- $u$ = 比内能 [kJ/kg]（小写 u）
- $P$ = 压力 [kPa]
- $v$ = 比容 [m³/kg]

> **这是什么意思？** $u$ 是内能（微观能量），$P \times v$ 叫做**流动功**（flow work）——把 1 kg 物质推过边界所需的功（压力 × 体积 = 力 × 面积 × 距离 = 功）。
>
> 焓 = 内能 + 流动功。它代表的是：1 kg 物质在进入/离开一个开口系统时携带的总能量。

### 5.2 为什么现在就要用？

因为物性表中除了列 $u$（比内能）和 $v$（比容），还列了 $h$（比焓）。查表时直接读 $h$ 比自己算 $u + Pv$ 方便。

> **现在你只需要知道**：$h$ 是一个为了方便而定义的组合量，单位 kJ/kg，物性表里有现成的值。完整的物理意义在第五篇讲开口系统时会展开。

## 六、干度（Quality, x）——混合物中气体的比例

### 6.1 定义

> **干度 $x$**：气液混合物中，气相质量占总质量的比例。

$$
x = \frac{m_{\text{vapour}}}{m_{\text{total}}} = \frac{m_g}{m_f + m_g}
$$

- $x = 0$：纯饱和液体（0% 气体）
- $x = 1$：纯饱和蒸气（100% 气体）
- $0 < x < 1$：气液混合物

> 下标记号说明（物性表标准记号，必须记住）：
>
> | 下标 | 来自 | 含义 | 在钟形上的位置 |
> |------|------|------|--------------|
> | $f$ | **f**luid | 饱和**液体** | 钟形左边界（$x=0$） |
> | $g$ | **g**as | 饱和**蒸气** | 钟形右边界（$x=1$） |
> | $fg$ | $g - f$ | 从液到气的**差值** | 相变过程的变化量 |

### 6.2 用干度算混合物的平均性质

混合物的比容、比内能、比焓都可以用干度在饱和液体值和饱和蒸气值之间插值：

$$
v = v_f + x \cdot (v_g - v_f) = v_f + x \cdot v_{fg}
$$

$$
u = u_f + x \cdot (u_g - u_f) = u_f + x \cdot u_{fg}
$$

$$
h = h_f + x \cdot (h_g - h_f) = h_f + x \cdot h_{fg}
$$

其中 $v_{fg} = v_g - v_f$，$u_{fg} = u_g - u_f$，$h_{fg} = h_g - h_f$（下标 $fg$ 表示从 $f$ 到 $g$ 的差值）。

> **怎么理解这个公式？** 混合物中 $x$ 比例是气体，$(1-x)$ 比例是液体。平均值 = 液体部分 + 气体部分 = $(1-x)v_f + x \cdot v_g = v_f + x(v_g - v_f)$。
>
> **PPT 里的 $v_{\text{avg}}$ 是什么？** 就是这里的 $v$——当前混合物的实际比容。PPT 写 "avg" 是因为它确实是两相的质量分数加权平均，但容易让人误以为是"时间平均"或"最大最小平均"。它既不是时间平均，也不是 $v_f$ 和 $v_g$ 的算术平均，而是按质量分数（$x$ 和 $1-x$）加权。

> **为什么能插值？** 因为广延量（$V, U, H$）可加——总体积 = 液体体积 + 蒸气体积。两边除以总质量后，就变成按质量分数（$x$ 和 $1-x$）加权平均。$v, u, h$ 本身是强度性质，但它们的大写版本（$V, U, H$）是广延量，可加性来自那里。
>
> **任何态都有 $v, u, h$ 吗？** 有。$v = V/m$（任何物质都占空间有质量），$u = U/m$（分子运动/相互作用 → 有内能），$h = u + Pv$（定义出来的组合量）。它们是状态函数，任何平衡态都有确定值。气体和液体各自都有 $v, u, h$——物性表里的 $v_f, v_g, u_f, u_g, h_f, h_g$ 就是同一种物质在液态和气态各自的值。
>
> **$x$ 只是一个数学比率**：按质量分数把液相值和气相值加权平均，跟"30% 糖 + 70% 水算平均密度"是同一逻辑。前提是必须在两相区（两相共存才能混合）。
>
> **前提**：$v_f$ 和 $v_g$ 必须在**同一个饱和压力/温度**下查表，因为混合物中液气和气体处于平衡（同 $P$、同 $T$）。
>
> **什么不能插值？** $T$ 和 $P$ 不能用 $x$ 插值——相变时它们恒定不变，跟 $x$ 无关。

> **注意**：干度 $x$ 只在**气液混合区**有意义。在过冷液体区或过热蒸气区，不用 $x$。

## 七、物性表——怎么查值

### 7.1 为什么需要表？

物质的 $P, T, v, u, h$ 之间关系太复杂，没法用简单公式描述。所以科学家做了大量实验，把结果整理成表。查表是热力学的核心技能。

### 7.2 三种表

| 表名                                          | 已知条件               | 查什么                                              |
| --------------------------------------------- | ---------------------- | --------------------------------------------------- |
| **饱和温度表** (Sat. Temperature Table) | 温度$T$              | $P_{\text{sat}}, v_f, v_g, u_f, u_g, h_f, h_g$ 等 |
| **饱和压力表** (Sat. Pressure Table)    | 压力$P$              | $T_{\text{sat}}, v_f, v_g, u_f, u_g, h_f, h_g$ 等 |
| **过热表** (Superheated Table)          | 压力$P$ + 温度 $T$ | $v, u, h$ 等                                      |

> **用什么表取决于你处于哪个区域**：
>
> - 过冷液体区 → 用饱和温度表（温度已知），近似取 $h \approx h_f(T)$
> - 气液混合区 → 用饱和温度表或压力表（已知 $T$ 或 $P$），再用干度 $x$ 算
> - 过热蒸气区 → 用过热表（需要 $P$ 和 $T$ 两个独立参数）

### 7.3 查表流程

```
1. 你有什么？（T 或 P？）
2. 你在哪个区域？（过冷液体？混合物？过热蒸气？）
3. 选对应的表
4. 查出饱和液体值（下标 f）和饱和蒸气值（下标 g）
5. 如果是混合物，用干度 x 算平均值
```

**怎么判断在哪个区域？**

- 已知 $T$ 和 $P$：

  - 查饱和表，看 $T$ 对应的 $P_{\text{sat}}$（或 $P$ 对应的 $T_{\text{sat}}$）
  - 如果 $P > P_{\text{sat}}(T)$ → 过冷液体（压力高于沸点压力，还没到沸点）
  - 如果 $P < P_{\text{sat}}(T)$ → 过热蒸气
  - 如果 $P = P_{\text{sat}}(T)$ → 饱和状态（可能是饱和液体、饱和蒸气或混合物）
- 已知 $T$（或 $P$）和 $x$：

  - $0 < x < 1$ → 气液混合物
  - $x = 0$ → 饱和液体
  - $x = 1$ → 饱和蒸气
- 已知 $T$（或 $P$）和 $v$（或 $h, u$）：

  - 查饱和表得到 $v_f$ 和 $v_g$
  - 如果 $v < v_f$ → 过冷液体
  - 如果 $v_f < v < v_g$ → 混合物，用 $x = \frac{v - v_f}{v_g - v_f}$ 算干度
  - 如果 $v > v_g$ → 过热蒸气

### 7.4 过冷液体的近似

过冷液体的性质值随压力变化很小，所以经常用**同温度下的饱和液体值**来近似：

$$
h_{\text{subcooled}}(T, P) \approx h_f(T)
$$

> 即：过冷液体的焓 ≈ 同温度下饱和液体的焓。这在考试中经常用。

## 八、理想气体方程

### 8.1 方程

$$
P \cdot V = m \cdot R \cdot T
$$

或用比容表示（两边除以 $m$）：

$$
P \cdot v = R \cdot T
$$

- $P$ = 绝对压力 [Pa 或 kPa]
- $V$ = 体积 [m³]（大写 = 总体积）
- $v$ = 比容 [m³/kg]（小写 = 每千克体积）
- $m$ = 质量 [kg]
- $R$ = **气体常数** [kJ/(kg·K)]，每种气体不同
- $T$ = 绝对温度 [K]

### 8.2 气体常数 R——跟高中 $PV=nRT$ 的关系

你可能记得高中的理想气体方程是 $PV = nRT$（$n$ 是摩尔数），这里的 $R$ 跟大学版的 $R$ **不是同一个东西**：

|       | 高中版$PV = nRT$                               | 大学版$PV = mRT$          |
| ----- | ------------------------------------------------ | --------------------------- |
| 物量  | $n$ = 摩尔数 [mol]                             | $m$ = 质量 [kg]           |
| $R$ | 通用常数$R_u = 8.314$ J/(mol·K)，所有气体一样 | 特定常数$R$，每种气体不同 |

**高中的 $R$ 就是大学的 $R_u$**，只是叫法不同。

**为什么是除法 $R = R_u / M$？**

从高中版出发，代入 $n = m/M$（摩尔数 = 质量 ÷ 摩尔质量）：

$$
PV = n \cdot R_u \cdot T = \frac{m}{M} \cdot R_u \cdot T = m \cdot \underbrace{\frac{R_u}{M}}_{\text{这就是 } R} \cdot T
$$

$R_u / M$ 自然凑在一起，定义为新的常数 $R$。**除法的来源就是 $n = m/M$ 这个换算**——把"每摩尔"方程变成"每千克"方程。

**为什么要转成"每千克"版本？** 工程上称东西用 kg、查表查 kJ/kg、算功率用 kW，全是按质量来的。用摩尔版每次都要 kg↔mol 来回换算，麻烦。

$$
R = \frac{R_u}{M}
$$

- $R_u$ = **通用气体常数** = 8.314 kJ/(kmol·K)，对所有气体都一样（就是高中的 $R$）
- $M$ = 摩尔质量 [kg/kmol]

常见气体的 $R$ 值：

| 气体         | $R$ [J/(kg·K)] | $R$ [kJ/(kg·K)] |
| ------------ | ----------------- | ------------------ |
| 空气         | 287               | 0.287              |
| 氮气$N_2$  | 297               | 0.297              |
| 氧气$O_2$  | 260               | 0.260              |
| 氢气$H_2$  | 4124              | 4.124              |
| 氦气$He$   | 2080              | 2.080              |
| 甲烷$CH_4$ | 518               | 0.518              |

> **不要混淆 $R$ 和 $R_u$**：$R_u$ = 8.314 是通用常数（所有气体一样），$R$ 是特定气体的常数（每个气体不同）。

### 8.3 适用条件

理想气体方程是一个**理想化模型**，假设：

- 分子之间没有作用力
- 分子本身不占体积

**什么时候适用？**

- 低压、高温时适用（分子间距大，相互作用弱）
- 常温常压下空气、氮气、氧气等适用

**什么时候不适用？**

- 高压（特别是接近临界点）
- 饱和状态附近（液气混合区）

**水蒸气能不能当理想气体？**

- 压力 < 10 kPa：可以，误差 < 0.1%
- 空调中的水蒸气：可以（压力很低）
- 蒸汽发电厂：**不行**（压力太高），必须查表

### 8.4 两状态关系

对于固定质量的理想气体，如果经历一个过程从状态 1 到状态 2：

$$
\frac{P_1 V_1}{T_1} = \frac{P_2 V_2}{T_2}
$$

> 这是理想气体方程在两个状态之间的比值形式，考试常用。注意 $T$ 必须用开尔文！

## 九、压缩因子 Z——理想气体不够准时

### 9.1 定义

当气体不满足理想气体条件时，引入**压缩因子** $Z$ 来修正：

$$
P \cdot v = Z \cdot R \cdot T
$$

- $Z = 1$：理想气体（分子间无作用力，分子不占体积）
- $Z \neq 1$：真实气体，偏离理想行为
- $Z < 1$：分子间有吸引力，气体比理想气体更易压缩
- $Z > 1$：分子间有排斥力

### 9.2 怎么查 Z？——对比压力与对比温度

**问题**：$Z$ 取决于气体种类和当前状态。**同一种气体**在不同的 $P$、$T$ 下 $Z$ 不同（不同气体之间也不同）。难道要给每种气体的每个状态都标一个 $Z$？

** trick：对应状态原理（Principle of Corresponding States）**

实验发现：如果把不同气体的压力和温度**除以各自的临界值**（归一化），它们会遵循**大致相同的 $Z$ 曲线**。这样只需要**一张通用图**就能查所有气体的 $Z$。

**什么是临界值？** 每种气体都有临界点（钟形顶点），临界压力 $P_{\text{cr}}$ 和临界温度 $T_{\text{cr}}$ 是固定值，可以查表得到。比如水：$P_{\text{cr}} = 22.1$ MPa，$T_{\text{cr}} = 374°C = 647$ K。

**归一化方法——对比参数**：

$$P_r = \frac{P}{P_{\text{cr}}} \quad \text{（对比压力 = 实际压力 ÷ 临界压力）}$$

$$T_r = \frac{T}{T_{\text{cr}}} \quad \text{（对比温度 = 实际温度 ÷ 临界温度）}$$

$P_r$ 和 $T_r$ 的物理意义：当前状态离临界点有多"远"。
- "r" = **r**educed（对比/约化），就是把实际值除以临界值
- 量纲消掉了（[Pa]/[Pa] = 无量纲，[K]/[K] = 无量纲），是纯数字 → 不同气体、不同单位制才能放在同一张图上
- $P_r = 1$ → 正好在临界压力
- $P_r < 1$ → 低于临界压力（可能在两相区）
- $P_r > 1$ → 高于临界压力

**怎么用（4 步）**：

1. 知道气体的 $P$、$T$（题目给）
2. 查表得到该气体的 $P_{\text{cr}}$、$T_{\text{cr}}$（固定值）
3. 算 $P_r = P / P_{\text{cr}}$，$T_r = T / T_{\text{cr}}$
4. 用通用压缩因子图：在图上找到 $T_r$ 对应的曲线 → 横轴找 $P_r$ → 纵轴读 $Z$

得到 $Z$ 后代入 $Pv = ZRT$ 算你需要的量。

> **为什么图上没有 $v$（比容）？** 因为 $v$ 不需要作为输入。$P$、$T$ 确定后状态就确定了（状态公设），$Z$ 和 $v$ 都跟着确定。查图时用 $P_r$、$T_r$ 得到 $Z$，然后 $v = ZRT/P$ 算出来——$v$ 是**输出**，不是输入。

> **考试提示**：如果题目给了 $R$、$T_{\text{cr}}$、$P_{\text{cr}}$，就是在提示你用压缩因子法。如果只给了 $R$ 没给临界值，说明可以直接用理想气体方程（$Z = 1$）。

## 十、典型例题

### 例题1：查表填空

> 补全水的性质表：

| $T$ (°C) | $P$ (kPa) | $h$ (kJ/kg) | $x$ | 相态 |
| ----------- | ----------- | ------------- | ----- | ---- |
| ?           | 200         | ?             | 0.7   | ?    |
| 140         | ?           | 1800          | ?     | ?    |
| ?           | 950         | ?             | 0.0   | ?    |
| 80          | 500         | ?             | ?     | ?    |
| ?           | 800         | 3162.2        | ?     | ?    |

**第 1 行**：$P = 200$ kPa，$x = 0.7$

- $x$ 在 0 和 1 之间 → 气液混合物
- 查饱和压力表（$P = 200$ kPa）：$T_{\text{sat}} = 120.21$°C，$h_f = 504.68$，$h_g = 2706.3$
- $h = h_f + x \cdot h_{fg} = 504.68 + 0.7 \times (2706.3 - 504.68) = 2046.0$ kJ/kg

**第 2 行**：$T = 140$°C，$h = 1800$ kJ/kg

- 查饱和温度表（$T = 140$°C）：$P_{\text{sat}} = 361.53$ kPa，$h_f = 589.13$，$h_g = 2733.1$
- $h_f < 1800 < h_g$ → 混合物
- $x = \frac{h - h_f}{h_g - h_f} = \frac{1800 - 589.13}{2733.1 - 589.13} = 0.565$

**第 3 行**：$P = 950$ kPa，$x = 0.0$

- $x = 0$ → 饱和液体
- 查饱和压力表（$P = 950$ kPa）：$T_{\text{sat}} = 177.66$°C，$h = h_f = 752$ kJ/kg

**第 4 行**：$T = 80$°C，$P = 500$ kPa

- 查饱和压力表（$P = 500$ kPa）：$T_{\text{sat}} = 151.8$°C
- $T = 80 < T_{\text{sat}} = 151.8$ → **过冷液体**
- 过冷液体近似：$h \approx h_f(80°C) = 335.02$ kJ/kg，$x$ 不适用

**第 5 行**：$P = 800$ kPa，$h = 3162.2$ kJ/kg

- 查饱和压力表（$P = 800$ kPa）：$h_g = 2768.3$
- $h = 3162.2 > h_g = 2768.3$ → **过热蒸气**
- 查过热表（$P = 800$ kPa，$h = 3162.2$）：$T = 350$°C，$x$ 不适用

### 例题2：R-134a 容器

> 9 m³ 容器装 300 kg R-134a，温度 20°C。求比焓 $h$。

**第 1 步**：算比容

$$
v = \frac{V}{m} = \frac{9}{300} = 0.03 \text{ m}^3\text{/kg}
$$

**第 2 步**：查 20°C 饱和表

$$
v_f = 0.0008161, \quad v_g = 0.035969
$$

$v_f < v = 0.03 < v_g$ → 气液混合物

**第 3 步**：算干度

$$
x = \frac{v - v_f}{v_g - v_f} = \frac{0.03 - 0.0008161}{0.035969 - 0.0008161} = 0.83
$$

**第 4 步**：算焓

$$
h = h_f + x \cdot h_{fg} = 79.32 + 0.83 \times (261.59 - 79.32) = 230.6 \text{ kJ/kg}
$$

### 例题3：理想气体——氮气密度

> 氮气，$P = 5$ bar = 500 kPa，$T = 50$°C = 323 K，$R = 297$ J/(kg·K)。求密度。

$$
P \cdot v = R \cdot T \implies v = \frac{RT}{P} = \frac{297 \times 323}{500000} = 0.1919 \text{ m}^3\text{/kg}
$$

$$
\rho = \frac{1}{v} = \frac{1}{0.1919} = 5.21 \text{ kg/m}^3
$$

### 例题4：刚性容器加热（理想气体）

> 刚性容器中理想气体，初态 $T_1 = 260$°C = 533 K，表压 1.50 bar，大气压 98 kPa。加热到表压 2.60 bar。求终态温度。

**假设**：刚性容器 → 体积不变 → $V_1 = V_2$，即 $v_1 = v_2$

先算绝对压力：

$$
P_1 = P_{\text{gauge}} + P_{\text{atm}} = 150 + 98 = 248 \text{ kPa}
$$

$$
P_2 = 260 + 98 = 358 \text{ kPa}
$$

等容过程，$v$ 不变：

$$
\frac{P_1}{T_1} = \frac{P_2}{T_2} \implies T_2 = T_1 \times \frac{P_2}{P_1} = 533 \times \frac{358}{248} = 769 \text{ K} = 496°\text{C}
$$

> **要点**：表压必须转绝对压力！刚性容器意味着 $v$ 不变，用 $\frac{P_1}{T_1} = \frac{P_2}{T_2}$。

## 十一、本篇小结

### 核心概念

| 概念                       | 要点                                           |
| -------------------------- | ---------------------------------------------- |
| 纯物质                     | 化学组成均匀不变（空气也算）                   |
| 相变五阶段                 | 过冷液体→饱和液体→混合物→饱和蒸气→过热蒸气 |
| 显热                       | 引起温度变化的热量                             |
| 潜热                       | 相变时吸收/释放的热量，温度不变                |
| 饱和温度$T_{\text{sat}}$ | 给定压力下的沸点                               |
| 饱和压力$P_{\text{sat}}$ | 给定温度下的相变压力                           |
| 临界点                     | 液气不可区分的状态                             |
| 干度$x$                  | 混合物中气相质量比例（0=纯液，1=纯气）         |
| 焓$h = u + Pv$           | 内能 + 流动功，第五篇详讲                      |

### 必记公式

| 公式                                          | 含义                            |
| --------------------------------------------- | ------------------------------- |
| $h = u + Pv$                                | 焓定义                          |
| $x = \frac{m_g}{m_f + m_g}$                 | 干度定义                        |
| $v = v_f + x \cdot v_{fg}$                  | 混合物比容                      |
| $h = h_f + x \cdot h_{fg}$                  | 混合物比焓                      |
| $u = u_f + x \cdot u_{fg}$                  | 混合物比内能                    |
| $Pv = RT$                                   | 理想气体方程（比容形式）        |
| $PV = mRT$                                  | 理想气体方程（总量形式）        |
| $R = R_u / M$                               | 气体常数 = 通用常数 ÷ 摩尔质量 |
| $Pv = ZRT$                                  | 真实气体方程（含压缩因子）      |
| $\frac{P_1 V_1}{T_1} = \frac{P_2 V_2}{T_2}$ | 理想气体两状态关系              |

## 十一、查表完全指南——从此不再"欠约束"

### 11.0 你有几张表？

热力学里只有**四类表**，每张的用途完全不同：

| 表名 | 输入 | 输出 | 适用场景 |
|------|------|------|---------|
| **饱和表（按温度）** | T | \(P_{\text{sat}}, v_f, v_g, u_f, u_g, h_f, h_g\) | 已知 T，想知道沸点和两相性质 |
| **饱和表（按压力）** | P | \(T_{\text{sat}}, v_f, v_g, u_f, u_g, h_f, h_g\) | 已知 P，想知道沸点和两相性质 |
| **过热水蒸气表** | P, T（T > T_sat） | \(v, u, h\) | 蒸气温度超过沸点，单相 |
| **压缩液体表** | P, T（T < T_sat） | \(v, u, h\) | 液体温度低于沸点（考试通常用近似：≈ 同温饱和液的 \(v_f, u_f, h_f\)） |

> **理想气体**（空气、氮气等）不用表——用 \(Pv = RT\)，\(h = c_pT\)，\(u = c_vT\)。

---

### 11.1 核心心智：状态公设 → 自由度判断

> **状态公设**：简单可压缩物质的平衡态由**两个独立强度性质**完全确定。

这意味着：**只要你有任意两个独立的（P, T, v, u, h, x 中的两个），状态就锁死了，所有其他量都能查出来。** 你应该永远不感觉"欠约束"——如果觉得欠，说明你漏看了一个已知条件，或者没意识到某个条件等价于一个性质。

**自由度对照表**：

| 已知条件 | 自由度 | 能否求解 | 说明 |
|---------|--------|---------|------|
| P + T（且 T > T_sat） | 2（独立） | ✅ | 过热蒸气，直接查过热表 |
| P + T（且 T < T_sat） | 2（独立） | ✅ | 压缩液体，查压缩液表或用饱和近似 |
| P + T（且 T = T_sat） | **1**（P, T 被锁定在一起了！） | ❌ 欠约束 | 需要 x 或 v/h/u 来定位 |
| P + x | 2 | ✅ | 饱和混合物，查 P→饱和表→插值 |
| T + x | 2 | ✅ | 饱和混合物，查 T→饱和表→插值 |
| P + h（且 h_f < h < h_g） | 2 | ✅ | h 不是 P 的单值函数，可解 x |
| 仅 P | 1 | ❌ | 除非默认饱和液体（x=0）或饱和蒸气（x=1） |
| "饱和液体" | 2（隐含 x=0 + P 或 T 中的一个） | ✅ | 沿着饱和液体线 |
| "饱和蒸气" | 2（隐含 x=1 + P 或 T 中的一个） | ✅ | 沿着饱和蒸气线 |

---

### 11.2 查表总流程

```
拿到一个状态 → 我需要查什么？
│
├─ 是理想气体？（题目说"空气"且远离两相区）
│   └─ Pv = RT,  h = c_p·T,  u = c_v·T  → 不需要表！
│
├─ 是水/水蒸气/R-134a？
│   │
│   ├─ 第 0 步：已知哪两个量？
│   │
│   ├─ 第 1 步：判断相态
│   │   │  已知 P 和 T：
│   │   │    查饱和表 → 得 T_sat(P) 或 P_sat(T)
│   │   │    T > T_sat(P)  → 过热蒸气 → 跳过热气表
│   │   │    T < T_sat(P)  → 压缩液体 → 用饱和液近似
│   │   │    T = T_sat(P)  → 在饱和线上 → 需要 x！
│   │   │
│   │   │  已知 P（或 T）和 v/u/h：
│   │   │    查饱和表 → 得 v_f, v_g（或 u_f, u_g 等）
│   │   │    v < v_f  → 压缩液体
│   │   │    v_f ≤ v ≤ v_g  → 两相混合物 → 算 x
│   │   │    v > v_g  → 过热蒸气
│   │
│   ├─ 第 2 步：按相态取数据
│   │    单相 → 直接查表读 v, u, h
│   │    两相 → 用 x 插值：y = y_f + x·(y_g − y_f)
│   │
│   └─ 第 3 步：查出来的 h 或 u 直接代入能量方程
```

---

### 11.3 考试中最容易"欠约束"的三种场景——及破解

| 场景 | 为什么觉得欠 | 其实不缺 |
|------|------------|---------|
| 涡轮出口给了 P 和"干度 x=0.95" | 以为只有 P 一个独立量 | P + x = 两个独立量！查饱和表 → 插值出 h₂ |
| 节流阀给了入口 P,T，出口只给 P | 以为出口缺一个量 | 节流阀 h₁=h₂！入口已知 → h₂ 已知 → P + h = 两个独立量 |
| 题目说"饱和液体进入"但没给 T | 以为缺温度 | "饱和液体" = x=0，再加一个条件（如给了 P）就锁死 |
| 给了"过热蒸气，P=2MPa, T=400°C" | — | 两个独立量，直接查过热表，没毛病 |

### 11.4 下标含义速查

|------|------|------|
| \(f\) | fluid | 饱和液体 |
