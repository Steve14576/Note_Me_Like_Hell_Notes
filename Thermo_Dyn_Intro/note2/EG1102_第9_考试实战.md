## 一、解题7步法

PPT 给出的标准流程，考试按这个写能拿过程分：

| 步骤 | 内容 | 做什么 |
|------|------|--------|
| 1 | 题目陈述 | 写清楚"求什么" |
| 2 | 示意图 | 画图帮助理解（系统、边界、进出口） |
| 3 | 假设 | 列出假设（理想气体？稳态？绝热？无摩擦？） |
| 4 | 物理定律 | 写出要用的公式 |
| 5 | 性质 | 查表或用公式得到的值 |
| 6 | 计算 | 代入数字 |
| 7 | 验证 | 答案合理吗？ |

> **第 7 步很重要**：算完后花 10 秒检查——效率不可能 > 100%，温度不可能是负 K 值，汽车不可能 700 km/h。

## 三、真题拆解一：2021 B3（闭口系统）

### 题目 [10分]

> 氦气在气密无摩擦活塞气缸中膨胀，$P_1 = 500$ kPa, $T = 350$°C, $V_1 = 20$ L，膨胀到 $P_2 = 200$ kPa。
> $R = 2.080$ kJ/(kg·K), $c_p = 5.23$ kJ/(kg·K), $k = 1.66$
>
> (a) 求气体质量 [2分]
> (b) 求初始密度 [2分]
> (c) 等温膨胀的比功 [3分]
> (d) 完全绝热膨胀的比功 [3分]

### 第0步：整理变量 & 单位换算

| 变量 | 原始值 | 换算后 | 符号 |
|------|--------|--------|------|
| 压力1 | 500 kPa | 500,000 Pa | $P_1$ |
| 压力2 | 200 kPa | 200,000 Pa | $P_2$ |
| 温度 | 350°C | 623 K | $T$ |
| 初始体积 | 20 L | 0.02 m³ | $V_1$ |
| 气体常数 | 2.08 kJ/(kg·K) | 2080 J/(kg·K) | $R$ |
| 定压比热 | 5.23 kJ/(kg·K) | 5230 J/(kg·K) | $c_p$ |
| 比热比 | 1.66 | 1.66 | $k$ |

> **关键**：单位统一！压力用 Pa，温度用 K，$R$ 和 $c_p$ 用 J/(kg·K)。

### (a) 求质量 [2分]

**假设**：氦气为理想气体。

$$m = \frac{P_1 V_1}{RT} = \frac{500000 \times 0.02}{2080 \times 623} = 0.00773 \text{ kg}$$

### (b) 求密度 [2分]

$$\rho = \frac{m}{V_1} = \frac{0.00773}{0.02} = 0.387 \text{ kg/m}^3$$

（或 $\rho = \frac{P}{RT} = \frac{500000}{2080 \times 623} = 0.386$ kg/m³）

### (c) 等温膨胀比功 [3分]

**假设**：等温过程（$T$ 不变）→ $\Delta u = 0$ → $q = w$

先算 $V_2$（等温：$P_1 V_1 = P_2 V_2$）：

$$V_2 = \frac{P_1 V_1}{P_2} = \frac{500000 \times 0.02}{200000} = 0.05 \text{ m}^3$$

等温比功：

$$w = RT \ln\frac{P_1}{P_2} = 2080 \times 623 \times \ln\frac{500000}{200000} = 2080 \times 623 \times \ln(2.5)$$

$$w = 2080 \times 623 \times 0.9163 \approx 1187 \text{ kJ/kg}$$

### (d) 绝热膨胀比功 [3分]

**假设**：绝热（$Q = 0$），$n = k = 1.66$

先求 $T_2$（绝热 T-P 关系）：

$$\frac{T_2}{T_1} = \left(\frac{P_2}{P_1}\right)^{\frac{k-1}{k}} = \left(\frac{200}{500}\right)^{\frac{0.66}{1.66}} = 0.4^{0.3976} = 0.6846$$

$$T_2 = 623 \times 0.6846 = 426.5 \text{ K}$$

算 $c_v$：

$$c_v = \frac{R}{k - 1} = \frac{2080}{0.66} = 3152 \text{ J/(kg·K)}$$

绝热比功（$Q = 0$，$w = -\Delta u = c_v(T_1 - T_2)$）：

$$w = c_v(T_1 - T_2) = 3152 \times (623 - 426.5) = 3152 \times 196.5 \approx 619 \text{ kJ/kg}$$

> **对比**：等温功 (1187) > 绝热功 (619)。等温过程中吸热补充了能量，所以做功更多。

### 本题考点清单

1. ✅ 理想气体方程 $PV = mRT$
2. ✅ 单位换算（kPa→Pa, °C→K, kJ→J, L→m³）
3. ✅ 等温过程功：$w = RT \ln(P_1/P_2)$
4. ✅ 绝热过程功：$w = c_v(T_1 - T_2)$
5. ✅ $c_v = R/(k-1)$
6. ✅ 绝热温度关系：$T_2/T_1 = (P_2/P_1)^{(k-1)/k}$

## 四、真题拆解二：2019 B4（开口系统）

### 题目 [10分]

> 空气进入涡轮：$P_1 = 800$ kPa, $T_1 = 100$°C, $D_1 = 100$mm, $V_1 = 70$ m/s
> 出口：$P_2 = 120$ kPa, $D_2 = 400$mm
> $R = 287$ J/(kg·K), $c_p = 1.005$ kJ/(kg·K), $k = 1.4$
>
> (a) 质量流量 [2分]
> (b) 绝热膨胀后温度 [2分]
> (c) 进出口比动能 [2分]
> (d) 比功和功率（忽略 KE 和 PE）[4分]

### (a) 质量流量 [2分]

**假设**：理想气体，稳态。

入口截面积：$A_1 = \frac{\pi D_1^2}{4} = \frac{\pi \times 0.1^2}{4} = 0.007854$ m²

入口密度：$\rho_1 = \frac{P_1}{RT_1} = \frac{800000}{287 \times 373} = 7.474$ kg/m³

$$\dot{m} = \rho_1 A_1 V_1 = 7.474 \times 0.007854 \times 70 = 4.108 \text{ kg/s}$$

### (b) 绝热膨胀后温度 [2分]

$$\frac{T_2}{T_1} = \left(\frac{P_2}{P_1}\right)^{\frac{k-1}{k}} = \left(\frac{120}{800}\right)^{\frac{0.4}{1.4}} = 0.15^{0.2857} = 0.582$$

$$T_2 = 373 \times 0.582 = 217 \text{ K}$$

### (c) 比动能 [2分]

入口比动能：

$$ke_1 = \frac{V_1^2}{2} = \frac{70^2}{2} = 2450 \text{ J/kg} = 2.45 \text{ kJ/kg}$$

出口：先算 $V_2$（稳态 $\dot{m}_1 = \dot{m}_2$）：

$\rho_2 = \frac{P_2}{RT_2} = \frac{120000}{287 \times 217} = 1.928$ kg/m³

$A_2 = \frac{\pi D_2^2}{4} = \frac{\pi \times 0.4^2}{4} = 0.1257$ m²

$V_2 = \frac{\dot{m}}{\rho_2 A_2} = \frac{4.108}{1.928 \times 0.1257} = 16.95$ m/s

$$ke_2 = \frac{V_2^2}{2} = \frac{16.95^2}{2} = 143.7 \text{ J/kg} \approx 0.144 \text{ kJ/kg}$$

### (d) 比功和功率 [4分]

**假设**：绝热（$q = 0$），忽略 KE 和 PE。

$$w = c_p(T_1 - T_2) = 1005 \times (373 - 217) = 1005 \times 156 = 156780 \text{ J/kg} \approx 156.8 \text{ kJ/kg}$$

$$\dot{W} = \dot{m} \times w = 4.108 \times 156.8 = 644 \text{ kW}$$

### 本题考点清单

1. ✅ 质量流量 $\dot{m} = \rho AV$
2. ✅ 稳态连续性方程 $\dot{m}_1 = \dot{m}_2$
3. ✅ 绝热温度关系（开口系统同样适用）
4. ✅ 开口系统能量方程：$w = c_p(T_1 - T_2)$
5. ✅ 理想气体密度 $\rho = P/(RT)$
6. ✅ 速度 $V$ ≠ 体积流量 $\dot{V} = AV$

## 五、真题拆解三：2020 B5（热阻网络）

### 题目 [10分]

> 房间 18°C，室外 −5°C，窗户面积 4 m²。
> $k_a = 0.02439$ W/(m·K), $k_g = 0.85$ W/(m·K), $h_a = 30$ W/(m²·K)
> 单层玻璃厚 8mm；双层玻璃：两层 3mm 玻璃 + 20mm 空气层
>
> (a) 画热阻图 [4分]
> (b) 单层玻璃热传递 [3分]
> (c) 双层玻璃热传递 [3分]

### (b) 单层玻璃 [3分]

**假设**：稳态，一维热流。

热阻网络：室内 ──[$R_{\text{conv}}$]──[$R_{\text{glass}}$]──[$R_{\text{conv}}$]── 室外

$$R_{\text{conv}} = \frac{1}{h_a A} = \frac{1}{30 \times 4} = 0.00833 \text{ K/W}$$

$$R_{\text{glass}} = \frac{L_s}{k_g A} = \frac{0.008}{0.85 \times 4} = 0.00235 \text{ K/W}$$

$$R_{\text{total}} = 0.00833 + 0.00235 + 0.00833 = 0.01902 \text{ K/W}$$

$$\dot{Q} = \frac{\Delta T}{R_{\text{total}}} = \frac{18 - (-5)}{0.01902} = \frac{23}{0.01902} \approx 1209 \text{ W}$$

### (c) 双层玻璃 [3分]

热阻网络：室内 ──[$R_{\text{conv}}$]──[$R_{\text{glass1}}$]──[$R_{\text{air}}$]──[$R_{\text{glass2}}$]──[$R_{\text{conv}}$]── 室外

$$R_{\text{glass1}} = \frac{0.003}{0.85 \times 4} = 0.000882 \text{ K/W}$$

$$R_{\text{air}} = \frac{0.02}{0.02439 \times 4} = 0.205 \text{ K/W}$$

$$R_{\text{total}} = 0.00833 + 0.000882 + 0.205 + 0.000882 + 0.00833 = 0.2234 \text{ K/W}$$

$$\dot{Q} = \frac{23}{0.2234} \approx 103 \text{ W}$$

> **双层玻璃热损失仅为单层的 ~8.5%！** 空气层导热系数极小 → 热阻极大 → 大幅减少热损失。

## 六、单位换算陷阱大全

| 陷阱 | 错误做法 | 正确做法 |
|------|----------|----------|
| 温度 | 用 °C 代入公式 | 必须 K：$T(\text{K}) = T(°\text{C}) + 273.15$ |
| 压力 | 用表压代入 | 必须绝对压：$P_{\text{abs}} = P_{\text{gauge}} + P_{\text{atm}}$ |
| kPa vs Pa | 混用 | 统一！注意 $R$ 和 $c_p$ 的单位要匹配 |
| kJ vs J | 混用 | 统一！ |
| L vs m³ | 直接用升 | 1 m³ = 1000 L |
| km/h vs m/s | 直接用 km/h | ÷ 3.6 |
| rpm vs rad/s | 直接用 rpm | $\omega = 2\pi \times \text{rpm} / 60$ |
| 速度 V vs 体积 V | 混淆符号 | 看上下文判断 |

## 七、考前速记公式表

### 热力学

| 公式 | 用途 |
|------|------|
| $PV = mRT$ | 理想气体方程 |
| $Pv = RT$ | 理想气体（比容形式） |
| $h = u + Pv$ | 焓定义 |
| $Q - W = \Delta U$ | 闭口系统第一定律 |
| $\dot{Q} - \dot{W} = \dot{m}(h_2 - h_1)$ | 开口系统能量平衡 |
| $c_p - c_v = R$ | 比热关系 |
| $c_v = R/(k-1)$ | 用 $R$ 和 $k$ 算 $c_v$ |
| $\Delta u = c_v \Delta T$ | 理想气体内能变化 |
| $\Delta h = c_p \Delta T$ | 理想气体焓变化 |
| $x = \frac{v - v_f}{v_g - v_f}$ | 干度 |
| $h = h_f + x \cdot h_{fg}$ | 混合物焓 |

### 过程方程

| 过程 | 功公式 |
|------|--------|
| 等压 | $W = P(V_2 - V_1)$ |
| 等容 | $W = 0$ |
| 等温 | $w = RT \ln(P_1/P_2)$ |
| 绝热 | $w = c_v(T_1 - T_2)$ |
| 多变 | $W = \frac{P_2 V_2 - P_1 V_1}{1 - n}$ |

### 温度-压力关系

$$\frac{T_2}{T_1} = \left(\frac{P_2}{P_1}\right)^{\frac{n-1}{n}} \quad \text{(多变)}, \quad \frac{T_2}{T_1} = \left(\frac{P_2}{P_1}\right)^{\frac{k-1}{k}} \quad \text{(绝热)}$$

### 第二定律

| 公式 | 用途 |
|------|------|
| $\eta_{\text{Carnot}} = 1 - \frac{T_L}{T_H}$ | 卡诺效率 |
| $\eta_{\text{th}} = 1 - \frac{Q_L}{Q_H}$ | 热效率 |
| $\text{COP}_R = \frac{T_L}{T_H - T_L}$ | 制冷 COP |
| $\text{COP}_{HP} = \frac{T_H}{T_H - T_L}$ | 热泵 COP |

### 传热

| 公式 | 用途 |
|------|------|
| $\dot{Q} = \frac{kA\Delta T}{L}$ | 导热 |
| $\dot{Q} = hA(T_s - T_\infty)$ | 对流 |
| $\dot{Q} = \varepsilon\sigma A(T_s^4 - T_{\text{sur}}^4)$ | 辐射 |
| $R_{\text{cond}} = \frac{L}{kA}$ | 导热热阻 |
| $R_{\text{conv}} = \frac{1}{hA}$ | 对流热阻 |
| $R_{\text{cyl}} = \frac{\ln(r_2/r_1)}{2\pi kL}$ | 圆柱热阻 |
| $\dot{Q} = \frac{\Delta T}{R_{\text{total}}}$ | 热阻网络 |
| $Re = \frac{VL}{\nu}$ | 雷诺数 |
| $Nu = \frac{hL}{k}$ | 努塞尔数 |
| $\dot{m} = \rho AV$ | 质量流量 |

### 常用常数

| 常数 | 值 |
|------|-----|
| $\sigma$ (Stefan-Boltzmann) | $5.670 \times 10^{-8}$ W/(m²·K⁴) |
| $R_u$ (通用气体常数) | 8.314 kJ/(kmol·K) |
| $R_{\text{air}}$ | 287 J/(kg·K) |
| $k_{\text{air}}$ | 1.4 |
| $c_{p,\text{air}}$ | 1005 J/(kg·K) |

## 八、常见考试问法与对应方法

| 问法 | 你应该用 |
|------|----------|
| "求质量" | $m = PV/(RT)$ |
| "求密度" | $\rho = P/(RT)$ |
| "求质量流量" | $\dot{m} = \rho AV$ |
| "等温过程求功" | $w = RT \ln(P_1/P_2)$ |
| "绝热过程求功" | $w = c_v(T_1 - T_2)$，先求 $T_2$ |
| "求终态温度" | $T_2 = T_1(P_2/P_1)^{(n-1)/n}$ |
| "刚性容器" | $V$ 不变 → $P_1/T_1 = P_2/T_2$ |
| "完全保温/绝热" | $Q = 0$ |
| "活塞自由移动" | $P = \text{const}$ → 等压 |
| "节流阀" | $h_1 = h_2$ |
| "求热损失" | 热阻网络 → $\dot{Q} = \Delta T / R_{\text{total}}$ |
| "求卡诺效率" | $\eta = 1 - T_L/T_H$（K！） |
| "求辐射散热" | $\dot{Q} = \varepsilon\sigma A(T_s^4 - T_{\text{sur}}^4)$（K！） |

## 十、终极忠告

1. **单位！单位！单位！** —— 90% 的错误来自单位不统一
2. **温度用开尔文** —— 永远先换算成 K
3. **压力用绝对压** —— 表压必须加大气压
4. **画图** —— 画 T-v 图或 P-V 图帮助理解
5. **检查合理性** —— 效率不可能 > 1，温度不可能是负 K
6. **先写公式再代数字** —— 即使算错也有过程分
7. **区分速度 $V$ 和体积 $V$** —— 用不同写法

> **热力学不难，就是概念多 + 公式多 + 单位容易搞错。把公式表背熟，把真题做透，5 天足够了。**
