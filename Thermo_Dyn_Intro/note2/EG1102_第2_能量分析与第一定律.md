## 一、能量的形式——回顾与补充

### 1.1 宏观 vs 微观

第一篇已经讲过总能量 $E = U + KE + PE$。这里补充更细的分类：

| 层面 | 能量形式 | 说明 |
|------|---------|------|
| **宏观** (Macroscopic) | 动能 $KE$ | 整个系统作为物体运动具有的能量 |
| | 势能 $PE$ | 整个系统因高度具有的能量 |
| **微观** (Microscopic) | 内能 $U$ | 系统内所有微观粒子能量的总和 |

### 1.2 内能的细分（了解即可）

PPT 把内能进一步分成几类，但明确说"不需要记住这些"：

| 子类 | 含义 | 例子 |
|------|------|------|
| 显热能 (Sensible) | 分子运动（平动、转动、振动）→ **引起温度变化** | 加热水，温度从 20°C 升到 80°C |
| 潜热能 (Latent) | 相变时分子间距离变化 → **温度不变，但吸/放热** | 冰融化成水，温度停在 0°C |
| 化学能 (Chemical) | 原子键中的能量 | 甲烷燃烧，化学键断裂重组释放能量 |
| 核能 (Nuclear) | 原子核内的能量 | 核裂变/聚变 |

> **你不需要记住这些子类的名字**，只需要知道：内能 $U$ 是所有微观能量的总和。第三篇讲相变时会用到"显热"和"潜热"的概念。

### 1.3 总能量守恒

总能量 $E = U + KE + PE$ 永远守恒——可以在各形式之间转换，但总和不变。

比如石头从山顶落下：$PE$ 减小，$KE$ 增大，$U$ 基本不变，$E$ 总量不变。

## 二、热量（Heat, Q）——能量怎样以"热"的形式过边界

### 2.1 什么是热量？

> **热量 $Q$**：由于**温度差**而穿过系统边界的能量。

关键点：
1. **必须有温差**才有热量传递。如果系统和外界温度相同，$Q = 0$。
2. 热量是**穿过边界**的能量——不是系统"拥有"的，而是"传递中"的。
3. 热量总是从**高温**流向**低温**（第二定律）。

### 2.2 热量的符号约定

| 情况 | 符号 | 含义 |
|------|------|------|
| 热量传入系统（TO） | $Q > 0$（正） | 系统吸热，能量流入 |
| 热量传出系统（FROM） | $Q < 0$（负） | 系统放热，能量流出 |
| 没有热量传递 | $Q = 0$ | 叫做**绝热**（Adiabatic） |

> **记忆方法**：想象自己在冷环境中——如果你变暖了（吸热），感觉是"正面的"；如果变冷了（放热），感觉是"负面的"。

### 2.3 绝热过程（Adiabatic Process）

> **绝热过程**：$Q = 0$，系统与外界没有热量交换。

什么时候近似绝热？
- 系统被完美隔热
- 过程发生得非常快，来不及换热（比如内燃机气缸内的快速压缩/膨胀）

> 绝热是热力学中非常重要的假设。后面很多过程分析都会用到。

### 2.4 热量传递的三种方式（预告）

PPT 提到热量传递有三种方式，这是第七篇和第八篇的内容，这里先知道有这三种就行：

| 方式 | 英文 | 简单理解 |
|------|------|---------|
| 导热 | Conduction | 热量在固体内部从高温端传到低温端 |
| 对流 | Convection | 流体运动带走热量（风吹你感到凉） |
| 辐射 | Radiation | 电磁波传热（太阳照你感到热），不需要介质 |

### 2.5 热传递率与比热量

**热传递率**（Heat Transfer Rate）：

$$\dot{Q} = \frac{Q}{t}$$

- $\dot{Q}$ = 热传递率 [W = J/s]（瓦特）
- $Q$ = 热量 [J]
- $t$ = 时间 [s]

> **符号约定**：字母上面加一个点（dot）表示"除以时间"，也就是"率"。这个记号贯穿全课程：
> - $Q$ = 热量 [J]，$\dot{Q}$ = 热传递率 [W]
> - $W$ = 功 [J]，$\dot{W}$ = 功率 [W]
> - $m$ = 质量 [kg]，$\dot{m}$ = 质量流量 [kg/s]

**比热量**（Specific Heat Transfer）：

$$q = \frac{Q}{m}$$

- $q$ = 比热量 [J/kg 或 kJ/kg]（小写 q）
- $Q$ = 热量 [J 或 kJ]（大写 Q）
- $m$ = 质量 [kg]

> 跟第一篇一样：大写 = 总量，小写 = 每千克。

## 三、功（Work, W）——能量怎样以"功"的形式过边界

### 3.1 什么是功？

> **功 $W$**：力作用通过一段距离时传递的能量。

$$W = F \times s$$

这是高中就学过的定义。在热力学中，功的含义更广——任何不是由温差引起的、穿过边界的能量传递都叫做"功"。

### 3.2 功的符号约定（回顾）

第一篇已经讲过，这里再强调一遍：

| 情况 | 符号 | 含义 |
|------|------|------|
| 系统**对外**做功（BY） | $W > 0$（正） | 能量从系统流出 |
| 外界**对系统**做功（ON） | $W < 0$（负） | 能量流入系统 |

### 3.3 功率（Power）

$$\dot{W} = \frac{W}{t}$$

- $\dot{W}$ = 功率 [W = J/s]（瓦特）
- $W$ = 功 [J]
- $t$ = 时间 [s]

> **为什么不用 $P$ 表示功率？** 因为 $P$ 在热力学中被预留给了**压力**（Pressure）。所以功率一律用 $\dot{W}$（W 上面加点）表示。

### 3.4 比功

$$w = \frac{W}{m}$$

- $w$ = 比功 [J/kg 或 kJ/kg]（小写 w）
- $W$ = 功 [J 或 kJ]

### 3.5 功的多种形式

PPT 介绍了多种机械功和非机械功。它们的本质都是"力 × 距离"的变体：

**（1）轴功（Shaft Work）**

旋转轴传递的功。力 $F$ 通过力臂半径 $r$ 产生扭矩 $T = F \times r$，转 $n$ 圈做的功：

$$W_{\text{sh}} = 2\pi n T$$

- $n$ = 转数 [圈]
- $T$ = 扭矩 [N·m]

**（2）弹簧功（Spring Work）**

拉/压线性弹簧做的功。弹簧力 $F = kx$（$k$ 为弹簧常数 [N/m]，$x$ 为位移）：

$$W_{\text{spring}} = \frac{1}{2}k(x_2^2 - x_1^2)$$

**（3）重力功（Gravitational Work）**

提升物体做的功 = 势能变化：

$$W_g = mg(z_2 - z_1) = \Delta PE$$

**（4）电功（Electrical Work）**

$$W_e = V \cdot I \cdot t$$

- $V$ = 电压 [V]
- $I$ = 电流 [A]
- $t$ = 时间 [s]

> 这些形式不需要全部记住公式，考试主要考的是**边界功**（下面讲）。但要知道"功"在热力学中不仅仅指 $F \times s$。

### 3.6 边界功（Moving Boundary Work）——最重要的功

> **边界功**：当系统边界移动时（比如活塞运动），系统对外做的功。

这是热力学中最核心的功的形式。边界功的计算公式是：

$$W_b = \int P \, dV$$

- $P$ = 压力 [Pa]
- $V$ = 体积 [m³]
- 积分含义：在 P-V 图上，过程曲线下方的面积 = 做的功

**压缩**（Compression）：活塞往里推，体积减小，$dV < 0$，$W < 0$（外界对系统做功）

**膨胀**（Expansion）：活塞往外推，体积增大，$dV > 0$，$W > 0$（系统对外做功）

> 边界功的完整计算（等压、等温、绝热等不同过程）在第四篇（闭口系统）详细讲。这里只需要知道：**边界功 = P-V 图上曲线下方的面积**。

## 四、热量与功的本质——路径函数

### 4.1 边界现象

> 热量和功都是在**边界处**才被识别的——它们是穿过边界的能量传递。

一个系统**拥有**能量（$U, KE, PE$），但不"拥有"热量或功。热量和功只在**传递过程中**才有意义。

### 4.2 状态函数 vs 路径函数

这是热力学中极其重要的区分：

| | 状态函数（Point Function） | 路径函数（Path Function） |
|---|---|---|
| 例子 | $T, P, v, U, h$ 等所有**性质** | $Q, W$（热量和功） |
| 取决于什么？ | 只取决于初末**状态** | 取决于**路径**（过程怎么走的） |
| 微分记号 | 精确微分 $d$（如 $dU, dT$） | 不精确微分 $\delta$（如 $\delta Q, \delta W$） |
| 能不能说 $\Delta W = W_2 - W_1$？ | 不能！功没有"状态"的概念 | ✅ 可以说 $\Delta U = U_2 - U_1$ |

> **通俗解释**：你在三楼，下到一楼，势能变化 $\Delta PE = mg(z_1 - z_3)$ 不管你坐电梯还是走楼梯都一样——势能是状态函数。但你走楼梯和坐电梯消耗的体力（功）不同——功是路径函数。
>
> 在 P-V 图上，从状态 1 到状态 2 可以走不同的路径，路径不同则曲线下方的面积（= 功）不同，但 $U_2 - U_1$ 始终一样。

## 五、热力学第一定律

### 5.1 陈述

> **热力学第一定律**：能量不能被创造或消灭，只能从一种形式转化为另一种形式。

换句话说：系统获得的总能量 = 进入系统的能量 − 离开系统的能量。

### 5.2 能量平衡方程

对于**闭口系统**（质量不进出，只有热量和功穿过边界）：

$$\Delta E = Q - W$$

展开：

$$\Delta U + \Delta KE + \Delta PE = Q - W$$

其中：
- $\Delta U = U_2 - U_1$ = 内能变化
- $\Delta KE = \frac{1}{2}m(V_2^2 - V_1^2)$ = 动能变化（$V$ = 速度！）
- $\Delta PE = mg(z_2 - z_1)$ = 势能变化
- $Q$ = 净热量传入（正 = 吸热）
- $W$ = 净系统做功（正 = 对外做功）

> **为什么是 $Q - W$ 而不是 $Q + W$？** 因为工程约定中 $W > 0$ 表示系统对外做功（能量流出），所以要用减号。如果 $W < 0$（外界对系统做功），代入后 $\Delta E = Q - (-|W|) = Q + |W|$，能量是增加的，符合直觉。

### 5.3 常见简化情况

| 条件 | 简化 | 含义 |
|------|------|------|
| 绝热 ($Q = 0$) | $\Delta E = -W$ | 没有换热，能量变化完全来自功 |
| 无功 ($W = 0$) | $\Delta E = Q$ | 没有做功，能量变化完全来自热 |
| 无 KE、PE 变化 | $\Delta U = Q - W$ | 大多数问题的默认假设 |
| 循环 ($\Delta E = 0$) | $Q = W$ | 回到起点，净能量变化为零 |

> **最常用的形式**：如果系统不移动、高度不变（$\Delta KE = 0$, $\Delta PE = 0$），则：
> $$\Delta U = Q - W$$
> 这是闭口系统的核心公式，考试必考。

### 5.4 能量传递的三种机制

| 机制 | 适用系统 | 说明 |
|------|---------|------|
| 热量 $Q$ | 闭口和开口 | 因温差传递的能量 |
| 功 $W$ | 闭口和开口 | 非温差传递的能量 |
| 质量流 | **仅开口系统** | 质量进出携带的能量 |

闭口系统只有热量和功两种能量传递方式。开口系统还多一种——质量流携带的能量（焓，第五篇讲）。

### 5.5 能量平衡的速率形式

对时间求导：

$$\dot{E}_{\text{in}} - \dot{E}_{\text{out}} = \frac{dE_{\text{system}}}{dt}$$

- $\dot{E}$ = 能量传递率 [kW]
- $\frac{dE}{dt}$ = 系统能量变化率 [kW]

对于稳态（steady state），$\frac{dE}{dt} = 0$，所以 $\dot{E}_{\text{in}} = \dot{E}_{\text{out}}$。

## 六、能量转换效率

### 6.1 一般定义

> **效率** $\eta$（读作 "eta"）：有用输出与所需输入之比。

$$\eta = \frac{\text{有用输出 (Desired Output)}}{\text{所需输入 (Required Input)}}$$

效率永远在 0 到 1 之间（或 0% 到 100%），不可能超过 100%（否则违反第一定律）。

### 6.2 燃烧效率

$$\eta_{\text{combustion}} = \frac{Q}{m \times HV}$$

- $Q$ = 实际放出的热量 [kJ]
- $m$ = 燃料质量 [kg]
- $HV$ = **热值**（Heating Value）[kJ/kg]

> **什么是热值 $HV$？** 1 kg 燃料完全燃烧能放出的热量。比如汽油的热值约 44,000 kJ/kg，意味着 1 kg 汽油完全燃烧最多放出 44,000 kJ。
>
> **单位检查**：$Q$ [kJ] 除以 $m \times HV$ [kg × kJ/kg = kJ]，结果无量纲，是效率。

PPT 写的是 $\eta = Q / HV$，这里隐含了一个条件：$Q$ 是**单位质量燃料**放出的热量 [kJ/kg]，与 $HV$ [kJ/kg] 同单位。更完整的写法是 $\eta = Q / (m \times HV)$。

### 6.3 其他效率

| 效率 | 公式 | 含义 |
|------|------|------|
| 热效率 (Thermal) | $\eta_{\text{th}} = \frac{\dot{W}_{\text{net}}}{\dot{Q}_{\text{in}}}$ | 热机将热转化为功的比例 |
| 机械效率 | $\eta_{\text{mech}} = \frac{E_{\text{out}}}{E_{\text{in}}}$ | 机械能传递效率 |
| 电机效率 | $\eta_{\text{motor}} = \frac{\dot{W}_{\text{shaft,out}}}{\dot{W}_{\text{el,in}}}$ | 电功率 → 轴功率的比例 |
| 泵效率 | $\eta_{\text{pump}} = \frac{\Delta E_{\text{mech,fl}}}{\dot{W}_{\text{shaft}}}$ | 轴功率 → 流体机械能的比例 |

> PPT 还提到 HHV（Higher Heating Value，高热值），跟 HV 本质一样，只是测量条件不同。考试中用 HV 就行。

## 七、解题7步法

PPT 给出的标准解题流程，考试时按这个写能拿过程分：

| 步骤 | 内容 | 做什么 |
|------|------|--------|
| 1 | 题目陈述 | 写清楚"求什么" |
| 2 | 示意图 | 画图帮助理解（系统、边界、进出口） |
| 3 | 假设与近似 | 列出假设（理想气体？稳态？绝热？无摩擦？） |
| 4 | 物理定律 | 写出要用的公式（$\Delta U = Q - W$？） |
| 5 | 性质 | 查表或用公式得到的值 |
| 6 | 计算 | 代入数字，算出答案 |
| 7 | 验证 | 答案合理吗？（汽车不可能 700 km/h） |

> **第 7 步很重要**：算完后花 10 秒检查答案是否离谱。效率不可能 > 100%，温度不可能是负的 K 值，汽车速度不可能 700 km/h。

## 八、例题

### 例题1：电暖器热量计算

> 6 kW 电暖器开了 2 小时 15 分钟。求传递给房间的能量 [J]。

**假设**：全部电能转化为热能。

**计算**：

先换算时间：$2 \text{ h } 15 \text{ min} = 2 \times 3600 + 15 \times 60 = 8100 \text{ s}$

$$Q = \dot{Q} \times t = 6000 \times 8100 = 48{,}600{,}000 \text{ J} = 48{,}600 \text{ kJ}$$

> **要点**：功率 [kW] × 时间 [s] = 能量 [kJ]。注意单位换算——小时和分钟必须转成秒。

### 例题2：风扇最小功率（重点！）

> 1m × 1m 方形管道中有风扇，气流速度 $V = 3$ m/s，空气密度 $\rho = 1.2$ kg/m³。求风扇电机最小功率。

这道题需要完整理解逻辑链条，一步步来：

**第 1 步：理解题目**

风扇的作用是什么？——把空气从静止加速到 3 m/s。空气获得了动能。风扇做的功 = 空气获得的动能。

"最小功率"意味着忽略所有损失（摩擦、湍流等），只算把空气加速到 3 m/s 所需的纯理论功率。

**第 2 步：算质量流量 $\dot{m}$**

先引入一个新概念——**质量流量**（Mass Flow Rate）$\dot{m}$：单位时间内流过某截面的质量。

$$\dot{m} = \rho \times A \times V$$

- $\dot{m}$ = 质量流量 [kg/s]
- $\rho$ = 密度 [kg/m³]
- $A$ = 截面积 [m²]
- $V$ = 流速 [m/s]（这里 $V$ 是速度！不是体积）

怎么理解这个公式？想象一截管道，空气以速度 $V$ 流过截面积 $A$。一秒钟内流过的体积 = $A \times V$ [m³/s]（截面积 × 速度 = 一秒钟"扫过"的体积），乘以密度就是质量流量。

代入数字：

$$\dot{m} = 1.2 \times (1 \times 1) \times 3 = 3.6 \text{ kg/s}$$

**第 3 步：算空气获得的动能功率**

每秒有 3.6 kg 空气被加速到 3 m/s。每千克空气获得的动能：

$$ke = \frac{1}{2}V^2 = \frac{1}{2} \times 3^2 = 4.5 \text{ J/kg}$$

每秒总共获得的动能 = 质量流量 × 比动能：

$$\dot{W} = \dot{m} \times \frac{1}{2}V^2 = 3.6 \times 4.5 = 16.2 \text{ W}$$

**第 4 步：理解 $\frac{1}{2}\dot{m}V^2$ 是什么**

$$\dot{W} = \frac{1}{2}\dot{m}V^2$$

- $\dot{m}$ = [kg/s]（质量流量）
- $V^2$ = [m²/s²]
- $\dot{m} \times V^2$ = [kg/s × m²/s²] = [kg·m²/s³] = [J/s] = [W]

所以 $\frac{1}{2}\dot{m}V^2$ 的单位是**瓦特 [W]**，是**功率**，不是能量。

对比高中动能 $\frac{1}{2}mV^2$ [J]：把质量 $m$ [kg] 换成质量流量 $\dot{m}$ [kg/s]，能量 [J] 就变成了功率 [W]——相当于"每秒钟流过的质量所拥有的动能"。

**第 5 步：为什么是"最小"功率？**

因为这里只算了把空气从静止加速到 3 m/s 所需的动能。实际中风扇还要克服摩擦、湍流等损失，实际功率 > 16.2 W。16.2 W 是理想情况下的下限。

> **逻辑链条总结**：
> 1. 风扇给空气加速 → 空气获得动能
> 2. 需要知道每秒有多少空气被加速 → 算质量流量 $\dot{m} = \rho A V$
> 3. 每秒空气获得的动能 = $\frac{1}{2}\dot{m}V^2$ [W]
> 4. 这就是最小功率（忽略摩擦等损失）

### 例题3：水泵摩擦损失

> 水泵提供 33 kW 轴功率，把水从低水库抽到高 45 m 的水库。流量 $0.05$ m³/s，密度 $1000$ kg/m³，$g = 9.81$ m/s²。求因摩擦转化为热能的机械功率。

**假设**：稳态操作；水库水面静止（$\Delta KE \approx 0$，水从低水库表面到高水库表面，两端速度都近似为零，动能变化可忽略）。

> 为什么能忽略动能？低水库和高水库都是大水体，水面基本不动。即使管道里有流速，题目也没给管道截面积——但更本质的原因是先假设了水库水面静止，所以入口和出口动能都≈0，变化也≈0。势能变化（$mg\Delta z = 22.1$ kW）是主要项，动能相比之下小到可以忽略。

**第 1 步：算质量流量**

$$\dot{m} = \rho \times \dot{V} = 1000 \times 0.05 = 50 \text{ kg/s}$$

> 注意：这里 $\dot{V}$ 是**体积流量** [m³/s]（= 截面积 × 速度），不是速度。$\dot{V} = A \times V$，但题目直接给了体积流量，不用算 $A$ 和 $V$。

**第 2 步：算提升水需要的有用功率**

水每秒被抬高 45 m，获得势能：

$$\dot{W}_{\text{useful}} = \dot{m} \times g \times z = 50 \times 9.81 \times 45 = 22{,}072.5 \text{ W} \approx 22.1 \text{ kW}$$

**第 3 步：摩擦损失 = 输入 − 有用输出**

$$\dot{W}_{\text{friction}} = \dot{W}_{\text{shaft}} - \dot{W}_{\text{useful}} = 33 - 22.1 = 10.9 \text{ kW}$$

> **逻辑**：水泵输入 33 kW，其中 22.1 kW 用于把水抬高（变成水的势能），剩下的 10.9 kW 被摩擦消耗掉变成热能。

### 例题4：汽车加速

> 800 kg 汽车从静止加速到 100 km/h。求所需能量。

**计算**：

换算速度：$100 \text{ km/h} = \frac{100}{3.6} = 27.78 \text{ m/s}$

$$\Delta KE = \frac{1}{2}m(V_2^2 - V_1^2) = \frac{1}{2} \times 800 \times (27.78^2 - 0) = 308{,}642 \text{ J} \approx 309 \text{ kJ}$$

> **要点**：速度必须先从 km/h 换算成 m/s。这是最常见的单位陷阱。

### 例题5：管道散热（对流预告）

> 直径 5 cm、长 10 m 的热水管，水温 80°C，环境空气 5°C。对流换热系数 $h = 25$ W/(m²·°C)。求热损失率 [W]。

**假设**：1. 稳态；2. 不考虑辐射。

**第 1 步：算管道表面积**

$$A_s = \pi \times D \times L = \pi \times 0.05 \times 10 = 1.571 \text{ m}^2$$

**第 2 步：用牛顿冷却定律**（第七篇详讲，这里先用）

$$\dot{Q} = h \times A_s \times (T_s - T_\infty) = 25 \times 1.571 \times (80 - 5) = 2944 \text{ W}$$

> 这道题是对流传热的预告，公式 $\dot{Q} = hA\Delta T$ 会在第七篇详细推导。

### 例题6：轴功率

> 扭矩 $T = 200$ N·m，转速 4000 rpm。求轴功率 [kW]。

**第 1 步：换算转速**

$$\omega = \frac{2\pi \times \text{rpm}}{60} = \frac{2\pi \times 4000}{60} = 418.9 \text{ rad/s}$$

**第 2 步：算功率**

$$\dot{W} = T \times \omega = 200 \times 418.9 = 83{,}776 \text{ W} \approx 83.8 \text{ kW}$$

> **要点**：rpm 必须换算成 rad/s。换算关系：$\omega = \frac{2\pi \times \text{rpm}}{60}$ [rad/s]。

## 九、本篇小结

### 核心公式

| 公式 | 含义 | 适用条件 |
|------|------|---------|
| $\Delta U = Q - W$ | 闭口系统第一定律 | 无 KE、PE 变化 |
| $\Delta E = Q - W$ | 闭口系统第一定律（完整） | 通用 |
| $\Delta E = \Delta U + \Delta KE + \Delta PE$ | 总能量变化 | — |
| $\Delta KE = \frac{1}{2}m(V_2^2 - V_1^2)$ | 动能变化 | $V$ = 速度 |
| $\Delta PE = mg(z_2 - z_1)$ | 势能变化 | $z$ = 高度 |
| $\dot{m} = \rho A V$ | 质量流量 | 开口系统 |
| $W_b = \int P \, dV$ | 边界功 | 闭口系统，准静态 |
| $W_{\text{sh}} = 2\pi n T$ | 轴功 | 旋转轴 |
| $\dot{W} = T \omega$ | 轴功率 | $\omega$ = rad/s |
| $\eta = \frac{\text{输出}}{\text{输入}}$ | 效率通用定义 | 0 ≤ η ≤ 1 |
| $\eta_{\text{comb}} = \frac{Q}{m \times HV}$ | 燃烧效率 | — |

### 符号约定总结

| 量 | 正号 | 负号 | 零 |
|---|------|------|---|
| $Q$ | 热量传入系统 | 热量传出系统 | 绝热（Adiabatic） |
| $W$ | 系统对外做功 | 外界对系统做功 | — |

### 关键概念

| 概念 | 要点 |
|------|------|
| 热量 $Q$ | 因温差穿过边界的能量，路径函数 |
| 功 $W$ | 非温差穿过边界的能量，路径函数 |
| 内能 $U$ | 微观能量总和，状态函数 |
| 路径函数 | 取决于路径（$Q, W$），用 $\delta$ |
| 状态函数 | 只取决于状态（$U, T, P$），用 $d$ |
| 绝热 | $Q = 0$ |
| 循环 | $\Delta E = 0$，故 $Q = W$ |
| 点上面的 dot | 除以时间（率），如 $\dot{Q}, \dot{W}, \dot{m}$ |
| 小写字母 | 除以质量（比性质），如 $q, w, u$ |

### 单位陷阱提醒

| 陷阱 | 正确做法 |
|------|---------|
| km/h vs m/s | 必须换算成 m/s：÷ 3.6 |
| rpm vs rad/s | $\omega = 2\pi \times \text{rpm} / 60$ |
| kW vs W | 注意乘 1000 |
| kJ vs J | 注意 $R$ 和 $c_p$ 的单位要匹配 |
| 温度 | 公式中一律用 K |
| 压力 | 公式中一律用绝对压力 Pa |
