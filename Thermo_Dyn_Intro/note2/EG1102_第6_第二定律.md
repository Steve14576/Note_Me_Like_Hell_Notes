# 第六篇：热力学第二定律

> **对应讲义**：Lect 09
>
> **前置知识**：第一篇（热量方向）、第二篇（第一定律、效率）、第五篇（循环、开口系统设备）
>
> 第一定律告诉你"能量守恒"，但不告诉你能量**能不能自发转化**。一杯咖啡放在房间里，第一定律不反对咖啡变热、房间变冷（总能量不变），但现实中永远不会发生——这就是第二定律管的事：**过程的方向**和**效率的上限**。

---

## 一、为什么需要第二定律？

### 1.1 第一定律的局限

第一定律只说"能量不生不灭"，但有些过程虽然满足能量守恒，却永远不会自发发生：

- 热咖啡放在冷房间里 → 咖啡变凉（✅ 会发生）
- 冷房间里的热量自动集中到咖啡里 → 咖啡变热（❌ 不会发生）
- 电通过电阻发热（✅ 会发生）
- 热量自动变成电（❌ 不会发生）

第一定律对这两个方向都不禁止，但现实只选一个方向。**第二定律就是告诉你哪个方向是可能的。**

### 1.2 第二定律的三个用途

1. **预测过程方向**：热量从热到冷，不会反过来
2. **确定效率上限**：热机效率不可能 100%，最高多少？
3. **定义温度标度**：与物质无关的热力学温标

---

## 二、热源与热汇

### 2.1 定义

> **热能储库（Thermal Energy Reservoir）**：一个理想化的物体，能吸收或释放有限热量而自身温度不变。

- **热源（Source, $T_H$）**：高温储库，提供热量
- **热汇（Sink, $T_L$）**：低温储库，吸收废热

例子：海洋、湖泊、大气——因为热容量巨大，吸收/释放一点热量温度几乎不变。

> 后面的热机和制冷机都需要两个储库：一个高温、一个低温。

---

## 三、热机（Heat Engine）

### 3.1 什么是热机？

> **热机**：把热能转化为功的循环设备。

工作流程：

1. 从高温热源 $T_H$ 吸收热量 $Q_H$
2. 把一部分热量转化为功 $W$
3. 把剩余废热 $Q_L$ 排到低温热汇 $T_L$
4. 循环回到起点

> **为什么要排热？** 因为第二定律（Kelvin-Planck 陈述）规定：不可能只从一个热源吸热就全部变成功。必须有一部分热量排到低温端，循环才能完成。

### 3.2 热效率（Thermal Efficiency）

$$
\eta_{\text{th}} = \frac{W_{\text{net}}}{Q_H} = \frac{Q_H - Q_L}{Q_H} = 1 - \frac{Q_L}{Q_H}
$$

- $W_{\text{net}}$ = 净输出功 [kJ]
- $Q_H$ = 从高温源吸收的热量 [kJ]
- $Q_L$ = 排到低温汇的废热 [kJ]

> 热效率永远小于 1（即小于 100%）。因为 $Q_L > 0$（必须排热），所以 $\eta < 1$。

> **注意**：这里 $Q_H$ 和 $Q_L$ 都取绝对值（正数），不考虑符号约定。$W_{\text{net}} = Q_H - Q_L$。

---

## 四、第二定律的两种陈述

### 4.1 Kelvin-Planck 陈述（针对热机）

> **不可能**造出一个循环运行的热机，只从一个热源吸热并**全部**转化为功。

通俗说：**热机效率不可能 100%**。你必须排一部分废热到低温端。

> 这不是摩擦导致的——即使没有摩擦，理论上也不可能 100%。这是自然规律。

### 4.2 Clausius 陈述（针对制冷机）

> **不可能**造出一个循环运行的设备，不消耗功就把热量从低温传到高温。

通俗说：**热量不会自发从冷到热**。要让热量"逆流"（从冷到热），必须输入功（比如压缩机的电）。

### 4.3 两种陈述的等价性

两种陈述看似不同，但实际上是等价的——违反其中一个，必然违反另一个。

|        | Kelvin-Planck          | Clausius                 |
| ------ | ---------------------- | ------------------------ |
| 针对   | 热机（产功）           | 制冷机/热泵（耗功）      |
| 说的是 | 不能 100% 把热变功     | 不能不花功就让热从冷到热 |
| 共同点 | 自然过程有方向，不可逆 | 同左                     |

## 五、制冷机和热泵

### 5.1 共同原理：$Q_L + W = Q_H$

制冷机和热泵是**同一台设备**，只是目的不同。核心能量流：

```
       冷端（T_L，冰箱里 / 室外）
          ↑
          │ Q_L ← 蹭来的免费热
    ┌─────┴─────┐
    │   压缩机    │ ← W = 花的电费
    └─────┬─────┘
          │ Q_H = Q_L + W ← 排出去的热
          ↓
       热端（T_H，厨房 / 室内）
```

> **第一定律铁律**：$Q_H = Q_L + W$。排出去的热 = 蹭来的热 + 花的电。这是后面所有 COP 公式变形的根。

| 量            | 是什么           | 冰箱例子                   |
| ------------- | ---------------- | -------------------------- |
| $Q_L$       | 从冷端吸走的热   | 从冷冻室搬出来的热量       |
| $W$         | 输入的功         | 压缩机耗的电（电表走的字） |
| $Q_H$       | 排到热端的热     | 冰箱背后散热片排出的热     |
| $Q_H - Q_L$ | =$W$，就是电功 | 能量守恒强制相等           |

---

### 5.2 制冷机 COP

> **目的**：制冷 → 你想要的 = 从冷库吸走的热 $Q_L$

$$
\text{COP}_R = \frac{Q_L}{W} = \frac{Q_L}{Q_H - Q_L}
$$

- 分子 $Q_L$：制冷量（你想要的）
- 分母 $W$：电费（你花的）
- 分母写 $Q_H - Q_L$：题目没给 W 时的替代写法（因为 W = Q_H − Q_L）
- T 版本 $\frac{T_L}{T_H - T_L}$：**卡诺制冷机**的理论极限，真实 COP 更低

> COP 可以 > 1。花 1 kJ 电搬 3 kJ 热 → COP = 3。

---

### 5.3 热泵 COP

> **目的**：制热 → 你想要的 = 排到室内的热 $Q_H$

$$
\text{COP}_{HP} = \frac{Q_H}{W} = \frac{Q_H}{Q_H - Q_L}
$$

- 分子 $Q_H$：制热量（你想要的，含你花的电）
- 分母 $W$：电费（你花的）
- T 版本 $\frac{T_H}{T_H - T_L}$：**卡诺热泵**的理论极限

> 冬天开空调制热就是热泵。COP > 1 → 比电热丝直接发热省电。

---

### 5.4 制冷机 vs 热泵——一张表

|          | 制冷机                                 | 热泵                   |
| -------- | -------------------------------------- | ---------------------- |
| 目的     | 制冷（吸$Q_L$）                      | 制热（排$Q_H$）      |
| COP      | $Q_L / W$                            | $Q_H / W$            |
| COP 关系 | $\text{COP}_{HP} = \text{COP}_R + 1$ | 同一个设备，换个角度看 |
| 例子     | 冰箱、空调制冷                         | 空调制热、空气源热水器 |

## 六、可逆与不可逆过程

### 6.1 为什么要区分？

可逆过程是理想化的**理论上限**——真实过程都是不可逆的，但可逆过程给出了最好的可能性能。

### 6.2 定义

> **可逆过程**：可以完全逆转，不留任何痕迹的过程。系统和外界都恢复到初始状态。

> **不可逆过程**：无法完全逆转的过程。"不可逆"= 不能回到之前的状态。

### 6.3 不可逆性的来源

任何真实过程都包含以下不可逆因素（至少一种）：

- 摩擦
- 有限温差下的热传递
- 自由膨胀（气体向真空中膨胀）
- 流体混合
- 电阻发热
- 非弹性变形
- 化学反应

> 一个过程只要包含以上任何一种，就是不可逆的。

## 七、卡诺循环（Carnot Cycle）

### 7.1 什么是卡诺循环？

> **卡诺循环**：理论上最高效的热机循环。完全可逆，无摩擦，无损失。

由四个被精确控制的**可逆过程**组成（全程无自发，每一步活塞都在人的手里）：

| 阶段 | 过程类型            | 谁在控制                          | 能量发生了什么                            |
| ---- | ------------------- | --------------------------------- | ----------------------------------------- |
| 1→2 | 等温膨胀（$T_H$） | **人松活塞 + 热源同步补热** | 吸热$Q_H$，全变成功（T 不变），U 不变   |
| 2→3 | 绝热膨胀            | **人松活塞，撤热源**        | Q=0，做功全花 U → U↓ → T↓             |
| 3→4 | 等温压缩（$T_L$） | **人推活塞 + 冷源同步吸热** | 排热$Q_L$，外界功全被冷源带走（T 不变） |
| 4→1 | 绝热压缩            | **人推活塞，撤冷源**        | Q=0，外界功全存 U → U↑ → T↑           |

> **P-V 图上的卡诺循环**：由两条等温线和两条绝热线围成的封闭曲线。围成的**面积 = 净输出功**。

```
P
↑
│    T_H 等温线（1→2）: PV = nRT_H，双曲线，较平缓
│     1 ··——→··· 2
│      \          \
│       \  吸 Q_H   \  绝热线（2→3）: PV^k = const，比等温线更陡
│        \          \
│        4 ··——←··· 3
│           排 Q_L
│    T_L 等温线（3→4）: PV = nRT_L，下方双曲线
│
└──────────────────────────→ V

1→2  等温膨胀（T_H）：吸热 Q_H，膨胀做功（沿上方双曲线向右下）
2→3  绝热膨胀：继续做功，T 从 T_H 降到 T_L（沿陡线向右下）
3→4  等温压缩（T_L）：排热 Q_L，外界压缩气体（沿下方双曲线向左上）
4→1  绝热压缩：外界继续压缩，T 从 T_L 升回 T_H（沿陡线向左上）
```

> 四条线围成的面积 = 净功 W。上面两个过程（1→2→3）对外做功，下面两个过程（4→1←3）外界对系统做功，差值就是有用功。

**T-v 图上的卡诺循环**：等温线 = 水平线，绝热线 = 曲线。比 P-v 图更直观——温度变化一目了然。

```
T
↑
│  1 ··——→··· 2    T_H 等温膨胀（吸 Q_H，v 增大）
│  │          │
│  │  绝热     │  绝热  绝热线：T 变、v 也变
│  │  压缩     │  膨胀
│  │          │
│  4 ··——←··· 3    T_L 等温压缩（排 Q_L，v 减小）
│
└──────────────────→ v
```

> T-v 图上，等温线是水平线（T 不变），绝热线是斜线（T 和 v 同时变）。封闭曲线围成的面积 ≠ 净功（T-v 图上的面积没有直接的功的含义，功要看 P-v 图）。T-v 图的好处是：一眼看出哪段吸热、哪段排热、温度在哪变。

### 7.2 卡诺效率

$$
\eta_{\text{Carnot}} = 1 - \frac{T_L}{T_H}
$$

- $T_H$ = 高温热源温度 [K]（**必须用开尔文！**）
- $T_L$ = 低温热源温度 [K]

> **卡诺效率是所有热机的理论上限**。没有任何热机在相同温度区间内能超过卡诺效率。

### 7.3 例子

沸水（100°C = 373 K）做热源，冰水（0°C = 273 K）做冷源：

$$
\eta_{\text{Carnot}} = 1 - \frac{273}{373} = 0.268 = 26.8\%
$$

6 atm 压力下沸水（约 280°C = 553 K）做热源：

$$
\eta_{\text{Carnot}} = 1 - \frac{273}{553} = 0.506 = 50.6\%
$$

> 温差越大 → 效率越高。这就是为什么发电厂要用高温高压蒸汽。

### 7.4 深度理解：为什么 Q 能换成 T？Q 和 T 不是正相关吗？

**Q 和 T 在热机里解耦了。** 你的直觉"Q↑ → U↑ → T↑"在定容加热时是对的，但热机循环里的等温过程把这链条截断了：

| 过程                            | Q | U             | T              | 能量去哪了     |
| ------------------------------- | - | ------------- | -------------- | -------------- |
| 定容加热（烧水）                | + | +             | +              | 全进内能，升温 |
| **等温膨胀**（卡诺 1→2） | + | 0（理想气体） | **不变** | 全变成功！     |
| 绝热膨胀（卡诺 2→3）           | 0 | −            | −             | 内能变功       |

等温膨胀里热量进来，温度纹丝不动——因为进来的热量直接变成了功，没存进内能。所以 **Q 的大小不能直接推 T 的大小，它们在这里是解耦的。**

**为什么只有可逆循环能把 Q 换成 T？** 卡诺四步全是可逆过程：

```
1→2 等温吸热（T_H）： ΔS = +Q_H/T_H
2→3 绝热膨胀：        ΔS = 0
3→4 等温排热（T_L）： ΔS = −Q_L/T_L
4→1 绝热压缩：        ΔS = 0
────────────────────────────────
循环总熵变 = Q_H/T_H − Q_L/T_L = 0  ← 可逆！
→ Q_H/T_H = Q_L/T_L → Q_L/Q_H = T_L/T_H → η = 1 − T_L/T_H
```

**如果循环里有任何不可逆性（摩擦、有限温差传热），总熵变 > 0 → Q_L/Q_H > T_L/T_H → 效率 < 卡诺。** 所以卡诺效率是上限——不是因为它叫"卡诺"，而是因为它是唯一全程可逆的四步循环。

**卡诺循环意味着什么——三个"铁律"**：

1. 效率只取决于温度比 T_L/T_H，跟工质无关（水、空气、R-134a 都一样）
2. T_L 不能是 0K → 必须排热 → 效率永远 < 100%
3. 它是基准——所有真实热机效率 = η_Carnot × (某个 < 1 的不可逆因子)

### 7.5 蒸汽卡诺循环（≠ 理想气体卡诺）

用蒸汽做工质时，T-v 图上的卡诺循环长得跟理想气体的不一样——**因为工质在两相区走，等温 = 等压，形状变成一个近似的矩形。** 常见考题给两个压力（如 30 bar 和 0.04 bar），让算效率。

**四个过程（蒸汽版）**：

| 阶段 | 过程          | T-v 图上                 | 物理                                |
| ---- | ------------- | ------------------------ | ----------------------------------- |
| 1→2 | 等温吸热，T_H | **水平线**         | 锅炉：饱和液 → 饱和汽，等压 30 bar |
| 2→3 | 绝热膨胀      | **斜线**（右下）   | 涡轮：干饱和汽做功，T↓、v↑        |
| 3→4 | 等温排热，T_L | **水平线**（下方） | 冷凝器：汽 → 液，等压 0.04 bar     |
| 4→1 | 绝热压缩      | **近似竖线**       | 泵：低干度湿蒸汽压缩回 30 bar       |

> **为什么 4→1 看起来是竖线（v 几乎不变）？** 因为点 4 是低干度湿蒸汽（x ≈ 0.28，大部分是液体），液体几乎不可压缩，泵加压时 v 只从 ~0.001 变到 ~0.001，肉眼看去就是竖线。这是蒸汽卡诺区别于理想气体的最大视觉特征。

**怎么算**：不靠 `c_pΔT`，靠查饱和表取 `h_f` 和 `h_g`，四段分别计算 Δh → Q 和 W → η。计算方法和理想气体卡诺完全一样，只是数值来源不同。

### 7.6 为什么绝热膨胀会降温？（≠ 热胀冷缩）

**直觉错了在哪**："热胀"描述的是**定压加热**——一边烧火一边胀。但绝热膨胀是：**拔掉火源，给活塞松绑，气体花自己的存款做功。**

```
1→2 等温吸热（存钱）：
       吸 Q_H，但 T 不变（全变成功）→ 账户余额 U 没动
   
2→3 绝热膨胀（花钱）：
       Q = 0，松活塞 → 气体推活塞做功
       没新钱进来 → 花的全是 U → U↓ → T↓
       气体不是因为冷而收缩，是因为花了 U 才变冷
```

**绝热压缩反过来——工资到账**：

```
4→1 绝热压缩（发工资）：
       Q = 0，外界推活塞压缩气体
       没地方排钱 → 外界做的功全部存进 U → U↑ → T↑
       打气筒发热就是这个原理
```

| 你的直觉           | 实际               | 为什么                        |
| ------------------ | ------------------ | ----------------------------- |
| 膨胀 = 吸热才能胀  | 绝热膨胀不吸热也胀 | 驱动力是点 2 存的 U，不是新热 |
| 压缩 = 排热才能缩  | 绝热压缩不排热也缩 | 外界做功存进 U，没排热出口    |
| 做功 = 热变功      | 绝热做功 = U 变功  | Q = 0 时唯一来源就是 U        |
| 气体冷了因为没吸热 | 气体冷了因为花了 U | 因果关系反过来                |

> **不违反第二定律**：绝热过程不涉及热传递（Q = 0），第二定律管的是热传递的方向和效率。绝热过程是纯机械能 ↔ 内能转换，U 涨跌由功决定，跟第二定律无关。

### 7.7 绝热过程是"自发"的吗？——全程无自发

**没有一步是自发的。** 卡诺循环是四个被精确控制的**准静态过程**，每一步活塞都被人缓慢操控，气体始终处于几乎平衡的状态。

```
误会："加热完了撤火，气体惯性继续胀？"  → 错
事实：人把手放在活塞上，缓慢往外移，气体推着人的手做功

误会："压缩完了松手，气体惯性继续压？"  → 错  
事实：人继续推活塞，缓慢往里移，功变内能
```

**气体没有惯性。** 气体不会"记得刚才被加热过"，它在任意时刻的行为只取决于当前的 P、T、v。绝热膨胀不是"撤火后气体自己跑"——是人持续控制活塞外移，气体跟着推，花自己的 U。

**四步分别谁在动手**：

| 阶段          | 谁的手                  | 怎么动                                           |
| ------------- | ----------------------- | ------------------------------------------------ |
| 1→2 等温膨胀 | **人松 + 热源补** | 人缓慢松活塞，气体推手做功；热源同步补热，T 不变 |
| 2→3 绝热膨胀 | **人松，撤热源**  | 包绝热层，人继续松活塞，气体花 U 做功，T↓       |
| 3→4 等温压缩 | **人推 + 冷源吸** | 人缓慢推活塞，冷源同步吸热，T 不变               |
| 4→1 绝热压缩 | **人推，撤冷源**  | 包绝热层，人继续推活塞，功全存 U，T↑            |

> **关键**：活塞从未离开人的手。只是人有时在松（让气体推自己往外），有时在推（自己推气体往里）。所有"自发"过程都是不可逆的——卡诺循环恰好因为全程无自发，才可逆、才是理论极限。

### 7.8 例题：蒸汽动力循环性能计算

#### 装置长什么样？

蒸汽电厂就四件套——跟你在第五篇学的开口系统设备完全对应：

```
                    ┌─────────┐
          q_in      │  锅炉    │  ← 烧火，把水变成高温高压蒸汽
    ┌─────────────→│ (Boiler) │
    │              └────┬────┘
    │                   │ 2：干饱和蒸汽 (30 bar, h=2803)
    │                   ↓
    │              ┌─────────┐
    │              │  涡轮    │  ← 高温蒸汽冲叶片，涡轮转，发电机发电
    │              │(Turbine) │     做功 W_T = h₂ − h₃
    │              └────┬────┘
    │                   │ 3：低压湿蒸汽 (0.04 bar, h=1863, x=0.72)
    │                   ↓
    │              ┌─────────┐
    │              │ 冷凝器   │  ← 冷却水把乏汽凝成水，排热 q_out
    │              │(Condenser)│
    │              └────┬────┘
    │                   │ 4：低压湿蒸汽 (0.04 bar, h=793, x=0.28)
    │                   ↓
    │              ┌─────────┐
    │   W_P        │   泵     │  ← 把冷凝水打回高压，耗功 W_P
    └──────────────│  (Pump)  │
                   └────┬────┘
                        │ 1：高压饱和液 (30 bar, h=1008)
                        └──→ 回到锅炉，再来一圈
```

**逐段走一遍**：

| 步骤 | 设备             | 发生了什么                         | 入口状态         | 出口状态         | Q      | W        |
| ---- | ---------------- | ---------------------------------- | ---------------- | ---------------- | ------ | -------- |
| 4→1 | **泵**     | 把低压湿蒸汽**加压**打回高压 | 0.04 bar, x=0.28 | 30 bar, 饱和液   | 0      | 耗功 215 |
| 1→2 | **锅炉**   | 高压水**吸热**变成干饱和蒸汽 | 30 bar, 饱和液   | 30 bar, 干饱和汽 | +1795  | 0        |
| 2→3 | **涡轮**   | 高温高压蒸汽**膨胀做功**     | 30 bar, 干饱和汽 | 0.04 bar, x=0.72 | 0      | 产功 940 |
| 3→4 | **冷凝器** | 乏汽**排热**冷凝，回到低压   | 0.04 bar, x=0.72 | 0.04 bar, x=0.28 | −1070 | 0        |

> **原理**：这不是卡诺循环——1→2 和 3→4 是**等压**（不是等温），所以是**兰金循环（Rankine Cycle）**。为什么不用卡诺？因为卡诺要求两相区等温压缩（4→1），湿蒸汽压缩极难实现，兰金循环用泵压液体（容易得多），虽然效率略低于卡诺，但工程上可行。这是全世界火力发电厂和核电站用的循环。
>
> ⚠️ **隐藏前提：四个点全部在饱和区。** 点 1 饱和液（x=0）、点 2 干饱和汽（x=1）在钟形边界上，点 3（x=0.72）和点 4（x=0.28）在钟形内部（湿蒸汽）。没有一个点进入过热区或压缩液区——所以全程只靠饱和表（h_f、h_g）就能打，不需要过热表。T 和 P 全程锁死，知道 P 就知道 T。

---

#### 计算

> **题干**：蒸汽动力循环在 30 bar 和 0.04 bar 之间运行。已知：
>
> - 30 bar：$h_f = 1008$ kJ/kg，$h_g = 2803$ kJ/kg，$T_H \approx 507$ K
> - 0.04 bar：$h_f = 121$ kJ/kg，$h_g = 2554$ kJ/kg，$h_{fg} = 2433$ kJ/kg，$T_L \approx 302$ K
> - 涡轮出口干度 $x_3 = 0.716$，泵入口干度 $x_4 = 0.276$
>
> 求：涡轮功、泵功、锅炉吸热、冷凝器排热、净功、效率、功比、蒸汽消耗率。

**第 1 步**：确定各点焓值。

| 点 | 位置                 | h 怎么来                                           | h [kJ/kg]      |
| -- | -------------------- | -------------------------------------------------- | -------------- |
| 1  | 泵出口（锅炉入口）   | $h_f$(30 bar)                                    | 1008           |
| 2  | 锅炉出口（涡轮入口） | $h_g$(30 bar)                                    | 2803           |
| 3  | 涡轮出口             | $h_f + x_3 h_{fg}$(0.04 bar) = 121 + 0.716×2433 | **1863** |
| 4  | 泵入口（冷凝器出口） | $h_f + x_4 h_{fg}$(0.04 bar) = 121 + 0.276×2433 | **793**  |

**第 2 步**：逐段写第一定律（$q - w = \Delta h$，忽略 KE/PE）。

**涡轮（2→3）**：$q = 0$（绝热），蒸汽膨胀推叶片做功

$$
w_{23} = h_2 - h_3 = 2803 - 1863 = 940 \text{ kJ/kg}
$$

> 每 kg 蒸汽流过涡轮，焓从 2803 跌到 1863，跌掉的 940 kJ 全变成了轴功。h 降 = 涡轮发的电。涡轮是赚钱的。

**泵（4→1）**：$q = 0$（绝热），外界耗功把水打到高压

$$
w_{41} = h_4 - h_1 = 793 - 1008 = -215 \text{ kJ/kg}
$$

> 每 kg 水被泵从 0.04 bar 打到 30 bar，焓从 793 涨到 1008。焓涨 215 kJ 来自外界输入的电功，所以 $w$ 为负（功进入系统）。泵是花钱的——这笔钱必须从涡轮赚的里面扣。

**锅炉（1→2）**：$w = 0$，高压水吸热变成高压蒸汽

$$
q_{12} = h_2 - h_1 = 2803 - 1008 = 1795 \text{ kJ/kg}
$$

> 每 kg 水在锅炉里从饱和液烧成干饱和汽，焓从 1008 涨到 2803。涨的 1795 kJ 就是烧的燃料热——这是循环的"燃料账单"，效率的分母。

**冷凝器（3→4）**：$w = 0$，乏汽排热给冷却水，冷凝降压

$$
q_{34} = h_4 - h_3 = 793 - 1863 = -1070 \text{ kJ/kg}
$$

> 每 kg 乏汽在冷凝器里从 x=0.72 的湿蒸汽凝成 x=0.28 的湿蒸汽，焓从 1863 跌到 793。跌掉的 1070 kJ 全作为废热排给了冷却水——这笔热要不回来，就是热机必须排的 $Q_L$。

**第 3 步**：汇总各指标。

---

**净功 $W_{net}$**——涡轮产的电，扣掉泵吃掉的电：

$$
W_{net} = W_T + W_P = 940 + (-215) = 725 \text{ kJ/kg}
$$

> 每 kg 蒸汽在涡轮里产 940 kJ，但泵要吃掉 215 kJ 把水打回高压——净赚 725。循环就是这样：涡轮做"大功"，泵做"小功"，差值才是能用的。

---

**效率 $\eta$**——花的 heat 里有多少变成了净功：

$$
\eta = \frac{W_{net}}{Q_{in}} = \frac{725}{1795} = 0.404 = 40.4\%
$$

- $Q_{in} = q_{12} = 1795$ kJ/kg（锅炉里烧火给蒸汽的热）
- $W_{net} = 725$ kJ/kg（最终拿到手的功）

> 这就是热机效率公式 $\eta = W/Q_H$ 的直接应用。锅炉烧了 1795 的热，最后变成 725 的功，剩下 1070 作为废热从冷凝器排走了。跟卡诺效率 $\eta_C = 1 - 302/507 = 40.4\%$ 完全一致——因为这个兰金循环的加热和排热正好在 $T_H$ 和 $T_L$ 两个等温线上，碰巧等于卡诺效率。

**第一定律验证**——蒸汽跑完一整圈，必须回到原点。

第一定律对完整循环：$\oint \delta Q = \oint \delta W$（因为 $\oint dU = 0$，状态函数一周净变化为零）。

算热端：

$$
Q_{net} = q_{12} + q_{34} = 1795 + (-1070) = 725 \text{ kJ/kg}
$$

算功端：

$$
W_{net} = w_{23} + w_{41} = 940 + (-215) = 725 \text{ kJ/kg}
$$

$$
Q_{net} = W_{net} = 725 \text{ kJ/kg} \quad \checkmark
$$

> 锅炉烧进去 1795，冷凝器排走 1070，净剩 725 的热 → 全部变成了 725 的净功。蒸汽跑回起点，U 不变，进来的能量全出去了——要么变功、要么排废热。验算必须过这关，不过就说明哪里算错了。

---

**功比 $r_w$**：

$$
r_w = \frac{W_{net}}{W_{turbine}} = \frac{W_T - W_P}{W_T}
$$

- $W_{turbine}$（$W_T$）= 涡轮总产功
- $W_{pump}$（$W_P$）= 泵耗功
- $W_{net} = W_T - W_P$ = 净输出功

> **物理含义**：涡轮发的电，有多少是"纯利润"？功比 0.77 意味着涡轮发 100 度电，泵吃了 23 度，净赚 77 度。功比越接近 1 → 辅机耗功占比越小，循环越经济。

**SSC（Specific Steam Consumption，蒸汽消耗率）**：

$$
SSC = \frac{3600}{W_{net}} \quad [\text{kg/kWh}]
$$

- $W_{net}$ = 净功 [kJ/kg]
- 3600 = 1 kWh 的千焦数（1 kWh = 3600 kJ）

> **物理含义**：每发一度电需要多少公斤蒸汽流过涡轮。SSC 越小 → 同样的蒸汽能发更多电 → 循环越优秀。本题 SSC = 4.97，意味着每度电要烧近 5 公斤蒸汽。

> ⚠️ **注意**：此题虽然挂着"卡诺"的名，但 1→2 和 3→4 是**等压**（不是等温）——实际上是**兰金循环（Rankine Cycle）**。兰金循环是真实电厂用的循环，它放弃了卡诺的两相区等温压缩（太难实现），改用泵压缩液体。

### 7.9 常见疑问：为什么点 1 和 2 刚好在饱和线上？

**Q：四个点覆盖了哪些相态？**

| 点 | 相态     | 干度 x | T-v 图上位置                                |
| -- | -------- | ------ | ------------------------------------------- |
| 1  | 饱和液   | 0      | 钟形**左边界**（30 bar 等压线左端点） |
| 2  | 干饱和汽 | 1      | 钟形**右边界**（30 bar 等压线右端点） |
| 3  | 湿蒸汽   | 0.72   | 钟形**内部**（0.04 bar 等压线上）     |
| 4  | 湿蒸汽   | 0.28   | 钟形**内部**（0.04 bar 等压线上）     |

**Q：点 1 是液体，为什么有 P 和 h？跟水压什么关系？**

这里的 P 就是热力学压力，跟 h = u + Pv 里的 P 是同一个东西。泵把水从 0.04 bar 打到 30 bar，水的热力学压力就变成了 30 bar。它跟"自来水水压"的区别只是大小——物理含义完全一样。h_f(30 bar) = 1008 kJ/kg 是 30 bar 下刚好要沸腾的液态水所具有的焓。

**Q：压力和温度的分界点是谁？**

泵和涡轮就是压力的分界点：泵提压（0.04→30），涡轮降压（30→0.04）。锅炉和冷凝器在各自等压下运行。

**Q：为什么不用过热蒸汽？为什么不压到压缩液体区？**

这是个**考试简化**——最简单的理想兰金循环只用饱和表数据，假设锅炉出口刚好到饱和汽（点 2）、泵出口刚好到饱和液（点 1）。真实电厂**一定会过热**（加热到过热区），因为：

- 干饱和汽一膨胀就变湿（你看点 3 的 x=0.72，28% 已经是水滴），水滴冲击涡轮叶片 → 侵蚀
- 过热蒸汽膨胀完还能保持干态，保护涡轮，效率也更高
- 题目省掉过热，就是为了让计算只依赖饱和表，不碰过热表——考试友好，工程失真

> **题目的"鸡贼"之处**：它把电厂的设计思想（兰金循环、四件套串联、饱和态做端点）当成已知前提，不再解释。考试现推容易懵，但一旦理解了这个模式，所有蒸汽动力循环题都是同一个套路：找四点 h → 逐段写 q−w=Δh → 汇总。

**Q：点 3 和点 4 的 x<1，不在饱和线上，为什么还能用饱和表？**

点 3 和点 4 **在钟形内部**——这叫**湿蒸汽**（两相混合物），不是"非饱和"。它是饱和液滴和饱和蒸汽的**热力学平衡混合物**：每一个液滴都是饱和液（跟点 1 同款），每一缕蒸汽都是饱和汽（跟点 2 同款），只是混在一起。

日常起雾 ≠ 湿蒸汽——起雾是空气里水汽没饱和就凝结了，不在平衡态。湿蒸汽里 T 和 P 锁死在饱和值，液和汽和平共处。

所以 x 加权有效：

$$
h = (1-x) \cdot h_f + x \cdot h_g
$$

比如点 3：28% 质量的饱和液（h=121）+ 72% 质量的饱和汽（h=2554）→ 混合物总 h = 0.28×121 + 0.72×2554 = 1863。不需要"两侧边界"——混合物内部的两相本身就是边界状态。

## 八、熵（Entropy, S）——了解即可

### 8.1 什么是熵？

> **熵 $S$**：衡量能量分散程度的物理量。有序系统熵低，无序系统熵高。

自然过程趋向于熵增加——冰融化、气体扩散、热冷混合达到均匀温度，都是熵增过程。

### 8.2 熵变计算

对于可逆等温过程：

$$
\Delta S = \frac{Q}{T}
$$

- $\Delta S$ = 熵变 [kJ/K]
- $Q$ = 热量 [kJ]
- $T$ = 温度 [K]

### 8.3 第二定律的熵表述

对于孤立系统：

$$
\Delta S_{\text{isolated}} \geq 0
$$

> 熵可以产生但不能消灭。自然过程总是使总熵增加。可逆过程是极限情况（$\Delta S = 0$）。

> PPT 说学生不需要推导熵的方程，只需要知道：熵是衡量过程方向的性质，自然过程趋向熵增。

## 九、本篇小结

### 核心概念

| 概念          | 要点                        |
| ------------- | --------------------------- |
| 第二定律      | 过程有方向，热机效率有上限  |
| Kelvin-Planck | 热机不能 100% 效率          |
| Clausius      | 热量不能自发从冷到热        |
| 热源/热汇     | 高温/低温储库               |
| 热机          | 热 → 功，必须排废热        |
| 制冷机        | 功 → 把热从冷搬到热        |
| 热泵          | 同制冷机，但目的是制热      |
| 可逆过程      | 理想化，无损失              |
| 卡诺循环      | 最高效循环，2 等温 + 2 绝热 |
| 熵            | 衡量无序度，孤立系统熵增    |

### 必记公式

| 公式                                                        | 含义                 |
| ----------------------------------------------------------- | -------------------- |
| $\eta_{\text{th}} = 1 - \frac{Q_L}{Q_H}$                  | 热机效率             |
| $\eta_{\text{Carnot}} = 1 - \frac{T_L}{T_H}$              | 卡诺效率（理论上限） |
| $\text{COP}_R = \frac{Q_L}{W} = \frac{T_L}{T_H - T_L}$    | 制冷机 COP           |
| $\text{COP}_{HP} = \frac{Q_H}{W} = \frac{T_H}{T_H - T_L}$ | 热泵 COP             |
| $\Delta S = \frac{Q}{T}$                                  | 可逆等温过程熵变     |

> **考试提醒**：所有公式中的温度必须用**开尔文 K**！
