# 第六篇：热力学第二定律

> **对应讲义**：Lect 09
>
> **前置知识**：第一篇（热量方向）、第二篇（第一定律、效率）、第五篇（循环、开口系统设备）
>
> 第一定律告诉你"能量守恒"，但不告诉你能量**能不能自发转化**。一杯咖啡放在房间里，第一定律不反对咖啡变热、房间变冷（总能量不变），但现实中永远不会发生——这就是第二定律管的事：**过程的方向**和**效率的上限**。

---

## 一、为什么需要第二定律？

### 1.1 第一定律的局限

第一定律只说"能量不生不灭"，但有些过程虽然满足能量守恒，却永远不会自发发生：

- 热咖啡放在冷房间里 → 咖啡变凉（✅ 会发生）
- 冷房间里的热量自动集中到咖啡里 → 咖啡变热（❌ 不会发生）
- 电通过电阻发热（✅ 会发生）
- 热量自动变成电（❌ 不会发生）

第一定律对这两个方向都不禁止，但现实只选一个方向。**第二定律就是告诉你哪个方向是可能的。**

### 1.2 第二定律的三个用途

1. **预测过程方向**：热量从热到冷，不会反过来
2. **确定效率上限**：热机效率不可能 100%，最高多少？
3. **定义温度标度**：与物质无关的热力学温标

---

## 二、热源与热汇

### 2.1 定义

> **热能储库（Thermal Energy Reservoir）**：一个理想化的物体，能吸收或释放有限热量而自身温度不变。

- **热源（Source, $T_H$）**：高温储库，提供热量
- **热汇（Sink, $T_L$）**：低温储库，吸收废热

例子：海洋、湖泊、大气——因为热容量巨大，吸收/释放一点热量温度几乎不变。

> 后面的热机和制冷机都需要两个储库：一个高温、一个低温。

---

## 三、热机（Heat Engine）

### 3.1 什么是热机？

> **热机**：把热能转化为功的循环设备。

工作流程：

1. 从高温热源 $T_H$ 吸收热量 $Q_H$
2. 把一部分热量转化为功 $W$
3. 把剩余废热 $Q_L$ 排到低温热汇 $T_L$
4. 循环回到起点

> **为什么要排热？** 因为第二定律（Kelvin-Planck 陈述）规定：不可能只从一个热源吸热就全部变成功。必须有一部分热量排到低温端，循环才能完成。

### 3.2 热效率（Thermal Efficiency）

$$
\eta_{\text{th}} = \frac{W_{\text{net}}}{Q_H} = \frac{Q_H - Q_L}{Q_H} = 1 - \frac{Q_L}{Q_H}
$$

- $W_{\text{net}}$ = 净输出功 [kJ]
- $Q_H$ = 从高温源吸收的热量 [kJ]
- $Q_L$ = 排到低温汇的废热 [kJ]

> 热效率永远小于 1（即小于 100%）。因为 $Q_L > 0$（必须排热），所以 $\eta < 1$。

> **注意**：这里 $Q_H$ 和 $Q_L$ 都取绝对值（正数），不考虑符号约定。$W_{\text{net}} = Q_H - Q_L$。

---

## 四、第二定律的两种陈述

### 4.1 Kelvin-Planck 陈述（针对热机）

> **不可能**造出一个循环运行的热机，只从一个热源吸热并**全部**转化为功。

通俗说：**热机效率不可能 100%**。你必须排一部分废热到低温端。

> 这不是摩擦导致的——即使没有摩擦，理论上也不可能 100%。这是自然规律。

### 4.2 Clausius 陈述（针对制冷机）

> **不可能**造出一个循环运行的设备，不消耗功就把热量从低温传到高温。

通俗说：**热量不会自发从冷到热**。要让热量"逆流"（从冷到热），必须输入功（比如压缩机的电）。

### 4.3 两种陈述的等价性

两种陈述看似不同，但实际上是等价的——违反其中一个，必然违反另一个。

|        | Kelvin-Planck          | Clausius                 |
| ------ | ---------------------- | ------------------------ |
| 针对   | 热机（产功）           | 制冷机/热泵（耗功）      |
| 说的是 | 不能 100% 把热变功     | 不能不花功就让热从冷到热 |
| 共同点 | 自然过程有方向，不可逆 | 同左                     |

## 五、制冷机和热泵

### 5.1 制冷机（Refrigerator）

> **制冷机**：把热量从低温空间搬到高温空间的设备（你家冰箱）。

工作流程：

1. 从低温冷库吸收热量 $Q_L$（制冷效果）
2. 输入功 $W$（压缩机耗电）
3. 把 $Q_H = Q_L + W$ 排到高温环境

### 5.2 制冷机的性能系数（COP）

制冷机不用"效率"而用**性能系数**（Coefficient of Performance）：

$$
\text{COP}_R = \frac{Q_L}{W} = \frac{Q_L}{Q_H - Q_L} = \frac{T_L}{T_H - T_L} \quad \text{（卡诺制冷机）}
$$

- $Q_L$ = 从冷库吸收的热量（你想要的）
- $W$ = 输入的功（你花的代价）

> **COP 可以大于 1！** 因为 $Q_L$ 可以大于 $W$——你花 1 kJ 的电，可能搬运 3 kJ 的热量。这跟热机效率不同。

### 5.3 热泵（Heat Pump）

> **热泵**：跟制冷机原理完全一样，但**目的不同**——制冷机目的是制冷（吸 $Q_L$），热泵目的是制热（排 $Q_H$）。

$$
\text{COP}_{HP} = \frac{Q_H}{W} = \frac{Q_H}{Q_H - Q_L} = \frac{T_H}{T_H - T_L} \quad \text{（卡诺热泵）}
$$

> 冬天用空调制热就是热泵——从室外冷空气吸热，排到室内。COP > 1 意味着比直接用电加热更高效。

## 六、可逆与不可逆过程

### 6.1 为什么要区分？

可逆过程是理想化的**理论上限**——真实过程都是不可逆的，但可逆过程给出了最好的可能性能。

### 6.2 定义

> **可逆过程**：可以完全逆转，不留任何痕迹的过程。系统和外界都恢复到初始状态。

> **不可逆过程**：无法完全逆转的过程。"不可逆"= 不能回到之前的状态。

### 6.3 不可逆性的来源

任何真实过程都包含以下不可逆因素（至少一种）：

- 摩擦
- 有限温差下的热传递
- 自由膨胀（气体向真空中膨胀）
- 流体混合
- 电阻发热
- 非弹性变形
- 化学反应

> 一个过程只要包含以上任何一种，就是不可逆的。

## 七、卡诺循环（Carnot Cycle）

### 7.1 什么是卡诺循环？

> **卡诺循环**：理论上最高效的热机循环。完全可逆，无摩擦，无损失。

由四个可逆过程组成：

| 阶段 | 过程类型                | 发生了什么                                   |
| ---- | ----------------------- | -------------------------------------------- |
| 1→2 | 等温膨胀 ($T_H$ 不变) | 从高温源吸热$Q_H$，气体膨胀做功            |
| 2→3 | 绝热膨胀                | 气体继续膨胀做功，温度从$T_H$ 降到 $T_L$ |
| 3→4 | 等温压缩 ($T_L$ 不变) | 外界压缩气体，排热$Q_L$ 到低温汇           |
| 4→1 | 绝热压缩                | 外界继续压缩，温度从$T_L$ 升回 $T_H$     |

> P-V 图上，卡诺循环是一个由两条等温线和两条绝热线围成的封闭曲线。围成的面积 = 净输出功。

### 7.2 卡诺效率

$$
\eta_{\text{Carnot}} = 1 - \frac{T_L}{T_H}
$$

- $T_H$ = 高温热源温度 [K]（**必须用开尔文！**）
- $T_L$ = 低温热源温度 [K]

> **卡诺效率是所有热机的理论上限**。没有任何热机在相同温度区间内能超过卡诺效率。

### 7.3 例子

沸水（100°C = 373 K）做热源，冰水（0°C = 273 K）做冷源：

$$
\eta_{\text{Carnot}} = 1 - \frac{273}{373} = 0.268 = 26.8\%
$$

6 atm 压力下沸水（约 280°C = 553 K）做热源：

$$
\eta_{\text{Carnot}} = 1 - \frac{273}{553} = 0.506 = 50.6\%
$$

> 温差越大 → 效率越高。这就是为什么发电厂要用高温高压蒸汽。

## 八、熵（Entropy, S）——了解即可

### 8.1 什么是熵？

> **熵 $S$**：衡量能量分散程度的物理量。有序系统熵低，无序系统熵高。

自然过程趋向于熵增加——冰融化、气体扩散、热冷混合达到均匀温度，都是熵增过程。

### 8.2 熵变计算

对于可逆等温过程：

$$
\Delta S = \frac{Q}{T}
$$

- $\Delta S$ = 熵变 [kJ/K]
- $Q$ = 热量 [kJ]
- $T$ = 温度 [K]

### 8.3 第二定律的熵表述

对于孤立系统：

$$
\Delta S_{\text{isolated}} \geq 0
$$

> 熵可以产生但不能消灭。自然过程总是使总熵增加。可逆过程是极限情况（$\Delta S = 0$）。

> PPT 说学生不需要推导熵的方程，只需要知道：熵是衡量过程方向的性质，自然过程趋向熵增。

## 九、本篇小结

### 核心概念

| 概念          | 要点                        |
| ------------- | --------------------------- |
| 第二定律      | 过程有方向，热机效率有上限  |
| Kelvin-Planck | 热机不能 100% 效率          |
| Clausius      | 热量不能自发从冷到热        |
| 热源/热汇     | 高温/低温储库               |
| 热机          | 热 → 功，必须排废热        |
| 制冷机        | 功 → 把热从冷搬到热        |
| 热泵          | 同制冷机，但目的是制热      |
| 可逆过程      | 理想化，无损失              |
| 卡诺循环      | 最高效循环，2 等温 + 2 绝热 |
| 熵            | 衡量无序度，孤立系统熵增    |

### 必记公式

| 公式                                                        | 含义                 |
| ----------------------------------------------------------- | -------------------- |
| $\eta_{\text{th}} = 1 - \frac{Q_L}{Q_H}$                  | 热机效率             |
| $\eta_{\text{Carnot}} = 1 - \frac{T_L}{T_H}$              | 卡诺效率（理论上限） |
| $\text{COP}_R = \frac{Q_L}{W} = \frac{T_L}{T_H - T_L}$    | 制冷机 COP           |
| $\text{COP}_{HP} = \frac{Q_H}{W} = \frac{T_H}{T_H - T_L}$ | 热泵 COP             |
| $\Delta S = \frac{Q}{T}$                                  | 可逆等温过程熵变     |

> **考试提醒**：所有公式中的温度必须用**开尔文 K**！
