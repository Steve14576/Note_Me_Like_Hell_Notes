> 考试速查用。按主题分类，每条公式标注适用条件和注意事项。
>
> **符号约定**：$Q > 0$ 吸热，$W > 0$ 系统对外做功，$\Delta U = Q - W$。

## 零、能量体系从头推演——E → U → Q,W → 第一定律 → PV → H

### 0.1 能量（E）：起点

$$E = \underbrace{U}_{\text{内能}} + \underbrace{KE}_{\text{动能}} + \underbrace{PE}_{\text{势能}}$$

- 状态函数，任何状态都有
- 闭口系统 $KE, PE$ 不变时 $\Delta E = \Delta U$

### 0.2 内能（U）：存了多少

- $U$ = 分子动能 + 分子势能，状态函数
- $\Delta U = U_2 - U_1$（状态函数的差，路径无关）
- 理想气体：$U = U(T)$，只跟温度有关
- 没法直接测 → 用比热 $c_v$ 间接算

### 0.3 热量（Q）与功（W）：能量怎么进出

| | $Q$ | $W$ |
|---|---|---|
| 驱动力 | 温差 | 力 × 距离 |
| 状态函数？ | **不是**（路径函数） | **不是**（路径函数） |
| 能写 $\Delta$？ | **不能**（$\Delta Q$ 是错的） | **不能** |

> **$\Delta$ 只能用在状态函数上**（$U, H, T, P, v$）。$Q$ 和 $W$ 是过程量，没有"初值减终值"。

### 0.4 第一定律：把 Q, W, U 串起来

$$\boxed{Q - W = \Delta U}$$

- 左边：路径函数（$Q, W$ 各自跟路径有关）
- 右边：状态函数（$\Delta U$ 跟路径无关）
- **两个路径函数的差 = 状态函数的变化**——第一定律的本质

### 0.5 边界功：W 的具体化

$$W_b = \int P\,dV \quad \text{（从 } F \times ds = PA \times ds = P \times dV \text{ 推）}$$

- 路径函数——P-V 图曲线下面积，不同路径不同
- 定压：$W_b = P\Delta V$；定容：$W_b = 0$

### 0.6 PV：一个数学乘积，不同上下文不同含义

| 场景 | $PV$ 含义 |
|------|-----------|
| 理想气体方程 | $PV = mRT$（状态方程） |
| 定压闭口系统 | $P\Delta V$ = 边界功 |
| 开口系统 | $Pv$ = 流动功（推质量穿边界） |
| 焓定义 | $H = U + PV$（纯组合） |

> 同一个 $PV$，不是天生属于某种功——是数学乘积，物理含义取决于上下文。

### 0.7 焓（H）：为什么要打包 $U + PV$

**构造动机**：定压闭口系统中，第一定律：

$$Q = \Delta U + P\Delta V = \Delta(U + PV) = \Delta H$$

定压时 $P$ 不变 → $\Delta(PV) = P\Delta V$ = 边界功 → 热量恰好等于焓变。

$$\boxed{H = U + PV}$$

- 状态函数（$U, P, V$ 都是 → 组合还是）
- 不绑定过程，任何状态都有 $H$
- 定压闭口系统中 $Q = \Delta H$（特别方便）
- 开口系统中 $h = u + Pv$ = 内能 + 流动功（流体携带的总能量）

**闭口 vs 开口对比**：$q - w = \Delta u$（闭口，用 $u$）vs $q - w = \Delta h$（开口，用 $h$），差别就是 $Pv$。

### 0.8 $\Delta$ 完整梳理

| 量 | 能写 $\Delta$？ | 原因 |
|---|---|---|
| $U, H$ | ✅ | 状态函数 |
| $T, P, v$ | ✅ | 状态函数 |
| $Q, W$ | ❌ | 路径函数（过程量） |

$$\Delta H = \Delta U + \Delta(PV)$$

理想气体：$\Delta(PV) = mR\Delta T$ → $\Delta h = \Delta u + R\Delta T$

### 0.9 比热：把 $\Delta U, \Delta H$ 跟温度连起来

| 比热 | 定义 | 为什么 |
|------|------|--------|
| $c_v = \frac{\partial u}{\partial T}$ | 定容时 $q = \Delta u$（$W = 0$） | 热量全进内能 |
| $c_p = \frac{\partial h}{\partial T}$ | 定压时 $q = \Delta h$（$=\Delta u + P\Delta v$） | 热量进内能 + 边界功 |

$$\Delta u = c_v \Delta T, \quad \Delta h = c_p \Delta T \quad \text{（理想气体近似）}$$

$$c_p - c_v = R \quad \text{（从 } h = u + RT \text{ 对 } T \text{ 求导）}$$

### 0.10 全链条

```
E（总能）= U + KE + PE
              ↓
         U（内能）——状态函数，可写 Δ
              ↓
    Q - W = ΔU（第一定律，闭口）
    ↑      ↑
   热量    功（路径函数，不能写 Δ）
              ↓
         W_b = ∫P dV（边界功，闭口）
              ↓
         PV（数学乘积，上下文决定含义）
              ↓
    H = U + PV（焓，状态函数，可写 Δ）
              ↓
    定压闭口：Q = ΔH | 开口：Q̇ - Ẇ = ṁ·ΔH
              ↓
    cv = ∂u/∂T, cp = ∂h/∂T（比热，连温度）
              ↓
    cp - cv = R（理想气体）
```

## 一、基本概念

### 1.1 系统分类

| 中文 | 英文 | 特征 |
|------|------|------|
| 闭口系统 | Closed System | 质量不进出，边界可动 |
| 开口系统 | Open System / Control Volume | 质量可进出，边界固定 |
| 绝热系统 | Adiabatic System | $Q = 0$ |

### 1.2 性质分类

| 中文 | 英文 | 特征 | 例子 |
|------|------|------|------|
| 强度性质 | Intensive Property | 分成两半值不变 | $T, P, \rho, v$ |
| 广延性质 | Extensive Property | 分成两半值减半 | $m, V, U, H, S$ |
| 比性质 | Specific Property | 广延量 ÷ 质量 = 强度量 | $v, u, h, s$ |

### 1.3 "比"与"率"归一化

| 符号规则 | 含义 | 例子 |
|----------|------|------|
| 大写字母 | 广延量（总量） | $V, U, H, Q, W$ |
| 小写字母 | 比量（÷ 质量） | $v = V/m,\; u = U/m,\; h = H/m$ |
| 字母上加点 | 率（÷ 时间） | $\dot{Q}, \dot{W}, \dot{m}, \dot{V}$ |

**公式配对规律**：大写换小写 = 全式 ÷ $m$，公式结构不变。

## 二、第一定律（能量守恒）

### 2.1 闭口系统

| 公式 | 说明 |
|------|------|
| $Q - W = \Delta U$ | 第一定律（广延量） |
| $q - w = \Delta u$ | 第一定律（比量，÷$m$） |

### 2.2 开口系统（稳态流动）

| 公式 | 说明 |
|------|------|
| $\dot{Q} - \dot{W} = \dot{m}\left(h_2 - h_1 + \frac{V_2^2 - V_1^2}{2} + g(z_2 - z_1)\right)$ | 完整形式 |
| $\dot{Q} - \dot{W} = \dot{m}(h_2 - h_1)$ | 简化形式（忽略 KE, PE） |
| $q - w = \Delta h$ | 比形式（÷$\dot{m}$） |

### 2.3 闭口 vs 开口对比

| | 闭口系统 | 开口系统 |
|---|---|---|
| 能量方程 | $q - w = \Delta u$ | $q - w = \Delta h$ |
| 功 | 边界功 $w_b = \int P\,dv$ | 流动功 $w_{\text{flow}} = Pv$（打包进焓） |
| 用 $u$ 还是 $h$ | $u$ | $h = u + Pv$ |

## 三、功

### 3.1 边界功（Boundary Work）——闭口系统

| 公式 | 适用条件 |
|------|----------|
| $W_b = \int P\,dV$ | 一般形式（准静态过程） |
| $W_b = P(V_2 - V_1)$ | 等压过程 |
| $W_b = P_1 V_1 \ln\frac{V_2}{V_1} = mRT\ln\frac{P_1}{P_2}$ | 等温过程（理想气体） |
| $W_b = \frac{P_2 V_2 - P_1 V_1}{1 - n}$ | 多变过程（$n \neq 1$） |
| $W_b = \frac{P_2 V_2 - P_1 V_1}{1 - k}$ | 绝热过程（$n = k$） |
| $W_b = -mc_v(T_2 - T_1)$ | 绝热过程（用内能） |
| $W_b = 0$ | 等容过程 |

### 3.2 流动功（Flow Work）——开口系统

| 公式 | 说明 |
|------|------|
| $w_{\text{flow}} = Pv$ | 把 1 kg 流体推过边界所需的功 |

> 推导：力 = $PA$，距离 = $v/A$，功 = $PA \times v/A = Pv$。面积消掉。

### 3.3 边界功 vs 流动功

| | 边界功 | 流动功 |
|---|---|---|
| 什么在动 | 边界本身 | 质量穿过固定边界 |
| 体积变不变 | 变 | 不变 |
| 公式 | $\int P\,dV$（积分） | $Pv$（乘法） |
| 系统 | 闭口 | 开口 |
| 在方程里 | 作为 $W$ 单独出现 | 打包进焓 $h = u + Pv$ |

## 四、焓与比热

### 4.1 焓

| 公式 | 说明 |
|------|------|
| $h = u + Pv$ | 焓定义（状态函数，不绑定过程） |
| $H = U + PV$ | 广延量形式 |

### 4.2 比热

| 公式 | 说明 |
|------|------|
| $c_v = \left(\frac{\partial u}{\partial T}\right)_v$ | 定容比热（定容时 $q = \Delta u$） |
| $c_p = \left(\frac{\partial h}{\partial T}\right)_p$ | 定压比热（定压时 $q = \Delta h$） |

### 4.3 比热关系

| 公式 | 说明 |
|------|------|
| $c_p - c_v = R$ | 理想气体比热差（$R$ 来自 $Pv=RT$ 求导） |
| $k = \frac{c_p}{c_v}$ | 比热比 |
| $c_v = \frac{R}{k-1}$ | 由 $k$ 和 $R$ 反算 |
| $c_p = \frac{kR}{k-1}$ | 由 $k$ 和 $R$ 反算 |

### 4.4 理想气体 $\Delta u$ 和 $\Delta h$

| 公式 | 说明 |
|------|------|
| $\Delta u = c_v \Delta T$ | 近似（$c_v$ 为常数） |
| $\Delta h = c_p \Delta T$ | 近似（$c_p$ 为常数） |
| $\Delta u = \int c_v(T)\,dT$ | 精确 |

## 五、纯物质性质

### 5.1 相变与饱和

| 公式 / 关系 | 说明 |
|-------------|------|
| $P_{\text{sat}} \leftrightarrow T_{\text{sat}}$ | 一一对应（知道一个就知道另一个） |
| $Q_{\text{latent}} = \Delta U + P\Delta V$ | 潜热 = 断键（~87%）+ 膨胀做功（~13%） |

### 5.2 干度与插值

| 公式 | 说明 |
|------|------|
| $x = \frac{m_g}{m_f + m_g}$ | 干度定义（气相质量分数） |
| $v = v_f + x \cdot v_{fg}$ | 混合物比容 |
| $u = u_f + x \cdot u_{fg}$ | 混合物比内能 |
| $h = h_f + x \cdot h_{fg}$ | 混合物比焓 |
| $v_{fg} = v_g - v_f$ | 饱和差值 |

> 下标：$f$ = fluid（饱和液体），$g$ = gas（饱和蒸气），$fg$ = $g - f$

### 5.3 理想气体方程

| 公式 | 说明 |
|------|------|
| $Pv = RT$ | 比容形式（每千克） |
| $PV = mRT$ | 总量形式 |
| $R = \frac{R_u}{M}$ | 特定气体常数 = 通用常数 ÷ 摩尔质量 |
| $\frac{P_1 V_1}{T_1} = \frac{P_2 V_2}{T_2}$ | 两状态关系（定质量） |

### 5.4 压缩因子

| 公式 | 说明 |
|------|------|
| $Pv = ZRT$ | 真实气体方程 |
| $P_r = \frac{P}{P_{\text{cr}}}$ | 对比压力（reduced，无量纲） |
| $T_r = \frac{T}{T_{\text{cr}}}$ | 对比温度（reduced，无量纲） |

> 查通用压缩因子图：知道 $P_r, T_r$ → 读 $Z$ → 代入 $Pv = ZRT$。

## 六、多变过程

### 6.1 过程方程

$$Pv^n = \text{const}$$

### 6.2 各过程参数

| 过程 | 英文 | $n$ | 边界功 | 特征 |
|------|------|-----|--------|------|
| 等压 | Isobaric | $0$ | $P\Delta V$ | $P$ 不变 |
| 等温 | Isothermal | $1$ | $mRT\ln(V_2/V_1)$ | $T$ 不变 |
| 绝热 | Adiabatic | $k$ | $\frac{P_2V_2 - P_1V_1}{1-k}$ | $Q = 0$ |
| 等容 | Isochoric | $\infty$ | $0$ | $V$ 不变 |
| 多变 | Polytropic | $n$ | $\frac{P_2V_2 - P_1V_1}{1-n}$ | 一般情况 |

### 6.3 T-P-V 关系

$$\frac{T_2}{T_1} = \left(\frac{P_2}{P_1}\right)^{\frac{n-1}{n}} = \left(\frac{V_1}{V_2}\right)^{n-1}$$

## 七、开口系统设备

### 7.1 质量守恒

| 公式 | 说明 |
|------|------|
| $\dot{m} = \rho A V$ | 质量流量 = 密度 × 面积 × 速度 |
| $\dot{m} = \frac{AV}{v}$ | 用比容表示 |
| $\dot{m}_{\text{in}} = \dot{m}_{\text{out}}$ | 稳态质量守恒 |

### 7.2 各设备能量方程

| 设备 | 英文 | 假设 | 能量方程 |
|------|------|------|----------|
| 喷管/扩压器 | Nozzle / Diffuser | 绝热、无功、忽略 PE | $h_1 + \frac{V_1^2}{2} = h_2 + \frac{V_2^2}{2}$ |
| 涡轮 | Turbine | 绝热、忽略 KE/PE | $\dot{W} = \dot{m}(h_1 - h_2)$ |
| 压缩机 | Compressor | 绝热、忽略 KE/PE | $\dot{W}_{\text{in}} = \dot{m}(h_2 - h_1)$ |
| 泵 | Pump | 同压缩机（处理液体） | $\dot{W}_{\text{in}} = \dot{m}(h_2 - h_1)$ |
| 节流阀 | Throttling Valve | 绝热、无功、忽略 KE/PE | $h_1 = h_2$（等焓） |
| 热交换器 | Heat Exchanger | 无功、忽略 KE/PE | $\dot{Q} = \dot{m}(h_2 - h_1)$ |

## 八、第二定律

### 8.1 热机与效率

| 公式 | 说明 |
|------|------|
| $\eta_{\text{th}} = \frac{W_{\text{net}}}{Q_H} = 1 - \frac{Q_L}{Q_H}$ | 热机热效率 |
| $\eta_{\text{Carnot}} = 1 - \frac{T_L}{T_H}$ | 卡诺效率（理论上限） |

> $T$ 必须用开尔文 K！

### 8.2 制冷机与热泵

| 公式 | 说明 |
|------|------|
| $\text{COP}_R = \frac{Q_L}{W} = \frac{Q_L}{Q_H - Q_L} = \frac{T_L}{T_H - T_L}$ | 制冷机 COP |
| $\text{COP}_{HP} = \frac{Q_H}{W} = \frac{Q_H}{Q_H - Q_L} = \frac{T_H}{T_H - T_L}$ | 热泵 COP |

> COP 可以 > 1。卡诺 COP 用开尔文 K。

### 8.3 熵

| 公式 | 说明 |
|------|------|
| $\Delta S = \frac{Q}{T}$ | 可逆等温过程熵变 |
| $\Delta S_{\text{isolated}} \geq 0$ | 孤立系统熵增原理 |

### 8.4 第二定律陈述

| 陈述 | 中文 | 针对对象 |
|------|------|----------|
| Kelvin-Planck | 不可能只从一个热源吸热全部变功 | 热机 |
| Clausius | 不可能不花功让热从冷到热 | 制冷机 |

### 9.1 导热

| 公式 | 说明 |
|------|------|

| $R_{\text{cond}} = \frac{L}{kA}$ | 导热热阻 |
| $R_{\text{cyl}} = \frac{\ln(r_2/r_1)}{2\pi kL}$ | 圆柱壁热阻 |

### 9.2 对流

| 公式 | 说明 |
|------|------|
| $\dot{Q} = hA_s(T_s - T_\infty)$ | 牛顿冷却定律 |
| $R_{\text{conv}} = \frac{1}{hA}$ | 对流热阻 |

### 9.3 辐射

| 公式 | 说明 |
|------|------|
| $E_b = \sigma T_s^4$ | Stefan-Boltzmann 定律（黑体） |
| $\dot{Q} = \sigma A_s T_s^4$ | 黑体总辐射功率 |
| $\dot{Q}_{\text{net}} = \varepsilon \sigma A_s (T_s^4 - T_{\text{sur}}^4)$ | 净辐射换热 |
| $\lambda_{\text{max}} T = b$ | Wien 位移定律 |
| $\varepsilon = \alpha$ | 基尔霍夫定律（热平衡时） |
| $\alpha + \rho + \tau = 1$ | 吸收 + 反射 + 透射 = 1 |

### 9.4 热阻网络

| 公式 | 说明 |
|------|------|
| $\dot{Q} = \frac{\Delta T}{R_{\text{total}}}$ | 类比欧姆定律 |
| $R_{\text{total}} = R_1 + R_2 + \cdots$ | 串联 |
| $\frac{1}{R_{\text{total}}} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots$ | 并联 |
