## 一、热力学是什么？

### 1.1 一句话定义

Thermodynamics = therme（热量）+ dynamis（力量）

> **热力学**：研究**热**和**温度**，以及它们跟**能量**和**功**之间关系的物理学分支。

说白了就是研究"能量怎么从一种形式变成另一种形式"。

工程师用热力学解决两类问题：
- **最少投入**：烧开一壶水最少需要多少电？让房间冷下来最少需要多少能量？
- **最大产出**：烧一吨煤最多能发多少电？一箱油最多能跑多远？

### 1.2 能量是什么？

> **能量（Energy）**：引起变化的能力。

更具体的说法：能量是在系统做功时、或系统吸热/放热时，传递进/出系统的"那个东西"。

你高中已经学过两种能量：
- **动能（Kinetic Energy, KE）**：物体因为整体运动而具有的能量
- **势能（Potential Energy, PE）**：物体因为位置（高度）而具有的能量

热力学里要引入第三种——**内能（Internal Energy, U）**：物体内部微观粒子（分子、原子）具有的能量总和。这是热力学独有的概念，后面专门讲。

三兄弟加起来就是系统的**总能量**：

$$E = U + KE + PE$$

- $E$ = 总能量 [J]（焦耳）
- $U$ = 内能 [J]——微观层面的能量，本篇第四节展开
- $KE$ = 动能 [J]——宏观运动的能量，你高中就学过
- $PE$ = 势能 [J]——宏观位置的能量，你也学过

> 这里先记住有这三个。KE 和 PE 你已经熟了，U 会在本篇后面专门讲。

### 1.3 能量守恒原理

> **能量守恒原理**：能量不能被创造或消灭，只能从一种形式转化为另一种形式。

这就是**热力学第一定律**（First Law of Thermodynamics）的雏形。PPT 用鸟的例子说明：

- 吃食物摄入 5 单位能量
- 飞行消耗 4 单位
- 存储到体内 1 单位
- 5 = 4 + 1，收支平衡

第一定律的完整公式 $\Delta U = Q - W$ 会在第二篇正式讲。这里只需要知道：**能量守恒是热力学的底层逻辑**。

### 1.4 热力学第二定律（预告）

> **第二定律**：能量不仅有数量，还有品质（质量）。热量自发地从高温流向低温。

你泡了杯 70°C 的咖啡放在 20°C 的房间里，咖啡会变凉——这就是第二定律在起作用。完整内容在第六篇讲，这里先留个印象。

### 1.5 两种研究方法

| | 经典热力学 (Classical) | 统计热力学 (Statistical) |
|---|---|---|
| 视角 | 宏观（把整个系统当作整体看） | 微观（看大量粒子的平均行为） |
| 需不需要知道粒子细节？ | 不需要 | 需要 |
| 本课程用哪个？ | **主要用经典热力学** | — |

本课程基本不涉及统计热力学。你不需要知道分子层面在干什么，只需要用宏观可测量的量（温度、压力、体积等）来分析系统。

## 二、单位制——热力学的"语言"

### 2.1 为什么要单独讲单位？

热力学计算中，**90% 的计算错误来自单位不统一**。温度忘了开尔文、压力忘了加大气压、kJ 和 J 混用……这些都是最常见的失分点。所以先把单位搞清楚，后面少踩很多坑。

### 2.2 基本量纲（Primary Dimensions）

物理量可以用少数几个基本量纲来描述。SI 国际单位制选了四个作为基本量：

| 基本量 | 符号 | SI 单位 |
|--------|------|---------|
| 质量 mass | $m$ | 千克 [kg] |
| 长度 length | $L$ | 米 [m] |
| 时间 time | $t$ | 秒 [s] |
| 温度 temperature | $T$ | 开尔文 [K] |

### 2.3 导出量纲（Secondary Dimensions）

其他物理量都可以用基本量纲组合出来，叫做导出量纲：

| 物理量 | 量纲 | SI 单位 | 怎么来的 |
|--------|------|---------|---------|
| 面积 | $[L]^2$ | m² | 长度×长度 |
| 体积 | $[L]^3$ | m³ | 面积×长度 |
| 速度 | $[L][t]^{-1}$ | m/s | 长度÷时间 |
| 加速度 | $[L][t]^{-2}$ | m/s² | 速度÷时间 |
| 力 | $[m][L][t]^{-2}$ | N = kg·m/s² | 质量×加速度 |
| 能量 | $[m][L]^2[t]^{-2}$ | J = N·m | 力×距离 |
| 功率 | $[m][L]^2[t]^{-3}$ | W = J/s | 能量÷时间 |
| 压力 | $[m][L]^{-1}[t]^{-2}$ | Pa = N/m² | 力÷面积 |

> 知道这些推导关系，就能自己检查单位是否匹配。比如算完一个力发现单位是 kg·m²/s²，那肯定是算错了（那是能量的单位）。

## 三、力、功、能量——高中物理快速复习

### 3.1 力（Force）

$$F = m \times a$$

- $F$ = 力 [N]（牛顿）
- $m$ = 质量 [kg]
- $a$ = 加速度 [m/s²]

1 牛顿的定义：让 1 kg 的物体产生 1 m/s² 加度的力。

### 3.2 功（Work）

$$W = F \times s$$

- $W$ = 功 [J]（焦耳）
- $F$ = 力 [N]
- $s$ = 位移 [m]

功是**标量**（只有大小，没有方向）。1 焦耳 = 1 N 的力让物体移动 1 m 所做的功。

> **注意**：热力学中"功"的含义比高中物理更广。除了力×位移，还有其他形式的功（电功、轴功、边界功等），但本质上都是"能量传递的一种方式"。后面会逐一讲到。

### 3.3 功的符号约定——极其重要！

热力学中功有正负号，代表能量传递的方向：

| 情况 | 符号 | 含义 |
|------|------|------|
| 系统**对外**做功（BY） | $W > 0$（正） | 能量从系统流出 |
| 外界**对系统**做功（ON） | $W < 0$（负） | 能量流入系统 |

> **记忆口诀**：BY（系统做的）= 正，ON（对系统做的）= 负。

> **符号约定警告**：这是**工程热力学**的约定。化学课程中可能用相反的符号（化学约定 $W > 0$ 表示外界对系统做功）。本课程以 PPT 为准，使用工程约定。上网查资料时也要注意区分。

这个符号约定直接决定了后面第一定律公式写成 $\Delta U = Q - W$（减号就是因为 $W > 0$ 表示能量流出）。

### 3.4 动能（Kinetic Energy, KE）

$$KE = \frac{1}{2} m V^2$$

- $KE$ = 动能 [J]
- $m$ = 质量 [kg]
- $V$ = **速度** [m/s]

> **变量名警告**：这里的 $V$ 是**速度**（velocity），不是体积（volume）！在热力学中 $V$ 大写经常表示体积，所以极其容易混淆。PPT 特别提醒了这一点。
>
> **怎么区分**：看上下文——如果跟 $\frac{1}{2}m$ 搭配就是速度，如果跟 $P$（压力）搭配（比如 $PV = mRT$）就是体积。在容易混淆的地方，PPT 有时用不同符号表示速度来避免歧义。

### 3.5 势能（Potential Energy, PE）

$$PE = m g z$$

- $PE$ = 势能 [J]
- $m$ = 质量 [kg]
- $g$ = 重力加速度 $\approx 9.81$ m/s²
- $z$ = **高度** [m]

> **为什么用 $z$ 而不用 $h$？** 你高中习惯用 $h$ 表示高度，但热力学中 $h$ 被预留给了**焓**（enthalpy）——一个在第五篇（开口系统）会讲的重要概念。为了避免冲突，热力学中统一用 $z$（来自三维坐标系的 z 轴方向）表示高度。

### 3.6 计算例题

**例1**：力 $F = 24$ N，位移 $s = 6$ m，求功。

$$W = F \times s = 24 \times 6 = 144 \text{ J}$$

**例2**：质量 $m = 40$ kg，从 $z_1 = 0$ 提升到 $z_2 = 2$ m，$g = 9.8$ m/s²，求克服重力做的功（=势能变化）。

$$W = F \times z = mg \times z = 40 \times 9.8 \times 2 = 784 \text{ J}$$

**例3**：汽车 $m = 1000$ kg，从 0 km/h 加速到 90 km/h，求动能变化。

先换算单位：$90 \text{ km/h} = \frac{90}{3.6} = 25 \text{ m/s}$

$$\Delta KE = \frac{1}{2} m V^2 - 0 = \frac{1}{2} \times 1000 \times 25^2 = 312500 \text{ J} = 312.5 \text{ kJ}$$

> PPT 在这道题旁边写了两个条件："一定有一个力作用在边界上，边界必须移动。"这其实是在说做功的前提条件——后面讲闭口系统边界功时会用到。

## 四、内能（Internal Energy, U）

### 4.1 什么是内能？

> **内能 $U$**：系统中所有微观粒子（分子、原子）所具有的能量总和。

你在宏观上看到的物体是静止的，但微观层面，分子在不停地平动、振动、旋转，分子之间有相互作用力……这些微观运动携带的能量全部加在一起，就是内能。

PPT 的原话：内能是**微观层面的能量储存**。

### 4.2 内能怎么变？

- 给系统**加热**（$Q > 0$）→ 内能增加
- **对系统做功**（比如压缩气体，$W < 0$）→ 内能增加
- 系统**对外做功**（比如气体膨胀，$W > 0$）→ 内能减少
- 系统**放热**（$Q < 0$）→ 内能减少

### 4.3 内能 vs 动能 vs 势能

| 能量 | 符号 | 来源 | 层面 |
|------|------|------|------|
| 内能 | $U$ | 分子运动、振动、分子间作用力等 | **微观** |
| 动能 | $KE$ | 整个物体的宏观运动 | 宏观 |
| 势能 | $PE$ | 整个物体的宏观位置（高度） | 宏观 |

> PPT 说："You don't need to remember these different types of energies, just that they exist."——不需要记住内能具体包含哪些子类型（平动动能、振动能、转动能、化学键能……），只需要知道：**内能是微观层面所有能量的总和**。

### 4.4 比内能

就像体积 $V$ 除以质量得到比容 $v$，内能 $U$ 除以质量就得到**比内能** $u$：

$$u = \frac{U}{m}$$

- $u$ = 比内能 [J/kg 或 kJ/kg]
- $U$ = 内能 [J 或 kJ]
- $m$ = 质量 [kg]

> **大写 = 总量（广延），小写 = 每千克（强度）**——这个规律贯穿全课程。

### 4.5 总能量

一个系统的总能量 = 内能 + 动能 + 势能：

$$E = U + KE + PE$$

在大多数热力学问题中，系统整体不动（$KE$ 不变）、高度不变（$PE$ 不变），所以能量变化主要来自内能变化 $\Delta U$。这就是为什么**内能是热力学的核心量**。

> 记忆小贴士：PPT 用 "YOU (U) need lots of internal energy" 来帮你记住 $U$ = 内能。

## 五、系统和边界——你研究的那块东西

### 5.1 三个基本概念

| 概念 | 英文 | 定义 | 通俗解释 |
|------|------|------|---------|
| **系统** | System | 选定用于研究的一团物质或一块空间 | "我要盯着的那个东西" |
| **外界** | Surroundings | 系统之外的一切 | "系统以外的东西" |
| **边界** | Boundary | 分隔系统和外界的那层面 | "系统跟外界的分界线" |

边界可以是**固定的**（比如刚性容器的壁），也可以是**可移动的**（比如气缸里的活塞）。

边界可以是**真实的**物理表面（容器壁），也可以是**假想的**（在空间中画一个虚拟范围来研究——开口系统常这样做）。

### 5.2 闭口系统（Closed System）

> **闭口系统**：没有质量穿过边界，但能量可以穿过。

什么意思？——东西（质量）出不去也进不来，但热量可以透过容器壁传入传出，功可以通过活塞传递。

**例子**：密封的活塞气缸。气体出不去（质量守恒），但可以通过缸壁吸热/放热，活塞可以移动做功。

### 5.3 开口系统（Open System / Control Volume）

> **开口系统**：质量和能量都可以穿过边界。

开口系统也叫**控制体积**（Control Volume），因为你通常是在空间中画一个虚拟的"体积"来研究，物质流过这个体积。

**例子**：涡轮机。空气从一头进去，从另一头出来（质量穿过边界），同时输出轴功（能量穿过边界）。

> **为什么要区分闭口和开口？** 因为两种系统用的能量方程不同：
> - 闭口系统：$\Delta U = Q - W$（第二篇讲）
> - 开口系统：用到**焓** $h = u + Pv$（第五篇讲）
>
> 这是本课程最核心的分类，贯穿始终。拿到题目第一件事就是判断"这是闭口还是开口系统"。

### 5.4 孤立系统（Isolated System）

> **孤立系统**：既没有质量交换，也没有能量交换。

是闭口系统的特殊情况——连能量都不穿过边界。完全孤立的系统在现实中不存在，但作为理想化模型很有用（比如热力学第二定律分析）。

### 5.5 稳态（Steady State）

> **稳态**：开口系统的一种操作条件——进入的质量流量 = 离开的质量流量，且系统内部所有属性不随时间变化。

"稳态"是一个**假设/条件**，不是系统类型。如果题目说"稳态操作"或"steady state"，意味着你要加上以下假设：
1. **质量不积累**：$\dot{m}_{\text{in}} = \dot{m}_{\text{out}}$（进去多少出来多少）
2. **能量不积累**：系统内部总能量不随时间变化（$\frac{dE}{dt} = 0$）
3. 所有属性（$T, P, v$ 等）在空间每个点都不随时间变化

> 稳态假设让开口系统的方程大大简化。后面第五篇会大量使用。

## 六、性质——描述系统的"标签"

### 6.1 什么是性质？

> **性质**（Property）：系统的任何可测量的特征量。

熟悉的性质有：质量 $m$、体积 $V$、温度 $T$、压力 $P$、密度 $\rho$。

性质的特点：它只取决于系统当前的状态，跟系统怎么到达这个状态的无关。

### 6.2 强度性质 vs 广延性质——最重要的区分

| | 强度性质 (Intensive) | 广延性质 (Extensive) |
|---|---|---|
| 定义 | 与系统质量**无关** | 与系统质量**有关**（取决于系统大小） |
| 例子 | 温度 $T$、压力 $P$、密度 $\rho$ | 质量 $m$、体积 $V$、内能 $U$ |
| 判断方法 | 把系统分成两半，值**不变** | 把系统分成两半，值**减半** |

> **判断技巧**：一杯 20°C 的水倒成两杯，每杯还是 20°C（温度是强度性质），但每杯的质量和体积都减半了（质量和体积是广延性质）。

### 6.3 比性质（Specific Properties）

> **比性质**：广延性质除以质量后得到的值。用**小写字母**表示。

$$\text{比性质} = \frac{\text{广延性质}}{m}$$

比性质除掉了质量的影响，所以它变成了**强度性质**。

最重要的例子：

| 广延性质（大写） | 比性质（小写） | 关系 | 单位 |
|---|---|---|---|
| 体积 $V$ | 比容 $v$ | $v = \frac{V}{m}$ | m³/kg |
| 内能 $U$ | 比内能 $u$ | $u = \frac{U}{m}$ | kJ/kg |
| 焓 $H$ | 比焓 $h$ | $h = \frac{H}{m}$ | kJ/kg |
| 总能量 $E$ | 比总能 $e$ | $e = \frac{E}{m}$ | kJ/kg |
| 热量 $Q$ | 比热量 $q$ | $q = \frac{Q}{m}$ | kJ/kg |
| 功 $W$ | 比功 $w$ | $w = \frac{W}{m}$ | kJ/kg |
| 熵 $S$ | 比熵 $s$ | $s = \frac{S}{m}$ | kJ/(kg·K) |

> **大写 = 总量（广延），小写 = 每千克（强度）**。记住这个规律，后面看到 $h$ 就知道是比焓，看到 $u$ 就知道是比内能。

### 6.4 "率"——除以时间（字母上加点）

还有一套归一化：**字母上方加点 = 除以时间**（每秒），把总量变成"率"。

| 总量 | 率（加点） | 关系 | 单位 |
|---|---|---|---|
| 热量 $Q$ | 热传递率 $\dot{Q}$ | $\dot{Q} = Q/t$ | W, kW |
| 功 $W$ | 功率 $\dot{W}$ | $\dot{W} = W/t$ | W, kW |
| 质量 $m$ | 质量流量 $\dot{m}$ | $\dot{m} = m/t$ | kg/s |
| 体积 $V$ | 体积流量 $\dot{V}$ | $\dot{V} = V/t$ | m³/s |

> **两套归一化可以叠加**：比如 $\dot{Q} = \dot{m} \cdot q$（热传递率 = 质量流量 × 比热量），把 ÷时间 和 ÷质量 串起来了。开口系统大量用这种组合。

### 6.5 公式配对：大写换小写 = 全式 ÷m

**核心规律：把广延版公式中所有大写换成小写（全式除以 $m$），公式结构不变，依然成立。**

| 公式 | 广延版（大写，总量） | 比版（小写，÷m） |
|------|-------------------|-----------------|
| 第一定律（闭口） | $Q - W = \Delta U$ | $q - w = \Delta u$ |
| 焓定义 | $H = U + PV$ | $h = u + Pv$ |
| 总能 | $E = U + \frac{1}{2}mV^2 + mgz$ | $e = u + \frac{V^2}{2} + gz$ |
| 动能 | $KE = \frac{1}{2}mV^2$ | $ke = \frac{V^2}{2}$ |
| 势能 | $PE = mgz$ | $pe = gz$ |
| 理想气体 | $PV = mRT$ | $Pv = RT$ |
| 内能变化 | $\Delta U = mc_v\Delta T$ | $\Delta u = c_v\Delta T$ |
| 焓变化 | $\Delta H = mc_p\Delta T$ | $\Delta h = c_p\Delta T$ |
| 边界功 | $W = m\int P\,dv$ | $w = \int P\,dv$ |
| 流动功 | $W_{\text{flow}} = PV$ | $w_{\text{flow}} = Pv$ |

> **记忆诀窍**：
> 1. 大写 = 总量，小写 = 每千克，加点 = 每秒
> 2. 大写换小写 = 全式 ÷$m$，公式结构不变
> 3. $R$、$c_v$、$c_p$、$P$、$T$ 本身就是强度量，不需要变换
> 4. **注意**：$V$（体积）→ $v$（比容）时要 ÷$m$，但 $V$（速度）不变（速度本身就是强度量）

### 6.6 密度与比容

$$\rho = \frac{m}{V} \quad \text{（密度 = 质量 ÷ 体积）}$$

$$v = \frac{V}{m} = \frac{1}{\rho} \quad \text{（比容 = 体积 ÷ 质量 = 密度的倒数）}$$

- $\rho$ = 密度 [kg/m³]（读作 "rho"，希腊字母 ρ）
- $v$ = 比容 [m³/kg]（小写 v）
- $m$ = 质量 [kg]
- $V$ = 体积 [m³]（大写 V）

> 比容 $v$ 和密度 $\rho$ 互为倒数。知道一个就能算另一个。比容在热力学中用得非常多（比如 P-V 图的横轴就是 $v$），要特别熟悉。

**比容 vs 密度的物理意义区别**：

- 密度：给定一块空间，里面塞了多少东西——从"空间"出发看物质
- 比容：给定 1 kg 东西，它占了多大地方——从"物质"出发看空间

**为什么热力学偏爱比容而不是密度？**

1. **直觉舒服**：气体膨胀 → $v$ 增大 → P-v 图上往右走（"变大就是右"）。用密度的话膨胀反而往左走，别扭。
2. **公式自然**：$Pv = RT$（干净）vs $P/\rho = RT$（密度在分母里）；边界功 $\int P \, dv$（直接积分）vs $\int P \, d(1/\rho)$（要先变换）。
3. **配套"每千克"视角**：热力学大量使用比性质（$u, h, w, q$ 都是 kJ/kg），比容 [m³/kg] 跟它们单位配套。密度 [kg/m³] 不配套。

常见材料密度（20°C）：

| 材料 | 密度 [g/cm³] | 密度 [kg/m³] |
|------|-------------|-------------|
| 豆油 | ~0.91 | ~910 |
| 水 | 1.00 | 1000 |
| 铝 | ~2.7 | ~2700 |
| 钢 | ~7.9 | ~7900 |

## 七、状态与平衡

### 7.1 状态（State）

> **状态**：系统在某一时刻，所有性质都有确定值时的描述。

热力学只研究**平衡状态**（equilibrium state），因为只有平衡状态下，性质才有确定的、统一的值，才能用来计算。

### 7.2 平衡（Equilibrium）

> **平衡**：一种"没有不平衡的驱动力"的状态——系统内部没有任何自发变化的趋势。

如果系统不平衡，各处的性质不一样（比如一头热一头冷），你就没法用一个温度来描述整个系统。所以热力学要求系统处于平衡状态才能分析。

有四种平衡：

| 平衡类型 | 条件 | 通俗解释 |
|---------|------|---------|
| 热平衡 (Thermal) | 系统各处**温度相同** | 没有热量自发流动 |
| 力学平衡 (Mechanical) | 系统各处**压力不随时间变化** | 没有压力差驱动运动 |
| 相平衡 (Phase) | 各**相**的质量不再变化 | 比如冰水混合物中冰不再融化、水不再结冰 |
| 化学平衡 (Chemical) | **化学成分**不随时间变化 | 没有化学反应发生 |

> 一个系统要达到**热力学平衡**，必须同时满足以上所有条件。本课程大部分时候默认系统处于平衡状态。

### 7.3 状态公设（State Postulate）

> **状态公设**：一个**简单可压缩系统**的状态，至少由**两个独立的强度性质**完全确定。

逐个解释关键词：

**"简单可压缩系统"是什么？**——不涉及电、磁、重力运动、表面张力效应的系统。说白了就是普通的流体或气体在容器中的情况。本课程涉及的基本都是简单可压缩系统。

**"独立"是什么意思？**——两个性质不能由彼此推导出来。比如对于理想气体，$P$、$v$、$T$ 之间满足 $Pv = RT$，知道了其中两个就能算出第三个。所以任意两个是"独立"的，但三个里只有两个是独立的。

**"完全确定"是什么意思？**——一旦确定两个独立强度性质，系统的**所有其他性质**（$P, T, v, u, h$ 等）就都确定了。这就是为什么查物性表时只需要两个参数。

常见的状态确定方式：
- 压力 $P$ + 温度 $T$ → 查表确定状态
- 压力 $P$ + 比容 $v$ → 查表确定状态
- 温度 $T$ + 比容 $v$ → 查表确定状态

## 八、过程与循环

### 8.1 过程（Process）

> **过程**：系统从一个平衡状态变到另一个平衡状态。

比如气缸里的气体从状态 1（$P_1, T_1$）变到状态 2（$P_2, T_2$），这就是一个过程。

### 8.2 路径（Path）

> **路径**：过程中系统经历的一系列状态的连线。

从状态 1 到状态 2，可以走不同的路径。比如先等温再等压，或者先等压再等温——初末状态相同，但路径不同，做的功和热量也不同。

> 功和热量是**路径函数**（取决于路径），而内能是**状态函数**（只取决于初末状态）。这是第二篇的核心内容。

### 8.3 准静态过程（Quasistatic / Quasiequilibrium Process）

> **准静态过程**：过程进行得无限缓慢，系统在每个时刻都**无限接近平衡状态**。

**为什么要引入这个概念？** 热力学只能分析平衡状态，但过程是"变化"，变化意味着不平衡。准静态过程是一个理想化假设——变化足够慢，以至于系统在每个瞬间都可以当作平衡状态来处理。这样我们才能：
- 在 P-V 图上画出连续的曲线（每个点都是一个平衡状态）
- 用 $W = \int P \, dV$ 来计算功（第四篇讲）

**实际中什么时候可以当准静态？**——如果过程进行得足够慢（比如活塞缓慢移动让气体有时间重新分布均匀），就可以近似当作准静态过程处理。如果过程很快（比如爆炸），就不能用准静态假设。

### 8.4 P-V 图——热力学最重要的图

> **P-V 图**：纵轴是压力 $P$，横轴是比容 $v$（或体积 $V$），用来展示系统的状态和过程的图。

**为什么用 $P$ 和 $v$ 做坐标轴？**
- $P$（压力）和 $v$（比容）是两个独立的强度性质，根据状态公设，确定这两个就能确定状态
- 对于气体系统，$P$ 和 $V$ 的变化最能反映能量转换——P-V 图上曲线下方的面积 = 做的功（$W = \int P \, dV$，第四篇详细讲）

**怎么读 P-V 图？**
- 图上的一个点 = 一个平衡状态（一组 $P, v$ 值）
- 图上的一条线 = 一个过程（从状态 1 到状态 2 的路径）
- 线下方的面积 = 这个过程中做的功

> P-V 图在第四篇（闭口系统）会大量使用，这里先知道它是什么、怎么读就行。

### 8.5 特殊过程

根据过程中保持不变的量，给过程命名：

| 过程名 | 英文 | 保持不变的量 | 记忆方法 |
|--------|------|------------|---------|
| 等温过程 | Isothermal | 温度 $T$ | "thermal" = 热/温度 |
| 等压过程 | Isobaric | 压力 $P$ | "bar" 是压力单位 |
| 等容过程 | Isochoric | 比容 $v$ | "choric" = 容积 |
| 绝热过程 | Adiabatic | 热量 $Q = 0$（不换热） | 第四篇讲 |

> "Iso-" 前缀来自希腊语，意思是"相同的/不变的"。所以 Iso-thermal = 温度相同，Iso-baric = 压力相同，Iso-choric = 体积相同。

### 8.6 循环（Cycle）

> **循环**：初态和终态完全相同的过程。系统经历一系列变化后回到起点。

循环的意义：很多工程设备（如热机、制冷机）都是循环工作的——工质反复经历相同的状态变化，持续输出功或搬运热量。

> 循环中 $\Delta U = 0$（因为回到起点，内能不变）。这是第六篇分析热机和制冷机的基础。

## 九、第零定律——温度为什么能比较？

### 9.1 第零定律

> **热力学第零定律**：如果两个物体分别与第三个物体处于热平衡，那么这两个物体也彼此处于热平衡。

听起来像是废话？但它确立了一个非常重要的事实：**温度是一个可以统一比较的物理量**。

### 9.2 为什么叫"第零"定律？

因为这个定律最基本（没有它，温度的概念都不成立），但被认识到的重要性最晚。第一定律和第二定律已经编号了，只好叫"第零"。

### 9.3 实际意义

第零定律告诉我们：用温度计（第三个物体）分别接触两个系统，如果读数相同，就说明两个系统温度相同——即使它们没有直接接触。

**这就是温度计的工作原理。**

## 十、温度与压力——两个最重要的性质

### 10.1 温度标度

| 标度 | 符号 | 单位 | 用途 |
|------|------|------|------|
| 摄氏温标 | $T$ (°C) | °C | 日常生活 |
| 开尔文温标 | $T$ (K) | K | **热力学计算必须用 K** |

换算关系：

$$T(\text{K}) = T(°\text{C}) + 273.15$$

> **考试铁律**：所有热力学公式中的温度**必须用开尔文 K**。用 °C 代入公式是最常见的错误之一。

两个基准点（都是 1 atm 压力下）：
- **冰点**（Ice point）：冰水混合物与空气在 1 atm 下的平衡温度 = 0°C = 273.15 K
- **沸点**（Steam point）：纯水与水蒸气在 1 atm 下的平衡温度 = 100°C = 373.15 K

> **补充**：PPT 还提到了卡路里（Calorie）作为能量的替代单位：1 cal = 4.18681 J，定义为让 1 g 水从 14.5°C 升到 15.5°C 所需能量。考试中可能遇到，知道换算就行。

### 10.2 压力标度

> **压力**（Pressure）：流体作用在单位面积上的法向力（垂直于面的力）。

$$P = \frac{F}{A}$$

- $P$ = 压力 [Pa = N/m²]
- $F$ = 法向力 [N]
- $A$ = 面积 [m²]

> 1 Pa 很小，工程上常用 kPa（千帕）或 MPa（兆帕）。1 atm ≈ 101.325 kPa。

压力有三种表述方式，搞清楚它们的关系非常重要：

| 类型 | 定义 | 关系 |
|------|------|------|
| 绝对压力 $P_{\text{abs}}$ | 相对于绝对真空（零压力）的真实压力 | $P_{\text{abs}} = P_{\text{gauge}} + P_{\text{atm}}$ |
| 表压 $P_{\text{gauge}}$ | 压力表读数（= 绝对压力 − 大气压） | $P_{\text{gauge}} = P_{\text{abs}} - P_{\text{atm}}$ |
| 真空压力 $P_{\text{vac}}$ | 低于大气压时的读数（= 大气压 − 绝对压力） | $P_{\text{vac}} = P_{\text{atm}} - P_{\text{abs}}$ |

> **考试铁律**：所有热力学公式中的压力**必须用绝对压力**。压力表读的是表压，必须加上大气压（约 101.325 kPa）才是绝对压力。

用一条数轴帮你理解三者的关系：

```
绝对真空 ───────── 大气压 ───────── 系统压力
  (0 Pa)         (101.3 kPa)      (比如 300 kPa)

|←──────── 绝对压力 ──────────────→|  300 kPa
                  |←── 表压 ──→|     198.7 kPa

绝对真空 ───────── 大气压 ───────── 系统压力
  (0 Pa)         (101.3 kPa)      (比如 50 kPa)

|←──────── 绝对压力 ──────→|          50 kPa
                  |← 真空压力 →|       51.3 kPa
```

## 十一、本篇小结

### 核心概念速查

| 概念 | 一句话总结 |
|------|----------|
| 热力学 | 研究能量转换的科学 |
| 能量守恒 | 能量不生不灭，只转换形式（第一定律的雏形） |
| 内能 $U$ | 微观粒子能量总和，热力学核心量 |
| 总能量 $E = U + KE + PE$ | 内能 + 动能 + 势能 |
| 闭口系统 | 质量不进出，能量可以 |
| 开口系统 | 质量和能量都可以进出 |
| 孤立系统 | 质量和能量都不进出 |
| 稳态 | 开口系统中，进出平衡、属性不随时间变 |
| 强度性质 | 与质量无关（$T, P, \rho$） |
| 广延性质 | 与质量有关（$m, V, U$） |
| 比性质 | 广延性质 ÷ 质量 → 强度性质（小写字母） |
| 状态 | 所有性质都有确定值的平衡描述 |
| 平衡 | 没有不平衡驱动力的状态 |
| 状态公设 | 两个独立强度性质确定一个状态 |
| 过程 | 从一个平衡态到另一个 |
| 准静态过程 | 无限慢，每时刻都接近平衡 |
| 循环 | 初末态相同 |
| 第零定律 | 热平衡的传递性 → 温度可比较 |

### 必记公式

| 公式 | 含义 |
|------|------|
| $E = U + KE + PE$ | 总能量 = 内能 + 动能 + 势能 |
| $KE = \frac{1}{2}mV^2$ | 动能（$V$ = 速度，不是体积！） |
| $PE = mgz$ | 势能（$z$ = 高度，不用 $h$） |
| $W = F \times s$ | 功 = 力 × 位移 |
| $v = \frac{V}{m} = \frac{1}{\rho}$ | 比容 = 体积/质量 = 1/密度 |
| $u = \frac{U}{m}$ | 比内能 = 内能/质量 |
| $T(\text{K}) = T(°\text{C}) + 273.15$ | 温度换算 |
| $P_{\text{abs}} = P_{\text{gauge}} + P_{\text{atm}}$ | 绝对压力 = 表压 + 大气压 |

### 符号约定与易混淆点

| 符号 | 含义 | 注意 |
|------|------|------|
| $W > 0$ | 系统对外做功 | 工程约定（BY = 正） |
| $W < 0$ | 外界对系统做功 | 工程约定（ON = 负） |
| $V$ | **速度** 或 **体积** | 看上下文！跟 $\frac{1}{2}m$ 搭配 = 速度，跟 $P$ 搭配 = 体积 |
| $z$ | 高度 | 不用 $h$（$h$ = 焓，后面讲） |
| 大写字母 | 广延性质（总量） | $U, V, E$ |
| 小写字母 | 强度性质（每千克） | $u, v, e$ |
