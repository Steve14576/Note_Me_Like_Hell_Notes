# 查表完全指南——从此不再"欠约束"

> 热力学考试中，查表是挂科的头号杀手。不是算不对，是根本不知道从哪张表开始查、查出来的是什么、题干给的条件够不够。
>
> 这份指南解决三个问题：
>
> 1. 我手里有几张表？每张输入什么输出什么？
> 2. 题干给的条件够不够确定状态？（自由度判断）
> 3. 查出来的值往哪个公式里代？
>
> **前置知识**：第三篇（物质性质、相变、干度）、第一篇（状态、强度量/广延量）

---

## 一、你有几张表？

热力学里只有**四类表**，每张的用途完全不同：

| 表名                       | 输入              | 输出                                      | 适用场景                                                             |
| -------------------------- | ----------------- | ----------------------------------------- | -------------------------------------------------------------------- |
| **饱和表（按温度）** | T                 | \(P_{sat}, v_f, v_g, u_f, u_g, h_f, h_g\) | 已知 T，想知道沸点和两相性质                                         |
| **饱和表（按压力）** | P                 | \(T_{sat}, v_f, v_g, u_f, u_g, h_f, h_g\) | 已知 P，想知道沸点和两相性质                                         |
| **过热水蒸气表**     | P, T（T > T_sat） | \(v, u, h\)                               | 蒸气温度超过沸点，单相                                               |
| **压缩液体表**       | P, T（T < T_sat） | \(v, u, h\)                               | 液体温度低于沸点（考试通常用近似：≈ 同温饱和液的\(v_f, u_f, h_f\)） |

> **理想气体**（空气、氮气等）不用表——用 \(Pv = RT\)，\(h = c_pT\)，\(u = c_vT\)。

### 1.4 压缩液体表——长这样

> 输入：P = 5 MPa，T = 100°C。先查 T_sat(5 MPa) = 263.99°C → T < T_sat → 压缩液体。

**P = 5 MPa（\(T_{sat} = 263.99\)°C）**：

| \(T\) °C | \(v\) m³/kg | \(u\) kJ/kg | \(h\) kJ/kg |
| :-------: | :----------: | :---------: | :---------: |
|    20    |  0.0009995  |    83.65    |    88.65    |
|    100    |  0.0010410  |   417.52   |   422.72   |
|    200    |  0.0011530  |    848.1    |    853.9    |

> **怎么读**：P=5 MPa, T=100°C → h ≈ 422.72 kJ/kg。注意压缩液体 h 比同温饱和液 h_f(100°C)=419.04 稍大（因压力高，略有压缩功），但考试中常用 \(h \approx h_f(T)\) 近似，误差很小。

> **考试捷径**：大多数考试不要求查压缩液体原表——用"同温饱和液近似"即可：\(v \approx v_f(T)\), \(u \approx u_f(T)\), \(h \approx h_f(T)\)。

### 1.5 u, v, h, T, P 的自变量与因变量关系

**核心层级**：P、T、v 是"原始量"，u 和 h 是"派生量"——u 和 h 永远是被计算出来的，不是你直接测量的。

```
原始量（可直接测量）          派生量（永远靠算/查表）
   P ──┐                      u = f(T, v)  ──┐
   T ──┤  状态公设：任选两个    h = u + Pv  ──┤  查表读出
   v ──┘  锁死一切             h 也可直接 f(P,T)─┘
```

**各量的因果关系**：

| 量          | 角色               | 怎么得到                      | 备注                                                  |
| ----------- | ------------------ | ----------------------------- | ----------------------------------------------------- |
| **P** | 自变量             | 测出来（压力表）              | 最基本的输入之一                                      |
| **T** | 自变量             | 测出来（温度计）              | 最基本的输入之一                                      |
| **v** | 自变量或中间量     | 测出来（V/m）或从 P,T 算      | 单相区 v=f(P,T)；两相区 v 反推 x                      |
| **u** | **纯因变量** | 从 T,v（或 P,T）查表或公式    | **u 永远不直接测！** 对理想气体 u=u(T) 只依赖 T |
| **h** | **纯因变量** | h = u + Pv，或从 P,T 直接查表 | 定义量，打包了 u 和 Pv                                |

**为什么表按 P、T 组织？** 因为 P 和 T 是工程上最容易测量的两个量。你不可能拿个仪表直接读 u 或 h——它们永远靠查表或公式算出。所以表的设计就是：给你 P 和 T（你能测的），吐出 v, u, h（你不能测的）。

**不同物质类型的依赖关系**：

| 物质类型                   | u 依赖什么   | h 依赖什么   | v 依赖什么     |
| -------------------------- | ------------ | ------------ | -------------- |
| **理想气体**         | T            | T            | P, T           |
| **真实气体（单相）** | T, v         | P, T         | P, T（通过 Z） |
| **两相混合物**       | P（或 T）, x | P（或 T）, x | P（或 T）, x   |
| **压缩液体**         | ≈ T         | ≈ T         | ≈ T           |

**每种类型的公式/查表展开**：

| 类型                   | 公式                                        | 怎么查                                               |
| ---------------------- | ------------------------------------------- | ---------------------------------------------------- |
| **理想气体**     | \(u = c_v T\)，\(h = c_p T\)，\(v = RT/P\)  | 不用表！给\(c_v, c_p, R\) 直接算                     |
| **真实气体单相** | 无简单公式，u 和 h 是 (P,T) 的非线性函数    | 查过热表（T>T_sat）或压缩液表（T<T_sat）             |
| **两相混合物**   | \(y = y_f + x \cdot y_{fg}\)（y = v, u, h） | 查饱和表得\(y_f, y_g\) → 已知 x 插值；已知 y 反解 x |
| **压缩液体**     | \(y \approx y_f(T)\)（同温饱和液近似）      | **查饱和表（按 T）**，取 `y_f` 即可，不管 P。实际值略大于近似值（高压有微量压缩功），误差 < 1% |

> **关键心法**：拿到题目先问自己——"我需要 u 或 h 来代能量方程。u 和 h 依赖于什么？依赖于 P 和 T（或 x）。题目给我 P 和 T（或 x）了吗？给了 → 查表。没给 → 找另一个方程（能量守恒、质量守恒、过程约束）把缺的量解出来。"

### 1.6 函数视角：谁决定谁

用数学函数的形式，一目了然：

```
                    ┌─────────────────────┐
                    │   状态公设：2个独立    │
                    │   自变量 → 锁死一切   │
                    └─────────┬───────────┘
                              │
          ┌───────────────────┼───────────────────┐
          ▼                   ▼                   ▼
   理想气体              真实气体单相          两相混合物
          │                   │                   │
   u = u(T)            u = u(T, v)          P ⇔ T 互锁（非独立）
   h = h(T)            h = h(P, T)          u = u_f + x·u_fg
   v = RT/P            v = ZRT/P            h = h_f + x·h_fg
   Pv = RT             Z = Z(P, T)          v = v_f + x·v_fg
          │                   │                   │
          └───────────────────┴───────────────────┘
                              │
                    h = u + Pv（永远成立，定义式）
```

### 1.7 为什么 P-v-T 有三个轴却不冗余？

P-v-T 3D 图看起来像三个自由度，但所有真实状态只能落在一张**二维曲面**上——这张曲面就是状态方程 `f(P, v, T) = 0` 的几何化身。3 个坐标，1 个约束 → 2 个真正独立。

**类比**：地球表面。三个坐标（经度、纬度、海拔），但海拔 = f(经, 纬) → 只有 2 个独立自由度。画 3D 地形图是为了看清山的形状，不是为了多一个自由度。

|             | 地球表面         | P-v-T 曲面                         |
| ----------- | ---------------- | ---------------------------------- |
| 三个坐标    | 经度、纬度、海拔 | P、v、T                            |
| 一个约束    | 海拔 = f(经, 纬) | f(P, v, T) = 0                     |
| 真正自由度  | 2                | 2                                  |
| 3D 图的作用 | 看山的起伏       | 看相变拓扑（钟形、临界点、三重线） |

**为什么 2D 投影"够用"但会丢信息？**

| 投影   | 丢掉了什么                                             |
| ------ | ------------------------------------------------------ |
| P-v 图 | 不知道这条等压线上温度怎么变                           |
| T-v 图 | 不知道这条等温线上压力怎么变                           |
| P-T 图 | 两相钟形坍缩成一条线（饱和蒸气压曲线），干度信息全消失 |

**三重线**：3D 曲面上的固-液-气三线共存是一个线段。但在任意 2D 投影里坍缩成一个**点**（三重点）。没有 3D 视角，你永远看不到三重线的拓扑结构。

**3D 曲面就是状态公设的几何证明**：曲面没有折叠（单值），说明给定任意两个坐标，第三个唯一确定——这正是"2 个自由度锁死一切"的几何含义。

**但两相区有一个关键坑**：曲面虽不折叠，但 P 和 T 被锁在一起（等压即等温），所以 (P, T) 退化成 1 个自由度——这正是状态公设强调"**独立**"强度性质的原因。单相区任意两个独立；两相区 (P, T) 不独立，必须换 (P, x) 或 (P, v) 等。

## 二、核心心智：状态公设 → 自由度判断

> **状态公设**：简单可压缩物质的平衡态由**两个独立强度性质**完全确定。

这意味着：**只要你有任意两个独立的（P, T, v, u, h, x 中的两个），状态就锁死了，所有其他量都能查出来。** 你应该永远不感觉"欠约束"——如果觉得欠，说明你漏看了一个已知条件，或者没意识到某个条件等价于一个性质。

**自由度对照表**：

| 已知条件                  | 自由度                               | 能否求解  | 说明                                     |
| ------------------------- | ------------------------------------ | --------- | ---------------------------------------- |
| P + T（且 T > T_sat）     | 2（独立）                            | ✅        | 过热蒸气，直接查过热表                   |
| P + T（且 T < T_sat）     | 2（独立）                            | ✅        | 压缩液体，查压缩液表或用饱和近似         |
| P + T（且 T = T_sat）     | **1**（P, T 被锁定在一起了！） | ❌ 欠约束 | 需要 x 或 v/h/u 来定位                   |
| P + x                     | 2                                    | ✅        | 饱和混合物，查 P→饱和表→插值           |
| T + x                     | 2                                    | ✅        | 饱和混合物，查 T→饱和表→插值           |
| P + h（且 h_f < h < h_g） | 2                                    | ✅        | h 不是 P 的单值函数，可解 x              |
| 仅 P                      | 1                                    | ❌        | 除非默认饱和液体（x=0）或饱和蒸气（x=1） |
| "饱和液体"                | 2（隐含 x=0 + P 或 T 中的一个）      | ✅        | 沿着饱和液体线                           |
| "饱和蒸气"                | 2（隐含 x=1 + P 或 T 中的一个）      | ✅        | 沿着饱和蒸气线                           |

## 三、查表总流程

### 3.0 第一步——判断相态、选表（用 P 和 T，不用 h！）

**判断相态最快的方法是比较 P 和 T 跟饱和值的关系，不需要先查 h。**

```
题干给了我什么？
│
├─ 给了 P 和 T（最常见）
│   │
│   ├─ 查饱和表（按 P）→ 得 T_sat(P)
│   │   ├─ T > T_sat(P)  → 过热蒸气 → 查过热表（按 P 翻到对应页，找 T 那一行）
│   │   ├─ T < T_sat(P)  → 压缩液体 → 用同温饱和液近似 h ≈ h_f(T)
│   │   └─ T = T_sat(P)  → 正好在沸点上，但不知道干度 → 需要 x！
│   │
│   └─ 等价地：查饱和表（按 T）→ 得 P_sat(T)，比较 P 和 P_sat(T)
│
├─ 给了 P 和 x（或 T 和 x）
│   └─ 直接就是两相混合物 → 查对应饱和表 → 用 x 插值
│
├─ 给了 P 和 v（或 u 或 h）——没给 T
│   │
│   ├─ 查饱和表（按 P）→ 得 v_f, v_g
│   │   ├─ v < v_f → 压缩液体（没到沸点）
│   │   ├─ v_f ≤ v ≤ v_g → 两相混合物 → 算 x
│   │   └─ v > v_g → 过热蒸气 → 查过热表
│   │
│   └─ 同理用 u 或 h 跟 u_f/u_g 或 h_f/h_g 比较
│
└─ 只给了 P（或只给了 T）+ 描述词
    ├─ "饱和液体" → x=0
    ├─ "饱和蒸气" → x=1
    └─ "过热蒸气，P=2MPa, T=400°C" → 直接查过热表
```

> **核心**：有 P 和 T → 优先用 P/T 比大小，最快。没有 T 但手里有 v/u/h → 用 v/u/h 跟饱和值比大小，一样能判断。两种路径都行，取决于题干给了什么。比如喷管例题：能量方程先算出 h₂，没有 T₂ → 用 h₂ vs h_g 判断出过热，再查过热表反推 T₂。

### 3.1 完整查表流程

```
拿到一个状态 → 我需要查什么？
│
├─ 是理想气体？（题目说"空气"且远离两相区）
│   └─ Pv = RT,  h = c_p·T,  u = c_v·T  → 不需要表！
│
├─ 是水/水蒸气/R-134a？
│   │
│   ├─ 第 0 步：已知哪两个量？
│   │
│   ├─ 第 1 步：判断相态
│   │   │  已知 P 和 T：
│   │   │    查饱和表 → 得 T_sat(P) 或 P_sat(T)
│   │   │    T > T_sat(P)  → 过热蒸气 → 查过热表
│   │   │    T < T_sat(P)  → 压缩液体 → 用饱和液近似
│   │   │    T = T_sat(P)  → 在饱和线上 → 需要 x！
│   │   │
│   │   │  已知 P（或 T）和 v/u/h：
│   │   │    查饱和表 → 得 v_f, v_g（或 u_f, u_g 等）
│   │   │    v < v_f  → 压缩液体
│   │   │    v_f ≤ v ≤ v_g  → 两相混合物 → 算 x
│   │   │    v > v_g  → 过热蒸气
│   │
│   ├─ 第 2 步：按相态取数据
│   │    单相 → 直接查表读 v, u, h
│   │    两相 → 用 x 插值：y = y_f + x·(y_g − y_f)
│   │
│   └─ 第 3 步：查出来的 h 或 u 直接代入能量方程
```

## 四、考试中最容易"欠约束"的三种场景——及破解

| 场景                             | 为什么觉得欠          | 其实不缺                                                   |
| -------------------------------- | --------------------- | ---------------------------------------------------------- |
| 涡轮出口给了 P 和"干度 x=0.95"   | 以为只有 P 一个独立量 | P + x = 两个独立量！查饱和表 → 插值出 h₂                 |
| 节流阀给了入口 P,T，出口只给 P   | 以为出口缺一个量      | 节流阀 h₁=h₂！入口已知 → h₂ 已知 → P + h = 两个独立量 |
| 题目说"饱和液体进入"但没给 T     | 以为缺温度            | "饱和液体" = x=0，再加一个条件（如给了 P）就锁死           |
| 给了"过热蒸气，P=2MPa, T=400°C" | —                    | 两个独立量，直接查过热表，没毛病                           |
