## 符号速查表

| 符号 | 含义 | 单位 | 注意 |
|------|------|------|------|
| $P$ | 压力 | Pa, kPa | 公式中用绝对压力 |
| $T$ | 温度 | K | 公式中用开尔文 |
| $V$ | 体积 **或** 速度 | m³ 或 m/s | 看上下文！ |
| $v$ | 比容 | m³/kg | 小写 = 每千克 |
| $U, u$ | 内能 / 比内能 | J, J/kg | 大写=总量，小写=每千克 |
| $H, h$ | 焓 / 比焓 | J, J/kg | $h = u + Pv$ |
| $Q$ | 热量 | J, kJ | 正=吸热，负=放热 |
| $W$ | 功 | J, kJ | 正=系统对外做功 |
| $\dot{Q}, \dot{W}$ | 热传递率 / 功率 | W, kW | 点=除以时间 |
| $\dot{m}$ | 质量流量 | kg/s | $\dot{m} = \rho AV$ |
| $R$ | 气体常数 | J/(kg·K) | 每种气体不同 |
| $R_u$ | 通用气体常数 | 8.314 kJ/(kmol·K) | 所有气体相同 |
| $k$ | 比热比 $c_p/c_v$ | — | 空气=1.4 |
| $c_v, c_p$ | 定容/定压比热 | J/(kg·K) | $c_p - c_v = R$ |
| $x$ | 干度 | — | 0=纯液，1=纯气 |
| $\eta$ | 效率 | — | 0~1 |
| $\sigma$ | Stefan-Boltzmann 常数 | $5.67 \times 10^{-8}$ W/(m²·K⁴) | 辐射用 |
| $Re$ | 雷诺数 | — | 惯性力/粘性力 |
| $Pr$ | 普朗特数 | — | 动量扩散/热扩散 |
| $Nu$ | 努塞尔数 | — | 对流/导热 |
| $Gr$ | 格拉晓夫数 | — | 浮力/粘性力 |

## 符号约定

| 约定 | 说明 |
|------|------|
| $Q > 0$ | 热量传入系统（吸热） |
| $Q < 0$ | 热量传出系统（放热） |
| $W > 0$ | 系统对外做功（BY） |
| $W < 0$ | 外界对系统做功（ON） |
| 大写字母 | 广延性质（总量） |
| 小写字母 | 强度性质（每千克） |
| 字母上方点 | 除以时间（率） |
| 下标 $f$ | 饱和液体 (fluid) |
| 下标 $g$ | 饱和蒸气 (gas) |
| 下标 $fg$ | $g - f$（差值） |
