## 一、开口系统回顾

### 1.1 什么是开口系统？

> **开口系统（控制体积, Control Volume）**：质量可以穿过边界进出，能量也可以穿过边界。

与闭口系统的关键区别：

| | 闭口系统 | 开口系统 |
|---|---------|---------|
| 质量变化 | 不变 ($m = \text{const}$) | 可变（进出） |
| 能量传递方式 | 热量 $Q$ + 功 $W$ | 热量 $Q$ + 功 $W$ + **质量流携带的能量** |
| 典型设备 | 活塞气缸 | 涡轮、压缩机、泵、喷管、节流阀 |
| 核心方程 | $\Delta U = Q - W$ | $\dot{Q} - \dot{W} = \dot{m}(h_2 - h_1)$ |

> **开口系统比闭口系统多了一种能量传递方式**——质量流。物质进进出出的同时，也带进带出能量。这就是为什么开口系统的方程里多了一个焓 $h$。

## 二、质量守恒

### 2.1 质量守恒原理

> **质量守恒**：质量不能被创造或消灭。控制体积内质量的变化 = 进入的质量 − 离开的质量。

$$m_{\text{in}} - m_{\text{out}} = \Delta m_{\text{CV}}$$

用浴缸类比：放水进来 50 kg，排水出去 30 kg，浴缸里多了 20 kg。

### 2.2 稳态条件下的质量守恒

> **稳态（Steady State）**：控制体积内的质量不随时间变化（$\Delta m_{\text{CV}} = 0$）。

此时：

$$\dot{m}_{\text{in}} = \dot{m}_{\text{out}}$$

即进入的质量流量 = 离开的质量流量。

对于单入口单出口的设备（如涡轮、压缩机）：

$$\dot{m}_1 = \dot{m}_2$$

### 2.3 质量流量公式

$$\dot{m} = \rho \times A \times V$$

- $\dot{m}$ = 质量流量 [kg/s]
- $\rho$ = 密度 [kg/m³]
- $A$ = 截面积 [m²]
- $V$ = 流速 [m/s]（速度，不是体积！）

> **怎么理解**：截面积 $A$ × 流速 $V$ = 每秒"扫过"的体积 [m³/s]（体积流量 $\dot{V} = AV$），再乘以密度 $\rho$ 就是质量流量。

也可以用比容表示（$\rho = 1/v$）：

$$\dot{m} = \frac{A \times V}{v}$$

> **流体力学 vs 热力学开口系统——同一套骨架，层层升级**
>
> 流体力学里，体积流量守恒 + 伯努利方程联立 = 完备（类似电路的 KCL + KVL）。热力学开口系统完全沿用这个骨架，但把每个部件都升了级：
>
> | | 流体力学（不可压） | 热力学开口系统 | 升级原因 |
> |---|---|---|---|
> | "KCL" | \(A_1V_1 = A_2V_2\) | \(\dot{m} = \rho AV = AV/v\) | 密度会变（压缩/膨胀），体积流量不守恒，只能守质量 |
> | "KVL" | 伯努利 | 稳态流动能量方程 | 撤掉了 q=0, w=0, u不变, 不可压 四个假设 |
> | 压力能 | \(P/\rho\) | \(Pv\)（打包进 \(h = u+Pv\)） | v 会变，ρ 不是常数 |
> | 内能 | 隐藏（假设 u 不变） | u 出场 | 摩擦生热、压缩升温 |
> | 热和功 | 不存在 | q 和 w 进场 | 涡轮/压缩机/换热器 |
>
> 两式联立就是开口系统 1D 稳态流动的完整骨架，解两个未知数，没漏。需要更多才补物性方程（\(Pv=RT\)、查表）或第二定律（等熵效率）。

## 三、流动功与焓——为什么开口系统要用焓

### 3.1 流动功（Flow Work）

> **流动功**：把质量推过控制体积边界所需的功。

**一句话理解**：流体不会自己飞进控制体积，得被后面的人**挤**进去。挤进去要克服压力，这个"挤"的功就是流动功。

- **闭口系统** = 密封盒子，盒子壁在动（活塞），动就做功 → **边界功**
- **开口系统** = 有门的房间，门不动，但人要进出。人被挤进去克服压力做的功 → **流动功**

每 1 kg 流体进来，交一份"入场费" $Pv$。每 1 kg 流体出去，也交一份"出场费" $Pv$。这份费用只跟当前状态（$P$、$v$）有关，所以打包进了焓 $h = u + Pv$——开口系统用焓不用内能，就是因为进出时要算这份入场/出场费。

> **入场费/出场费——谁付谁收？**
>
> **入口（入场费 $P_1 v_1$）**：
> - **谁付**：上游流体（它把新流体"推"进来，推的功 = $P_1 v_1$）
> - **谁收**：控制体积（能量**进入**系统）
>
> **出口（出场费 $P_2 v_2$）**：
> - **谁付**：控制体积（它把流体"推"出去，推的功 = $P_2 v_2$）
> - **谁收**：下游流体（能量**离开**系统）
>
> 入口收到的 $P_1 v_1$ 和内能 $u_1$ 都是"随流体进来的"，出口付出的 $P_2 v_2$ 和 $u_2$ 都是"随流体出去的"。打包成 $h = u + Pv$ 后，能量平衡里直接写 $(h_2 - h_1)$，入场费和出场费自动算进去了。

对 1 kg 流体（比容 $v$），流动功为：

$$w_{\text{flow}} = P \times v$$

- $P$ = 压力 [kPa]
- $v$ = 比容 [m³/kg]
- $P \times v$ = [kPa × m³/kg] = [kN/m² × m³/kg] = [kN·m/kg] = [kJ/kg]

> **流动功是开口系统独有的概念。** 闭口系统没有质量进出，所以没有流动功。闭口系统对应的是**边界功**（$W_b = \int P\,dV$，边界本身在动）——两者是闭口/开口系统的分水岭。

> **为什么流动功 = $Pv$（推导）**：推 1 kg 流体（比容 $v$）通过开口截面 $A$，反抗压力 $P$：
>
> - 力 = $P \times A$（压力 × 面积）
> - 距离 = $v / A$（体积 ÷ 面积 = 推进的长度）
> - 功 = 力 × 距离 = $P \times A \times (v/A) = P \times v$
>
> 面积 $A$ 消掉了 → 不管开口多大，流动功都是 $Pv$。**不需要积分**（跟边界功不同），因为这不是渐进过程，是"把这块流体推过去"一次性的事。

> **整条流道是一根接力棒**：入口收棒（上游推我，$P_1v_1$ 进入），出口交棒（我推下游，$P_2v_2$ 离开）。流体本身从头到尾只带 u+KE+PE；Pv 是流道每一站之间传递的接力棒，只存在于交接的瞬间。压缩机 $P_2>P_1$ → 交出去的比收进来的多 → 净支出 → 需要轴功 w 来补。涡轮 $P_1>P_2$ → 收进来的比交出去的多 → 净收入 → 输出轴功 w。

### 3.2 流动流体的总能量

**闭口系统**（流体不穿过边界，但可以在内部运动），每千克流体自身拥有的能量：

$$e = u + \frac{V^2}{2} + gz$$

- $u$ = 比内能 [kJ/kg]——分子运动的能量
- $\frac{V^2}{2}$ = 比动能 [kJ/kg]——流体在系统内部跑多快（$V$ = 速度；$V=0$ 时此项归零）
- $gz$ = 比势能 [kJ/kg]——流体在系统内部的高度

**开口系统**（流体穿过边界进出），除了流体自身携带的 $u + V^2/2 + gz$，每进 1 kg 流体还伴随一笔"入场费"——上游流体把它推进控制体积时做的功 $Pv$：

$$e_f = \underbrace{Pv}_{\text{入场费（上游做的功）}} + \underbrace{u + \frac{V^2}{2} + gz}_{\text{流体自身拥有的}}$$

> **Pv 不是流体"具有的"能量，而是流体进场事件附带发生的能量转移。** 会计上按"每 kg 进/出了什么"记账，所以把这笔入场费也贴在流体身上一起算。就好比进场看球赛：你自己的体温（u）+ 跑起来的速度（KE）+ 你在几楼（PE）是你自己拥有的；检票口被人群推了一把塞进来的那一推（Pv）不是你的能量，但系统确实因为它多收了一笔功——按人头记账时就把它捆在你身上了。
>
> **所以 Pv 不计入 E = U + KE + PE 里面。** E 统计的是系统内部的能量。Pv 是流过边界时的一笔功，属于能量传递（类似 Q 和 W），不是系统内部存储的能量。只是开口系统的会计习惯把它跟流体本身打包在一起构成焓 $h = u + Pv$，让能量平衡方程更简洁。
>
> **为什么推了一把不算进 KE？** 因为 Pv 不是让流体加速的功——它是克服边界压力、把流体塞过门的功。进门之后流体跑多快是自己的事，已经体现在 $V^2/2$ 里了。两笔账互不替代。
>
> **为什么是状态量？** $Pv = P \cdot A \cdot (v/A) = F \times s$。和势能 $mgz$ 一样的逻辑——$mgz$ 也是 $F \times s$（重力 × 高度），但知道 $z$ 就能算，不用管"谁推的、推了多久"。Pv 同理：知道入口的 $P$ 和 $v$ 就能算，打包完固化成了状态量。一旦塞进 $h = u + Pv$，焓就是状态函数，开口系统直接用 $h_2 - h_1$，入场费出场费自动抵消。

### 3.3 焓的完整解释

回忆第三篇提前定义的焓：

$$h = u + Pv$$

把它代入流动流体的能量表达式：

$$e_f = h + \frac{V^2}{2} + gz$$

> **现在你明白焓为什么存在了**：$h = u + Pv$ 把"内能"和"流动功"打包在一起，这样流动流体的能量公式就很简洁——焓 + 动能 + 势能。
>
> **焓的物理意义**：1 kg 流体在流过控制体积边界时携带的"热力学能量"（内能 + 流动功）。开口系统的方程用焓而不是内能，就是因为流体进出时同时携带了内能和流动功。

> **$e_f$ 和 $h$ 一样，是边界会计量，不是流体一直"具有的"能量。**
>
> `e_f` 只存在于穿越边界的瞬间——流体进场时，上游推的那一把（Pv）作为入场费贴在身上；进场之后，Pv 已经作为功进入了系统的能量平衡，**流体内部只剩下 $u + V^2/2 + gz$**。好比环形循环水流：泵出口高压 → `e_f` 很大；但水在管道内部跑时，Pv 并不"飘在水里"——它只在一进一出的边界处一闪而过。
>
> 所以 `e_f` 和 `h` 都是**为边界会计服务的构造量**，不是流体真实的、恒常携带的能量。如果你在管道中间截一段，你不会看到 Pv 堆积——它已经被消耗在系统的能量转移中了，再算就重复。

> **那多加 Pv 到底有什么用？——为了让开口系统方程跟闭口系统一样简洁。**
>
> 如果不用焓，稳态开口系统的能量平衡得写成这样（每 kg，一进一出各拆两笔）：
>
> $$(u_1 + KE_1 + PE_1) + P_1v_1 + q = (u_2 + KE_2 + PE_2) + P_2v_2 + w$$
>
> $u_1$ 和 $P_1v_1$ 总是成对出现在同一侧，$u_2$ 和 $P_2v_2$ 亦然。把它们打包成 $h = u + Pv$，方程瞬间变一行：
>
> $$h_1 + KE_1 + PE_1 + q = h_2 + KE_2 + PE_2 + w$$
>
> 移项就是标准稳态流能量方程：
>
> $$q - w = (h_2 - h_1) + \Delta KE + \Delta PE$$
>
> 跟闭口系统的 $q - w = \Delta u + \Delta KE + \Delta PE$ 对比——形式完全一致，只是 $u$ 换成了 $h$。这就是打包的全部动机：**代数简洁，不是为了捏造新物理量。** `e_f` 只是这个打包在"总能量"层面的延伸——它确实比 $e = u + KE + PE$ 多了 Pv，但多出来的不是假的，是真实进入系统的入场功，只是会计把它贴在流体身上一起记账。

> **Pv 的本质：潜伏的压强能，在边界处兑现。**
>
> 环形循环流道里，流体在每一点都有压强 P 和比容 v。P×v 是一个真实存在的能量额度——如果在这里割一个口子，每 kg 流体会对外做 Pv 的功（喷射出来）。但没有口子的时候，这笔能量**潜伏在受压流体和管壁的弹性形变中**，从不单独出现在 E = U + KE + PE 里。
>
> 割开口子的瞬间：边界出现 → Pv 从潜伏的压强能兑现为真实的流动功。**Pv 不是人造虚构量——它是真实存储的弹性势能，只是在闭口系统里没有"出场机会"，只在边界穿越时被记账。**
>
> 入口检录 → Pv 进入系统黑箱 → 可能变成 KE↑（加速）、u↑（摩擦生热）、PE↑（打水上楼）、管壁形变、轴功 w…… → 出口处重新凝聚成一笔 P₂v₂ 付给下游。
>
> **`e_f` 的名称与物理意义**：流体总比能（specific total energy of a flowing fluid）= 1 kg 流体穿越控制体积边界时，伴随它进入系统的**全部能量**。= 流体自身携带的（u+KE+PE）+ 此刻正在被推的这一把（Pv）。它是一个**事件量**——只在穿越边界事件中定义，不是恒常属性。

## 四、开口系统能量平衡

### 4.1 一般形式

$$\dot{Q} - \dot{W} = \dot{m}\left(h_2 - h_1 + \frac{V_2^2 - V_1^2}{2} + g(z_2 - z_1)\right)$$

- $\dot{Q}$ = 热传递率 [kW]（正 = 吸热）
- $\dot{W}$ = 功率 [kW]（正 = 系统对外做功）
- $\dot{m}$ = 质量流量 [kg/s]
- $h_1, h_2$ = 进出口比焓 [kJ/kg]
- $V_1, V_2$ = 进出口速度 [m/s]
- $z_1, z_2$ = 进出口高度 [m]

### 4.2 简化形式

**大多数情况下，动能和势能变化可以忽略**（流体速度变化不大，高度差不大）：

$$\dot{Q} - \dot{W} = \dot{m}(h_2 - h_1)$$

除以 $\dot{m}$ 得到比形式（每千克）：

$$q - w = h_2 - h_1 = \Delta h$$

> **对比闭口系统**：
> - 闭口系统：$q - w = \Delta u$（内能变化）
> - 开口系统：$q - w = \Delta h$（焓变化）
> - 两者只差一个字母，因为焓把流动功自动打包了

> **稳态流动能量方程与伯努利方程的关系**：
>
> 稳态流动能量方程是**伯努利的"完整版"**。伯努利只是它的一个特例——加上四个约束条件后退化得到：
>
> $$h_1 + \frac{V_1^2}{2} + gz_1 + q = h_2 + \frac{V_2^2}{2} + gz_2 + w \quad\text{（完整版）}$$
>
> 令 $q=0$（无热）、$w=0$（无轴功）、$v=1/\rho$（不可压缩）、$u_1=u_2$（无摩擦、无温变）：
>
> $$h = u + Pv = u + \frac{P}{\rho} \;\xrightarrow{u_1=u_2}\; \boxed{\frac{P_1}{\rho} + \frac{V_1^2}{2} + gz_1 = \frac{P_2}{\rho} + \frac{V_2^2}{2} + gz_2}$$
>
> 这就是伯努利方程。所以伯努利不跟 Pv 矛盾——伯努利的 $P/\rho$ **就是** Pv。
>
> **Pv 会变，但能量始终守恒。** 文丘里管里流速变快 → 压强降低 → $P/\rho$ 变小、$V^2/2$ 变大，总和不变。泵里 Pv 变大 ← 消耗了轴功 w。涡轮里 Pv 变小 → 输出了功。Pv 的变化正是能量在不同形式之间转换的体现——压力能 ↔ 动能 ↔ 势能 ↔ 轴功 ↔ 内能，总和永远守恒。
>
> 差异就在于流动功 $Pv$。开口系统多了流动功，所以用焓代替内能。

### 4.3 稳态假设

> 稳态意味着：控制体积内质量和能量都不随时间积累。

稳态假设下：
1. $\dot{m}_{\text{in}} = \dot{m}_{\text{out}}$（质量不积累）
2. $\frac{dE_{\text{CV}}}{dt} = 0$（能量不积累）
3. 所有属性在每个点不随时间变化

> **考试中除非题目特别说明，否则默认稳态。**

## 五、常见稳态流动设备

PPT 介绍了多种工程设备。每个设备有其特定的假设条件，导致能量方程简化成不同的形式。

### 5.1 喷管与扩压器（Nozzles & Diffusers）

| | 喷管 (Nozzle) | 扩压器 (Diffuser) |
|---|---|---|
| 功能 | 增加速度 | 增加压力 |
| 代价 | 压力下降 | 速度下降 |
| 例子 | 火箭尾喷管、花园水管 | 喷气发动机进气口 |

**假设**：
- 绝热（$Q = 0$，过程很快来不及换热）
- 无功（$\dot{W} = 0$，没有转轴）
- 忽略势能（$\Delta PE = 0$）
- **不忽略动能**（这正是喷管/扩压器的核心！）

**能量方程**：

$$0 = h_2 - h_1 + \frac{V_2^2 - V_1^2}{2}$$

$$h_1 + \frac{V_1^2}{2} = h_2 + \frac{V_2^2}{2}$$

> 速度增加 → 焓减少（压力下降）；速度减少 → 焓增加（压力上升）。

### 5.2 涡轮与压缩机（Turbines & Compressors）

| | 涡轮 (Turbine) | 压缩机 (Compressor) |
|---|---|---|
| 功能 | 流体推动叶片 → 输出轴功 | 输入轴功 → 压缩流体 |
| 能量方向 | 流体能量 → 功 | 功 → 流体能量 |
| $W$ 的符号 | $W > 0$（系统对外做功） | $W < 0$（外界对系统做功） |
| 处理对象 | 气体或蒸汽 | 气体 |

**泵（Pump）**：跟压缩机一样，但处理的是液体。

**假设**：
- 通常绝热（$Q \approx 0$）
- 忽略 KE 和 PE

**能量方程**（绝热，忽略 KE/PE）：

$$-\dot{W} = \dot{m}(h_2 - h_1)$$

对涡轮（$W > 0$，膨胀做功，$h_2 < h_1$）：

$$\dot{W} = \dot{m}(h_1 - h_2)$$

对压缩机（$W < 0$，压缩耗功，$h_2 > h_1$）：

$$\dot{W}_{\text{in}} = \dot{m}(h_2 - h_1)$$

### 5.3 节流阀（Throttling Valve）

> **节流阀**：造成流体压力显著下降的流动限制装置。你家水龙头就是一个节流阀。

**假设**：
- 绝热（$Q = 0$）
- 无功（$W = 0$）
- 忽略 KE 和 PE

**能量方程**：

$$0 = h_2 - h_1 \implies h_1 = h_2$$

> **节流过程的核心结论：焓不变**（$h_1 = h_2$）。压力下降，但焓保持恒定。

> **为什么忽略 KE 和 PE？** 节流阀进出口管径一般相同 → 流速接近 → ΔKE ≈ 0（流体过窄口先加速再减速，进出口速度自抵消）。阀体很小 → Δz ≈ 0 → ΔPE ≈ 0。而且典型焓变在 kJ/kg 量级，KE 和 PE 在 J/kg 量级，差了 2-3 个数量级，忽略完全合理。

> 节流阀广泛应用于制冷和空调系统——压力下降导致温度下降。

### 5.4 混合室（Mixing Chamber）

> **混合室**：两股或多股流体混合的地方。

**假设**：
- 绝热（$Q = 0$，通常近似）
- 无功（$W = 0$）
- 忽略 KE 和 PE

> **为什么忽略 KE 和 PE？** 混合室的目标是汇流混合，不是加速。各进出口管径一般相同 → 流速接近 → ΔKE ≈ 0。通常水平放置 → ΔPE ≈ 0。混合带来的 Δh 在 kJ/kg 量级，KE 和 PE 在 J/kg 量级，差了上千倍，忽略无影响。

**质量守恒**：$\dot{m}_1 + \dot{m}_2 = \dot{m}_3$（进口流量之和 = 出口流量）

**能量方程**：

$$\dot{m}_1 h_1 + \dot{m}_2 h_2 = \dot{m}_3 h_3$$

> 两股流体混合后，出口焓是进口焓的质量流量加权平均。

### 5.5 热交换器（Heat Exchanger）

> **热交换器**：两股流体不混合但交换热量。

**假设**：
- 无功（$W = 0$）
- 忽略 KE 和 PE
- 整体绝热（两股流体之间的热交换是内部的，对外界绝热）

> **为什么忽略 KE 和 PE？** 热交换器的目标是换热，不是加速流体。进出口管径一般相同 → 流速接近 → ΔKE ≈ 0。通常水平放置 → ΔPE ≈ 0。且换热带来的 Δh 在 ~200 kJ/kg 量级，Ke 和 PE 在 ~20 J/kg 量级（差一万倍），忽略毫无影响。

**能量方程**（两股流体）：

$$\dot{m}_A(h_{A,\text{in}} - h_{A,\text{out}}) = \dot{m}_B(h_{B,\text{out}} - h_{B,\text{in}})$$

> A 流体放出的热量 = B 流体吸收的热量。

## 六、各设备方程总结

| 设备 | 特殊条件 | 简化方程 |
|------|---------|---------|
| 喷管/扩压器 | $Q=0$, $W=0$, $\Delta PE=0$ | $h_1 + \frac{V_1^2}{2} = h_2 + \frac{V_2^2}{2}$ |
| 涡轮 | $Q \approx 0$, 忽略 KE/PE | $\dot{W} = \dot{m}(h_1 - h_2)$ |
| 压缩机/泵 | $Q \approx 0$, 忽略 KE/PE | $\dot{W}_{\text{in}} = \dot{m}(h_2 - h_1)$ |
| 节流阀 | $Q=0$, $W=0$, 忽略 KE/PE | $h_1 = h_2$ |
| 混合室 | $Q=0$, $W=0$, 忽略 KE/PE | $\sum \dot{m}_{\text{in}} h_{\text{in}} = \sum \dot{m}_{\text{out}} h_{\text{out}}$ |
| 热交换器 | $W=0$, 忽略 KE/PE | $\dot{m}_A \Delta h_A = -\dot{m}_B \Delta h_B$ |

## 七、例题与方法论

### 7.1 开口系统解题总流程

```
1. 识别设备类型（喷管？涡轮？节流阀？混合室？热交换器？）
2. 写出稳态流动能量方程的一般形式：
   q − w = (h₂ − h₁) + (V₂² − V₁²)/2 + g(z₂ − z₁)
3. 根据设备类型列假设，划掉为 0 的项：
   - Q=0？→ 绝热
   - W=0？→ 无轴功
   - ΔKE≈0？→ 进出口速度差不多（或题目给了速度就保留）
   - ΔPE≈0？→ 高度差可忽略（或给了高度就保留）
4. 得到该设备的简化方程
5. 用物性表或理想气体公式补数据：
   - 查表：已知 P,T → 查 h（过热水蒸气表/饱和表）
   - 理想气体：Δh = c_p,avg × ΔT
6. 代入求解
```

> **核心心态**：开口系统题看起来复杂，但每次都是同一套公式。**关键是识别设备 → 列假设 → 套简化方程。** 不要被"涡轮""压缩机"这些词吓住——它们只是帮你确定 W 的符号和哪些项可以忽略。

---

### 例题1：喷管（Nozzle）

> 蒸汽进入喷管：$P_1 = 2$ MPa, $T_1 = 400$°C, $V_1 = 50$ m/s。出口 $P_2 = 0.1$ MPa, $V_2 = 500$ m/s。绝热，忽略势能。求出口焓 $h_2$ 和出口温度（如果是过热蒸汽）。已知 $h_1 = 3248$ kJ/kg。

**第 1 步**：识别 → 喷管。假设：$q = 0$, $w = 0$, $\Delta PE = 0$。

**第 2 步**：套简化方程：

$$h_1 + \frac{V_1^2}{2} = h_2 + \frac{V_2^2}{2}$$

注意单位！速度 m/s → 动能 kJ/kg 要 ÷1000：

$$\frac{V_1^2}{2} = \frac{50^2}{2 \times 1000} = 1.25 \text{ kJ/kg}$$

$$\frac{V_2^2}{2} = \frac{500^2}{2 \times 1000} = 125 \text{ kJ/kg}$$

**第 3 步**：解 $h_2$：

$$h_2 = h_1 + \frac{V_1^2 - V_2^2}{2} = 3248 + 1.25 - 125 = 3124.25 \text{ kJ/kg}$$

**第 4 步**：查表判断相态。手里有 $P_2 = 0.1$ MPa 和 $h_2 = 3124.25$ kJ/kg，没有 $T_2$。查饱和表（按 P=0.1 MPa）→ $h_g = 2675$ kJ/kg，$h_f = 417.5$ kJ/kg。$h_2 > h_g$ → **过热蒸气**。查过热水蒸气表（P=0.1 MPa），找 h≈3124 的行 → $T_2 \approx 280$°C。

> **坑**：速度单位！m²/s² = J/kg，kJ/kg 要 ÷1000。忘记换算会导致结果差三个数量级。

---

### 例题2：涡轮（Turbine）

> 蒸汽进入涡轮：$\dot{m} = 2$ kg/s, $P_1 = 5$ MPa, $T_1 = 500$°C, $h_1 = 3425$ kJ/kg。出口 $P_2 = 0.1$ MPa, 干度 $x_2 = 0.95$。绝热，忽略 KE/PE。求涡轮功率。已知饱和表：$P=0.1$ MPa 时 $h_f = 417.5$, $h_g = 2675$ kJ/kg。

**第 1 步**：识别 → 涡轮。假设：$q = 0$, 忽略 KE/PE。

**第 2 步**：套简化方程（涡轮：系统对外做功，$w > 0$）：

$$w = h_1 - h_2$$

**第 3 步**：算 $h_2$（出口是两相混合物，用于干度插值）：

$$h_2 = h_f + x_2 \cdot (h_g - h_f) = 417.5 + 0.95 \times (2675 - 417.5) = 2562.1 \text{ kJ/kg}$$

**第 4 步**：算功率：

$$\dot{W} = \dot{m} \times w = 2 \times (3425 - 2562.1) = 1725.8 \text{ kW}$$

> **坑**：出口是两相混合物时，焓要用干度插值 $h = h_f + x \cdot h_{fg}$，不能直接查过热度表。

---

### 例题3：压缩机（Compressor）

> 空气压缩机入口 $P_1 = 100$ kPa, $T_1 = 300$ K，出口 $P_2 = 800$ kPa，流量 $\dot{m} = 0.5$ kg/s。绝热，忽略 KE/PE。空气 $c_p = 1.005$ kJ/(kg·K), $k = 1.4$。求最小输入功率。

**第 1 步**：识别 → 压缩机。假设：$q = 0$, 忽略 KE/PE。

**第 2 步**：套简化方程（压缩机：外界对系统做功，$w_{\text{in}} > 0$）：

$$w_{\text{in}} = h_2 - h_1$$

**第 3 步**：理想气体，$h_2 - h_1 = c_p(T_2 - T_1)$，需要 $T_2$。假设可逆绝热（最小功 = 等熵过程）：

$$T_2 = T_1 \left(\frac{P_2}{P_1}\right)^{\frac{k-1}{k}} = 300 \times \left(\frac{800}{100}\right)^{\frac{0.4}{1.4}} = 300 \times 8^{0.286} \approx 544 \text{ K}$$

**第 4 步**：

$$w_{\text{in}} = 1.005 \times (544 - 300) = 245.2 \text{ kJ/kg}$$

$$\dot{W}_{\text{in}} = \dot{m} \times w_{\text{in}} = 0.5 \times 245.2 = 122.6 \text{ kW}$$

> **坑**："最小功率" = 可逆绝热（等熵）。如果不是最小功率，题目会给出口温度或等熵效率。

---

### 例题4：节流阀（Throttling Valve）

> 制冷剂 R-134a 进入节流阀：$P_1 = 1.2$ MPa, 饱和液体（$x=0$），出口 $P_2 = 0.2$ MPa。绝热，忽略 KE/PE。求出口干度 $x_2$。

已知：$P=1.2$ MPa 时 $h_f = 117.8$ kJ/kg。$P=0.2$ MPa 时 $h_f = 38.47$, $h_g = 241.3$ kJ/kg。

**第 1 步**：识别 → 节流阀。节流阀的核心：$h_1 = h_2$。

**第 2 步**：入口饱和液体 → $h_1 = h_f(1.2\text{ MPa}) = 117.8$ kJ/kg。

**第 3 步**：$h_2 = h_1 = 117.8$ kJ/kg。出口 $P_2 = 0.2$ MPa，查饱和表：

$$x_2 = \frac{h_2 - h_f}{h_g - h_f} = \frac{117.8 - 38.47}{241.3 - 38.47} = 0.391$$

> **节流阀题的精髓**：$h_1 = h_2$ 是铁律。入口已知 → $h_1$ 已知 → $h_2$ 已知 → 在出口压力下反查干度或温度。

---

### 例题5：混合室（Mixing Chamber）

> 两股水混合：$\dot{m}_1 = 2$ kg/s, $T_1 = 80$°C（$h_1 = 335$ kJ/kg）；$\dot{m}_2 = 0.5$ kg/s, $T_2 = 20$°C（$h_2 = 83.9$ kJ/kg）。绝热，忽略 KE/PE。求出口温度。

**第 1 步**：识别 → 混合室。假设：$q = 0$, $w = 0$, 忽略 KE/PE。

**第 2 步**：混合室方程：

$$\dot{m}_1 h_1 + \dot{m}_2 h_2 = \dot{m}_3 h_3$$

质量守恒：$\dot{m}_3 = \dot{m}_1 + \dot{m}_2 = 2.5$ kg/s。

**第 3 步**：

$$h_3 = \frac{2 \times 335 + 0.5 \times 83.9}{2.5} = 284.8 \text{ kJ/kg}$$

查饱和水表：$h_3 = 284.8$ kJ/kg → $T_3 \approx 68$°C。

---

### 例题6：热交换器（Heat Exchanger）

> 逆流热交换器，热水侧：$\dot{m}_A = 1.5$ kg/s, $T_{A,\text{in}} = 90$°C, $T_{A,\text{out}} = 50$°C；冷水侧：$\dot{m}_B = 2$ kg/s, $T_{B,\text{in}} = 20$°C。水 $c_p \approx 4.18$ kJ/(kg·K)。求冷水出口温度。

**第 1 步**：识别 → 热交换器。$w = 0$, 忽略 KE/PE, 整体绝热。

**第 2 步**：热交换器方程：

$$\dot{m}_A (h_{A,\text{in}} - h_{A,\text{out}}) = \dot{m}_B (h_{B,\text{out}} - h_{B,\text{in}})$$

水 $c_p$ 近似常数 → $h = c_p T$（以某个参考温度为基准，差值消掉参考）：

**第 3 步**：

$$1.5 \times 4.18 \times (90 - 50) = 2 \times 4.18 \times (T_{B,\text{out}} - 20)$$

$$1.5 \times 40 = 2 \times (T_{B,\text{out}} - 20)$$

$$T_{B,\text{out}} = 20 + \frac{60}{2} = 50\text{°C}$$

## 八、非稳态流动（了解）

PPT 简要提到了非稳态过程——控制体积内的质量或能量随时间变化的情况。

**例子**：向刚性容器充气（从空到满），或从容器放气（从满到空）。

**质量平衡**：$m_{\text{in}} - m_{\text{out}} = m_{\text{final}} - m_{\text{initial}}$

**能量平衡**：

$$Q - W = m_{\text{final}}u_{\text{final}} - m_{\text{initial}}u_{\text{initial}} - m_{\text{in}}h_{\text{in}} + m_{\text{out}}h_{\text{out}}$$

> 这部分考试一般不深究，知道有这个概念就行。

## 九、本篇小结

### 核心概念

| 概念 | 要点 |
|------|------|
| 开口系统 | 质量和能量都可穿过边界 |
| 控制体积 | 开口系统的研究区域 |
| 稳态 | 质量和能量都不随时间积累 |
| 质量流量 $\dot{m} = \rho AV$ | 每秒流过的质量 |
| 流动功 $Pv$ | 推动流体进出控制体积所需的功 |
| 焓 $h = u + Pv$ | 内能 + 流动功，开口系统的核心量 |
| 节流 | $h_1 = h_2$（焓不变） |

### 必记公式

| 公式 | 含义 |
|------|------|
| $\dot{m} = \rho AV = \frac{AV}{v}$ | 质量流量 |
| $\dot{Q} - \dot{W} = \dot{m}(h_2 - h_1)$ | 稳态开口系统能量平衡（忽略 KE/PE） |
| $q - w = \Delta h$ | 比形式 |
| $\dot{W} = \dot{m}(h_1 - h_2)$ | 涡轮功率（绝热） |
| $h_1 = h_2$ | 节流阀 |
| $h_1 + \frac{V_1^2}{2} = h_2 + \frac{V_2^2}{2}$ | 喷管/扩压器 |
| $\sum \dot{m}_{\text{in}} h_{\text{in}} = \sum \dot{m}_{\text{out}} h_{\text{out}}$ | 混合室 |

### 闭口 vs 开口对比

| | 闭口系统 | 开口系统 |
|---|---------|---------|
| 质量 | 不变 | 可变 |
| 能量方程 | $Q - W = \Delta U$ | $\dot{Q} - \dot{W} = \dot{m}\Delta h$ |
| 核心量 | 内能 $u$ | 焓 $h = u + Pv$ |
| 功类型 | 边界功 $\int P\,dV$ | 轴功、流动功 |
| 典型设备 | 活塞气缸 | 涡轮、压缩机、泵、喷管、节流阀 |
